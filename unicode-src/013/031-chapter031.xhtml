೩೧ :: ಆದೊಡಂ ಪಣ್ಣೆಂಬುದಂ-ಸೈರಿಸಮ್

ತಳಿರೆಂದೊಡೆ ಅಸುಗೆಯೊಂದೊಡೆ|
ಗಿಳಿಯೆಂದೊಡೆ ಕರೆವ ಕೋಗಿಲೆಗಳೆಂದೊಡೆ ಬಂ||
ದೆಳ ಮಾವೆಂದೊಡೆ ನೆರೆದಿಂ|
ಗಳ ತಣ್ಬೆಳಗೆಂದೊಡೆ ಉದ್ದಂ ಉರಿವಂ ಗೊರವಂ||
ಪುದಿದ ಉರಿಗಣ್ಣನ ಆಣೆಯನ್ ಅದೇವೋಗಳ್ವೆಂಃ
-ಮುಗುಳೆಂದು ಮುಟ್ಟಬಾ|
ರದು ಬನಂ ಎಂದು ಬಾಯ್ದೆರಯಬಾರದು
ಸಂಪಗೆಯೆಂದು ಸೂಡಬಾ| ರದು
ಕೊಳನೆಂದು ಕೆಯ್ದುಡಕಬಾರದು ತಾವರೆಯೆಂದು ತೋರಬಾ|
ರದು ಪೊಸ ಸುಗ್ಗಿಯೆಂದೂ
ಉಸಿರಬಾರದು ತನ್ನಯ ಮಂಜುವೆಟ್ಟಿನೊಳ್||ವಚನ||
ಮತ್ತಂ ಆತಂ ಸೀರೆಯೊಳ್
ಆದೊಡಂ ಅಂಚೆಯೆಂಬುದಂ,
ಕೆರೆಯೊಳ್ ಆದೊಡಂ ಬಾಳೆಯೆಂಬುದಂ ಸಿರಿಕಂಡದೊಳ್
ಆದೊಡಂ ಆರಡಿಯೆಂಬಿದಂ,
ಕೊಡಗೂಸಿನುಡುಗೆಯೊಳ್ ಆದೊಡಂ
ಸೋಗೆಯೆಂಬುದಂ,
ಬೆದಂಡೆಯೊಳ್ ಆದೊಡಂ ಪಾಡು ಎಂಬುದಂ, ಕೇರಿಯೊಳ್
ಆದೊಡಂ ನೆರೆಯೆಂಬುದಂ,
ಚಪ್ಪರಣೆಯೊಳೆ ಆದೊಡಂ ಪಣ್ಣೆಂಬುದಂ-ಸೈರಿಸಮ್...||

-ಆಂಡಯ್ಯ, ಕಬ್ಬಿಗರ ಕಾವಂ, ೧೧೫-೧೧೬

೩೧ :: ಹೆಣ್ಣ ಸುದ್ದಿಯ ಇತ್ತ ಎತ್ತಲೇ ಬೇಡಿ

ಆ ಗೊರವ ಉರಿಯುವನು, ಅವನೆದುರು ಆಡದಿರಿ
ಬಾಯ ತಪ್ಪಿಯೂ, ಜೋಕೆ, ತಳಿರು ಎಂದೆನ್ನದಿರಿ!
ಅಶೋಕ, ಗಿಳಿ, ಮಾವು, ಕುಹೂ ಕುಹೂ ಕೋಗಿಲೆಯ
ತಂಬೆಳಕ ತಿಂಗಳ ಸುದ್ದಿ ಎತ್ತ ಬೇಡಿ!

ಮೊಗ್ಗರಳಿ ಹೂವಾಯ್ತೆಂದು ಮುಟ್ಟಬಾರದು ನೀವು;
ಬಾಯ್ತೆರೆದು ಹೇಳೀರಿ: ''ಆಹಾ ಏನು ಸೊಬಗಿನ ತೋಟ!''"
ಕೂಡದದು, ಮುಡಿಯಲು ಬೇಡಿ ಕಂಪು ಸಂಪಿಗೆ ಹೂ-
ತಿಳಿಗೊಳದಿ ಆಟಕ್ಕೆಂದು ಕೈ ಕಾಲನಾಡಿಸಬೇಡಿ,
ಅಲ್ಲಿ ಅರಳಿದ ಕಮಲ ನೋಡಿ ಹಾಡಲು ಬೇಡಿ -
ಸುಗ್ಗಿ ಬಂದಿರಬಹುದು, ಹಿಗ್ಗದಿರಿ, ಗಮನಿಸದೆ ಸುಮ್ಮನಿರಿ!

ಬಟ್ಟೆ ತುದಿ ಇದ್ದೀತು, ಎನ್ನಬಾರದು 'ಅಂಚೆ';
ಕೆರೆಯ ಮೀನಿರಬಹುದು, 'ಬಾಳೆ' ಎಂದೀರಿ;
ಚಂದನದ ಮರ ಬೆಳೆದು 'ಆರಡಿ'ಯಾಯ್ತೆನ್ನದಿರಿ;
ಹುಡುಗಿಯುಡುಗೆಯ ತುದಿಗೆ 'ಸೋಗೆ'? ಅದು ಸಲ್ಲ;
'ನೆರೆ' ಎಂದು ಅಕ್ಕಪಕ್ಕವ ನೀವು ಸೂಚಿಸುವುದೇಕೆ?
ಚಪ್ಪರಿಸುವುದಿರಬಹುದು, 'ಹಣ್ಣು' ಎಂದೇಕೆ?
