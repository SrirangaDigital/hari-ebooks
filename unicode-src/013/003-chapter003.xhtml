೩ :: ಪತಿಸ್ ತುರೀಯೋಽಹಂ

1ಸೋಮೋ ದದದ್ ಗನ್ಧರ್ವಾಯ,
ಗನ್ಧರ್ವೋ ದದದ್ ಅಗ್ನಯೇ|
ರಯಿಂ ಚ ಪುತ್ರಾಂಶ್ ಚ ಅದಾದ್
ಅಗ್ನಿರ್ ಮಹ್ಯಂ ಅಥೋ ಇಮಾಮ್||

ಸೋಮಃ ಪ್ರಥಮೋ ವಿವಿದೇ,
ಗನ್ಧರ್ವೋ ವಿವಿದ ಉತ್ತರಃ |
ತೃತೀಯೋ ಅಗ್ನಿಷ್ಟೇ ಪತಿಸ್
ತುರೀಯೋ ಅಹಂ ಮನುಷ್ಯಜಾ||

-ಋಗ್ವೇದ ೧೦:೮೫:೪೧

೩ :: ವಧುವಿಗೆ ವರನ ಬೆಣ್ಣೆಮಾತು

ವಧು, ನೀನು ಬೆಳೆವಾಗ ಶುಚಿಗೆ ಆದ್ಯತೆಯಿತ್ತೆ-
ಹುಡುಗಿ, ನಿನ್ನನು 'ಸೋಮ' ವರಿಸಿದನು ಆಗ;

ದಿನ ಕಳೆದು, ದನಿಯೊಡೆದು ಕೊಟ್ಟೆ ಶೃಂಗಾರಕೆ ಮನವ-
ತರುಣಿ, ನಿನ್ನನಾವರಿಸಿದನು 'ಗಂಧರ್ವ' ಆಗ;

ಪುಟಿದು ಚೈತನ್ಯದ ಬೆಂಕಿ, ಪ್ರೀತಿ ಪ್ರೇಮದ ಸೆಳೆತ-
ಯುವತಿ, ನಿನ್ನ ಮೈಯುಂಡಾಯ್ತು 'ಅಗ್ನಿ' ಆಗ;

ಇದೋ 'ನರ'ನು ಬಂದಿರುವೆ, ನಿನ್ನೆದುರು ನಿಂದಿರುವೆ,
ಸರದಿಯಲಿ ನಾಲ್ಕರವ, ನೀ ಆ ಮೂವರ ಒಡವೆ-

ಆದರೂ ಪುಟವಿಟ್ಟ ಚಿನ್ನದ ಕೊಡುಗೆ, ಸ್ವೀಕರಿಸುವೆ!||
