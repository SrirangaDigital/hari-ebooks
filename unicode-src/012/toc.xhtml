    1. ೧ :: ದೇಶ-ವಿದೇಶಗಳಲ್ಲಿ ಗುಬ್ಬಿ
    2. ೨ :: ಪ್ರಾಣಿಶಾಸ್ತ್ರದಲ್ಲಿ ಗುಬ್ಬಚ್ಚಿ
    3. ೩ :: ಸಂಸ್ಕೃತ-ಕನ್ನಡದ ಪಕ್ಷಿಸಂಕುಲ
    4. ೪ :: ಇಂಗ್ಲಿಷ್ ಮತ್ತು ಇತರ ವಿದೇಶೀ ಸಾಹಿತ್ಯದಲ್ಲಿ ಗುಬ್ಬಚ್ಚಿ
    5. ೫ :: ಬರುವಿರಾ ಗುಬ್ಬಿಗಳೇ
    6. ೬ :: ಗುಬ್ಬಿಯ ಎರಡು ಕತೆಗಳು
      1. (ಅ) ಕಟಕಟಕ-ಕಿಟಕಿಟಕಿ
      2. (ಆ) ಕ್ರೈಸ್ತ ಜನಪದ ಸಾಹಿತ್ಯದಲ್ಲಿ ಗುಬ್ಬಚ್ಚಿಯ ಒಂದು ಕಥೆ
    7. ೭ :: ಗುಬ್ಬಿ ಪದ್ಯಗಳ ಬಗ್ಗೆ ಒಂದು ಅನಿಸಿಕೆ
    8. ೮ :: ಗುಬ್ಬಚ್ಚಿಯ ಬಗ್ಗೆ ಆಯ್ದ ಕೆಲವು ಕನ್ನಡ ಕವನಗಳು
    9. ೯ :: ಆಕರ ಮತ್ತು ಪರಾಮರ್ಶನ ಗ್ರಂಥಗಳು
    10. ೧೦ :: ಸಹೃದಯ ಸ್ಪಂದನ
      1. ಅಮೆರಿಕದಲ್ಲಿ ಕನ್ನಡದ ರಾಯಭಾರಿ:
      2. ಎಲ್ಲಕ್ಕೂ ಮಿಗಿಲಾದ ಕನ್ನಡಾಭಿಮಾನ:
      3. ಹದವಾಗಿ ಬರೆಯಬಲ್ಲ ಶಕ್ತಿಯುಳ್ಳವರು:
      4. ಸಂವೇದನಾಶೀಲ ಸಹೃದಯತಾಪೂರ್ಣ ಬರವಣಿಗೆ:
      5. ಕನ್ನಡದ ಜೀವಸ್ವರವನ್ನು ಅನಿವಾಸಿ ಭಾರತೀಯ-ಕನ್ನಡಿಗರಿಗೆ ಕೇಳಿಸಿದವರು:
      6. 'ರಾಜ್ಯೋತ್ಸವ ಪ್ರಶಸ್ತಿ' ಪುರಸ್ಕೃತ ಅಮೆರಿಕನ್ನಡಿಗ:
      7. ಹರಿ ಅಮೆರಿಕದಲ್ಲಿನ ಕನ್ನಡದ ಆಸ್ತಿ:
      8. ಕನ್ನಡದ 'ಹರಿ'ಕಾರ:
      9. ಅಮೆರಿಕನ್ನಡ ಚಿಂತನೆಗಳು:
