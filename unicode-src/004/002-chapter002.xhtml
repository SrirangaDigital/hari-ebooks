ವಿಶಿಷ್ಟಪದ ಕೋಶ: (ಕ್ಲಿಷ್ಟ ಪದಗಳಿಗೆ ಅರ್ಥ)

ಅಡಕ = ಒಳಸೇರಿಸುವುದು

ಇಂಡೆ = ಹಾರ, ಮಾಲೆ

ಇಕ್ಕರ = ಭೂಮಿ

ಈರಾಡಿ = ಜಗಳವಾಡಿ

ಉಸಾಬರಿ = ಗೊಡವೆ

ಕಕ್ಕಸ = ಕಷ್ಟ

ಕವಡ = ಕಪಟ, ಕೆಟ್ಟ

ಕಸ = ಹೊಲಸು

ಕಾಟ = ಉಪದ್ರವ

ಕಿರುಬು = ಒದರು

ಕೋತು = ಶಕ್ತಿ, ಬಲ, ತ್ರಾಣ

ಕೋರು = ಅರ್ಧಪಾಲು

ಗಸನೆ = ಕೂಡಲೆ

ಗುಳೇವು = ಊರು ಬಿಡುವಾಗ

ಗಂಟ= ತನಕ, ವರೆಗೆ

ಚರಗಿ = ತಂಬಿಗೆ

ಚಿಬ್ಬ = ಒಡಕ (ಚರ್ಮರೋಗ)

ಚೇರೆ = ಚಹರೆ, ಲಕ್ಷಣ

ಜಂಬುಳಿ = ಹಾಸಿಗೆ

ಜಲ್ಲೆಂದು = ಗಾಬರಿಯಾಗಿ

ಜೀತ = ದುಡಿತ

ಜೊಂಡು = ಹೊಲಸು, ಪಾಚಿ

ಜೋಲಿ = ಕಾಣಿ

ತಗತಿ = ವಾಗಾತಿ, ಶಿಸ್ತು

ತಟಕು = ಅಲ್ಪಸ್ವಲ್ಪ, ಹನಿ

ತಳ್ಳು = ದೂಡು

ತುರಸು = ಕೆರಕೆ

ತೇದುಂಡ = ಅನುಭವಿಸಿದ

ದಗದ = ಕೆಲಸ, ಬೊಗಸೆ

ದೂದಿ = ಬೆಂಕಿಯ ಹಿಡಿಯುವ

ಪೀಚು, ಪೀಚಿ = ಚಿಕ್ಕ, ಸಣ್ಣ

ಬರಸು = ಬಿರಸು

ಮಡ್ಡು = ಗುಂಗು, ತನ್ಮಯತೆ

ಮಾಜು = ಮಾಯೆ

ಮಾಟ = ಚಂದ, ಅಚ್ಚುಕಟ್ಟು

ಮಾನೆ = ವತನ, ಹೊಲ

ಮಾಪ = ಕ್ಷಮೆ

ಮಾಳ = ಬಯಲು

ಮುಕ್ಕು = ಸಣ್ಣ ಬೊಗಸಿ, ಮುಷ್ಟಿ, ತುಸು= ಸ್ವಲ್ಪ

ವಸಿ = ಸ್ವಲ್ಪ

ವಾಳಿ = ಹೀನ, ನೀಚ

ಸಾಪು = ನೆಲಸಮ

ಸುಂಕ = ರಾಶಿ

ಸೆಳಕು = ಮಳೆಯ ಸರು

ಹದ = ಪಕ್ವ ಸ್ಥಿತಿ

ಹಾಗು = ಹಿಡಿತ

ಹಿಕ್ಕಿ = ಬಾಚಿ, ತೀಡು

ಹುಂಡ = ಹೆಡ್ಡ, ತುಂಟ, ಉಡಾಳ

ಹುನ್ನಾರ = ಹಂಚಿಕೆ, ಸಂಚು

ಹುಲುಸು = ಹೆಚ್ಚಳ, ಸಮೃದ್ಧಿ

ಹೂಟಿ = ಎಳೆಯಲು

ಹೂಡುವಿಕೆ = ತೊಡಗಿಸುವಿಕೆ

ಹೇರು = ಭಾರ

ಹೊಲಗೆಲಸ = ಹೊಲದ ಕೆಲಸ

ಹೊಳು = ಬಯಲು
