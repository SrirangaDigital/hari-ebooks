೨೦ಬಿಡುಗಡೆ

೧೮೬ :: ಕೊನೆಗೆ ಕೊಂಡೊಯ್ಯಲು ಎಷ್ಟು ಕೂಡಿಟ್ಟೆ?

ಅನಿತ್ಯಾನಿ ಶರೀರಾಣಿ ವಿಭವೋ ನೈವ ಶಾಶ್ವತಃ।
ನಿತ್ಯಂ ಸನ್ನಿಹಿತೋ ಮೃತ್ಯುಃ, ಕರ್ತವ್ಯೋ ಧರ್ಮಸಂಗ್ರಹ॥186

-ಸಮಯೋಚಿತ ಪದ್ಯ ಮಾಲಿಕಾ

ಕಣ್ಸೆಳೆವ ಒಡಲು?
ಮುಂದೊಂದು ದಿನ ಅದು ಬೂದಿ, ಮಣ್ಣ ಮಡಿಲು;
ಐಸಿರಿಯ ಸಂಬೋಳಿ? ಈಗಿದ್ದು ಈಗಿಲ್ಲ ಬೊಬ್ಬುಳಿ, ಹುಸಿ ರಮ್ಯ;
ಕರೆಯದೆಯೆ ಬರುತಿದೆ ನೋಡು,
ಮೃತ್ಯು ಹತ್ತಿರಹತ್ತಿರ ನಿನ್ನತ್ತ ತೆವಳಿ-
ಹಿಡಿ ಸರಿದಾರಿ, ಪುಣ್ಯ ಎಷ್ಟು ಕೂಡಿಡಲಷ್ಟಷ್ಟು ಬಾಳು ಧನ್ಯ॥

೧೮೭ :: ಆತ್ಮವನು ಅನುಸರಿಸಿ ಹೋಗುವುದು, ಏನದು ಗೊತ್ತ?

ಮೃತಂ ಶರೀರಂ ಉತ್‍ಸೃಜ್ಯ ಕಾಷ್ಠ-ಲೋಷ್ಠ-ಸಮಂ ಕ್ಷಿತೌ।
ವಿಮುಖಾ ಬಾಂಧವಾಃ ಯಾಂತಿ ಧರ್ಮಸ್ ತಂ ಅನುಗಚ್ಛತಿ॥187

ಮನುಸ್ಮೃತಿ ೪:೨೪೧

ಪ್ರಾಣ ಹಾರಿತು; 'ಇನ್ನು ಈ ದೇಹ ಬರಿ ಮಣ್ಣು ಒಣಮರಕೇನೆ
ಸಮ' ವೆಂದು ನೆಂಟರಿಷ್ಟರು ಅದನಲ್ಲೆ ಸುಟ್ಟು ನಡೆದಾರು ಬಿಟ್ಟು;
ಒಂದಂತೂ ಆತ್ಮವನು ಅನುಸರಿಸಿ ಹೋಗುವುದು, ಏನದು ಗೊತ್ತೆ?
ಅದೆ ಧರ್ಮ; ಅದೆ ಗಳಿಸಿ ಉಳಿಸಿಟ್ಟ ಪುಣ್ಯಪಾಪದೊಂದು ಕಟ್ಟು!

೧೮೮ :: ದೀಪ ಆರುವ ಮುನ್ನ

ಗ್ರೀಷ್ಮಸ್ಯ ಉಷ್ಮಾ ವ್ರಜತಿ ಘನತಾಂ ನೂನಂ ಆಸನ್ ನ ವೃಷ್ಟೇರ್
ನೈಶಂ ಗಾಢೀ ಭವತಿ ತಿಮಿರಂ ಸನ್ನಿಕೃಷ್ಟ ಪ್ರಭಾತಮ್।
ಜನ್ತೋರ್ ಏವಂ ಪ್ರಸಭ-ವಿಭವ ಸ್ಫಾರಸಂಪತ್ ಪ್ರಚಾರಾನ್
ನಿಷ್ಕ್ರಾಮನ್ತೀ ವಿಪದ್ ಉಪಚಿತ ಉಪದ್ರವ ಉದ್ರೇಕಂ ಏತಿ॥188

ಕಲ್ಹಣ, ರಾಜತರಂಗಿಣೀ ೭:೭೯೫

ಹೆಚ್ಚುವುದು ಬೇಸಗೆಯ ಧಗೆ,
ಇನ್ನೇನು ಮಳೆ ಬಂತು ಎನ್ನುವ ಮುನ್ನ;
ಹೊತ್ತು ಇನ್ನೇನು ಹುಟ್ಟೀತೆನ್ನುವ
ಮೊದಲ ಕವಿದೀತು ಇರುಳುಗತ್ತಲೆ ದಟ್ಟ;
ಅದೃಷ್ಟ ಸೌಭಾಗ್ಯ ವೈಭವ ಉಜ್ವಲಿಸಿ
ಬಾಳಹಾದಿಯ ಬೆಳಗುವ ತುಸ ಮುನ್ನ
ಜಾರಿ ಕರಗುವ ಕಡು ಕಷ್ಟ ಕೋಟಲೆಯ
ಹೆಚ್ಚು ಹೆಚ್ಚಾದೀತು ಭಾರಿ ಕಾಟ.

೧೮೯ :: ಯಾರ ಕೇಳಿ ಅವರು ಹೋದರು?

ಶಿವಾಯ ಲೋಕಸ್ಯ, ಭವಾಯ ಭೂತಯೇ,
ಯ ಉತ್ತಮ-ಶ್ಲೋಕ-ಪರಾಯಣಾ ಜನಾಃ ।
ಜೀವನ್ತಿ ನ ಆತ್ಮಾರ್ಥಂ ಅಸೌ ಪರಾಶ್ರಯಂ,
ಮುಮೋಚ ನಿರ್ವಿದ್ಯ ಕುತ್‍ಃ ಕಳೇವರಮ್॥189

ಶ್ರೀಮದ್ ಭಾಗವತ ೧:೪:೧೨

ತಮಗಾಗಿ ಜೀವಿಸದೆ ಪರರಿಗೇ ಇದ್ದಂಥವರು ಇವರು,
ಎಲ್ಲರುನ್ನತಿಯನ್ನ ಮೂಲೋಕದೊಳಿತ ಬಯಸುತಿದ್ದವರು,
ಅನುದಿನವೂ ಮನಸಾರೆ ಹಾಡಿ ಈ ಹೊಗಳಬೇಕಾದವರು-
ಯಾರ ಕೇಳದೆ ದೇಹ ಬಿಟ್ಟು ಹೋದುದಾದರೂ ಹೇಗೆ?

೧೯೦ :: ಎಲ್ಲೆಲ್ಲೂ ಭಯವೇ ಭಯ

ಭೋಗೇ ರೋಗಭಯಂ, ಕುಲೇ ಚ್ಯುತಿಭಯಂ,
ವಿತ್ತೇ ನೃಪಾಲಾದ್ ಭಯಂ
ಮಾನೇ ದೈನ್ಯಭಯಂ, ಬಲೇ ರಿಪುಭಯಂ,
ರೂಪೇ ಜರಾಯಾ ಭಯಂ।
ಶಾಸ್ತ್ರೇ ವಾದಿಭಯಂ, ಗುಣೇ ಖಲಭಯಂ,
ಕಾಯೇ ಕೃತಾಂತಾದ್ ಭಯಂ
ಸರ್ವಂ ವಸ್ತು ಭಯಾನ್ವಿತಾಂ ಭುವಿ ನೃಣಾಂ
ವೈರಾಗ್ಯಂ ಏವ ಅಭಯಂ ॥190

ಭರ್ತೃಹರಿ, ವೈರಾಗ್ಯಶತಕ ೩೨

ಭೋಗದಲಿ ರೋಗಭಯ, ಕುಲದ ಕುಡಿ ಕೊನೆಗೊಳ್ಳುವಾತಂಕ,
ಗಳಿಕೆಗುಳಿಸಿಕೊಳ್ಳುವ ಬಿಗುರು, ಮಾನಕ್ಕೆ ದೈನ್ಯದಂಜಿಕೆ ಡಿಳ್ಳ,
ಮುಪ್ಪಿನದೆ ಕಳವಳ ರೂಪಕ್ಕೆ, ಸೋಲಿನ ಅಳುಕು ಬಲಕ್ಕೆ,
ಅಲ್ಪವಿದ್ಯೆಗೆ ವಾದದ್ದೇ ತಬ್ಬಿಬ್ಬು, ಖಳರ ಕಿರಿಕಿರಿ ಗಾಬರಿ ಗುಣಕ್ಕೆ,
ಸಾವು ದೇಹಕ್ಕಂಟಿದ ದಿಗಿಲು, ವೈರಾಗ್ಯವದೊಂದೆ ಅಭಯ ತಾನೆ?

೧೯೧ :: ಯಮ ನನ್ನ ಮುಟ್ಟಬೇಡ

ಮೃತ್ರ್ಯೋರ್ ಭಿಭೇಷಿ ಕಿಂ ಮೂಢ!,
ಭೀತಂ ಮುಂಚತಿ ಕಿಂ ಯಮಃ।
ಅಜಾತಂ ನೈವ ಗೃಹ್ಣಾತಿ, ಕುರು ಯತ್ನಂ, ಅಜನ್ಮನಿ॥191

ಸುಭಾಷಿತ ರತ್ನಭಾಂಡಾಗಾರ ೩೭೫:೨೩೧

ಮೃತ್ಯುವಿಗೆ ಭಯಪಡುವೆ, ಮೂಢ, ನೀ ಆಶಿಸುವೆ:
ಯಮ ನನ್ನ ಬಿಟ್ಟಾನು; ಪಾಪ!, ಹೆದರಿದನೆಂದು;
ಇದು ಗುಟ್ಟು: ಮುಟ್ಟನೋ ಆ ಹುಟ್ಟದವನನ್ನ ಯಮ,
ಮತ್ತೆಂದೂ ಹುಟ್ಟದ ಹಾಗೆ ನೀ ಬಾಳಿ ಬದುಕೋ!'

೧೯೨ :: ಯಮ ಇರುವುದೆಲ್ಲಿ?

ಯಮೋ ವೈವಸ್ವತೋ ರಾಜಾ ಯಸ್ ತವ ಏಷ ಹೃದಿ ಸ್ಥಿತಃ ।
ತೇನ ಚೇದ್ ಅವಿವಾದಸ್ ತೇ,
ಮಾ ಗಂಗಾಂ ಮಾ ಕುರೂನ್ ಗಮಃ ॥192

ಮನುಸ್ಮೃತಿ ೮:೯೨

ಪಾಪ-ಪುಣ್ಯಗಳನ್ನ ಒಂದು ಪಟ್ಟಿಮಾಡಿಟ್ಟಲ್ಲಿ ಅವ,
ಎಲ್ಲಿಯೋ ಮೇಲೆ ಕುಳಿತೆಲ್ಲವನು ನೋಡುತಿಹನೆಂದು-
ಏಕೆ ಕಸಿವಿಸಿ ಪಡುವೆ? ಅಲ್ಲಿಲ್ಲ, ಇಲ್ಲಿಯೇ ಇಲ್ಲಿಯೇ
ನಿನ್ನೆದೆಯ ಮೂಲೆಯಲಿಹ ಯಮನ ಸುಳಿವ ಕಾಣಹೋದೆಯಲ್ಲ;
ತೀರ್ಥವೋ ಕ್ಷೇತ್ರವೋ ಸುತ್ತಾಡಿ ಮಿಂದರೆ ಬಂತೆ?
ಕಳೆ-ಗಳಿಸಿದನೆಲ್ಲ, ಇದ್ದೆಡೆಯೇ ನೀನಿಂತು ಮಾಡಬಹುದಲ್ಲ!

೧೯೩ :: ಸೆರೆ-ತೊರೆ

ಮನ ಏವ ಮನುಷ್ಯಾಣಾಂ, ಕಾರಣಂ ಬಂಧ ಮೋಕ್ಷಯೋಃ ।
ಬಂಧಯಾ ವಿಷಯಾಸಂಗೀಂ,
ಮೋಕ್ಷೋ ನಿರ್ವಿಷಯಾ ಸ್ಮೃತಮ್॥193

ಮೈತ್ರೀ ಉಪನಿಷತ್ ೬:೩೪

ತೊಡಗಿ ಜೊತೆಗೂಡುವುದೂ, ಕಳಚಿಕೊಂಡೋಡುವುದೂ,
ಎರಡೂನು ಮನದಾಟ, ಒಂದು ಬಗೆ ವ್ಯಾಪಾರವಣ್ಣಾ;
ಬಂಧನಕು ಬಿಡುಗಡೆಗೂ, ಮನೆವೊಂದೆ ಕಾರಣ, ಜಾಣ-
ಅಂಟಿಕೊಂಡಿರೆ ಸೆರೆ, ತಗಲದಲಿರದೇನೆ ಮೋಕ್ಷ, ಕಾಣ!

೧೯೪ :: ಬಿಡುಗಡೆಯ ಸೆರೆಮನೆಯ ಕಾಯುತಿಹರಾರು?

ಮೋಕ್ಷದ್ವಾರೇ ದ್ವಾರಪಾಲಾಃ, ಚತ್ವಾರಃ ಪರಿಕೀರ್ತಿತಾಃ ।
ಶಮೋ, ವಿಚಾರಸ್, ಸಂತೋಷಃ ಚತುರ್ಥ ಸಾಧು-ಸಂಗಮಃ॥194

ಯೋಗವಾಸಿಷ್ಠ ೨:೩೮

ಬಿಡುಗಡೆಯ ಅರಮನೆಯೊಳಗೆ ಹೋಗಲಪ್ಪಣೆ ಬೇಕು
ಭಟರ ಕಣ್ಣಗಾವಲು ಪಹರೆ, ಪ್ರತಿ ಬಾಗಿಲಲಿ ತುಂಬ ಬಿಗಿ;
ಸಾಧುಜನ ಸಹವಾಸ, ಶಾಂತಿ, ಸಂತಸ, ವಿಚಾರ-
ಚಿಂತನೆಯೇ ಆ ನಾಲ್ಕು ಉಕ್ಕಡದ ಜನ, ಇದೆ ಪರವಾನಿಗಿ?

೧೯೫ :: ಹಸಿ ಸುಳ್ಳು, ಮಾತಿನ ಜೊಳ್ಳು

ಮೃತೈಃ ಪ್ರಾಪ್ಯ ಸ್ವರ್ಗೋ ಯದ್ ಇಹ ಕಥಯತಿ
ಏತದ್ ಅನೃತಮ್।
ಪರೋಕ್ಷೋ ನ ಸ್ವರ್ಗೋ,
ಬಹುಗುಣಂ ಇಹ ಏವ ಏಷ ಫ಼ಲತಿ॥195

ಭಾಸ, ಪಾಂಚರಾತ್ರ ೧:೨೩

'ಮಾಡಿದೊಳ್ಳೆಯದಕ್ಕೆ ಸತ್ತು, ಸ್ವರ್ಗಕೆ ಹೋದ'-'
ಎಂಬ ಮಾತೇನಿಹುದು, ಅದು ಹಸಿಯ ಸುಳ್ಳು!
ಸ್ವರ್ಗ-ನರಕಗಳೆಲ್ಲ ಅಲ್ಲೆಲ್ಲೋ ಬೇರೆಡೆ ಇಲ್ಲ
-ಎಲ್ಲ ಫಲಿಸುವುದಿಲ್ಲೇ, ಬಿಡು ಮಾತು ಪೊಳ್ಳು!

೧೯೬ :: ಸಂತಸದಿ ನಲಿದಾಡು

ಯಾವತ್ ಜೀವೇತ್ ಸುಖಂ ಜೀವೇತ್,
ಋಣಂ ಕೃತ್ವಾ ಘೃತಂ ಪಿಬ।
ಭಸ್ಮೀಭೂತಸ್ಯ ದೇಹಸ್ಯ
ಪುನರ್ ಆಗಮನಂ ಕುತಃ? ॥196

ಚಾರ್ವಾಕ ದರ್ಶನ, ಸರ್ವದರ್ಶನ ಸಂಗ್ರಹ ೧:೪

ಬಾಳು! ಕಳೆದುದು ಸಿಗದು! ಸಂತಸದಿ ಓಲಾಡು!
ಸುಖವ ಅನುಭವಿಸುತ್ತ ಕೊನೆಯುಸಿರ ಹಾಡು!
ಸಾಲವಾದರೂ ಮಾಡು, ತಿಂದುಂಡು ನಲಿದಾಡು!
ಮತ್ತೆ ಬಂದೀತೆಲ್ಲಿಂದ ಸುಟ್ಟ ಮೇಲೀ ಗೂಡು!!

೧೯೭ :: ಬಿಡುಗಡೆಯ ಪಡೆಯಲೆನೆ

ಮುಕ್ತಿಂ ಇಚ್ಛಸಿ ಚೇತ್, ತಾತ, ವಿಷಯಾನ್ ವಿಷವತ್ ತ್ಯಜ!
ಕ್ಷಮಾ ಆರ್ಜವ ದಯಾ ಶೌಚಂ ಸತ್ಯಂ ಪಿಯೂಷವತ್ ಪಿಬ॥197

ಚಾಣಕ್ಯ ನೀತಿ ೯:೧

ಮಗು, ನಿನಗೆ ಬಿಡುಗಡೆಯ ಪಡೆಯಬೇಕೆಂದಿದೆ ಆಸೆ?
ಬಿಡು ವಿಷಯ ಸುಖಗಳನ್ನ ವಿಷವೆಂದು ಬಗೆದು;
ಕ್ಷಮೆ ಸರಳ ನಡತೆಗಳೇ, ದಯೆ ಶುಚಿಯೇ ಸತ್ಯವೇ
ನನ್ನ ಪಾಲಿನ ಅಮೃತ- ಎಂದು ತಿಳಿದು, ಕುಡಿದು.
