೮. ಪೌರುಷ

೫೩ :: ಎತ್ತಿದನು ಹತ್ತಾವತಾರಗಳ!

ಸೃಷ್ಟ್ವಾ ಪುರಾಣೇ ವಿವಿಧಾನಿ ಅಜಯಾತ್ಮಶಕ್ತ್ಯಾ
ವೃಕ್ಷಾನ್ ಸರೀಸೃಪ ಪಶೂನ್ ಖಗ-ದಂಶ-ಮತ್ಸ್ಯಾನ್
ತೈಃ ತೈಃ ಅತುಷ್ಟ ಹೃದಯಃ, ಮನುಜಂ ವಿಧಾಯ,
ಬ್ರಹ್ಮ ಅವಬೋಧ ಧಿಷಣಂ, ಮುದಂ ಆಪ ದೇವಃ॥053

ಭಾಗವತ

ಹಲವು ಬಗೆಯ ಮನೆಗಳನವ ತಾನಿರಲು ನಿರ್ಮಿಸಿಕೊಂಡ-
ಖನಿಜ, ಗಿಡ, ಮರ, ಮೀನು, ಹುಳು, ಹಾವು, ಹಕ್ಕಿ,
ಏನೇನೋ ಪ್ರಾಣಿಗಳು; ಆದರೂ, ದೊರಕದೆ ಸಾಕಷ್ಟು ತೃಪ್ತಿ-
ಸೃಜಿಸಿದನಾಗ ಮಾನವನ; ಹರಿಯಿತು ಆನಂದ ಉಕ್ಕಿ.

೫೪ :: ಬೇರೆ ಯಾರೇತಕೆ ಬೇಕು ಆಸನವನೇರಲಿಕೆ

ನ ಅಭಿಷೇಕೋ ನ ಸಂಸ್ಕಾರಃ ಸಿಂಹಸ್ಯ ಕ್ರಿಯತೇ ವನೇ।
ವಿಕ್ರಮ-ಅರ್ಜಿತ-ಸತ್ತ್ವಸ್ಯ ಸ್ವಯಂ ಏವ ಮೃಗೇಂದ್ರತಾ॥054

ಹಿತೋಪದೇಶ, ಸುಹೃದ್‍ಭೇದ ೧೬

ಬೇರೆ ಯಾರೋ ಅವಗೆ ಪಟ್ಟಕಟ್ಟಿದ್ದಲ್ಲ;
ಇನ್ನು ಯಾರೋ ಬೇರೆ ಕಿರೀಟವಿಟ್ಟಿದ್ದಲ್ಲ;
ತನ್ನ ತೋಳ್ಬಲದಿಂದ ತಾನೇ ಆಗಿಲ್ಲವೇನಯ್ಯ
ಸಿಂಹ ಕಾಡಿನ ರಾಜ, ಭೀಮಸಾಹಸತೇಜ?
ಬೇರೆ ಯಾರೇತಕೆ ಬೇಕು ಆಸನವನೇರಲಿಕೆ?
ಹೆಪ್ಪುಗಟ್ಟಿದೆ ಕಸುವು, ಎದೆಗಾರಿಕೆಯ ಬೀಜ!

೫೫ :: ಮಳೆ ಬಂದರೂ ಮೂಡದಿರಬಹುದು ಕಾಮನ ಬಿಲ್ಲು

ಉದ್ಯೋಗಿನಂ ಪುರುಷಸಿಂಹಂ ಉಪೈತಿ ಲಕ್ಷ್ಮೀಃ,
ದೈವಂ ನ ದೇಯಂ ಇತಿ ಕಾಪುರುಷಾಃ ವದನ್ತಿ।
ದೈವಂ ನಿಹತ್ಯ, ಕುರು ಪೌರುಷಂ ಆತ್ಮಶಕ್ತ್ಯಾ-
ಯತ್ನೇ ಕೃತೇ ಯದಿ ನ ಸಿದ್ಧ್ಯತಿ, ಕೋಽತ್ರ ದೋಷಃ?055

ಹಿತೋಪದೇಶ ೧:೨೨

ಅರಸಿ ತಾ ಬಂದಾಳು ಸಿರಿಯರಸಿ, ಕೆಲಸಗಾರನ ಬಲು ಗಟ್ಟಿಗನ;
'ದೈವ ಮೋಸವೆಸಗಿತು' - ಬಿಸುಗೈಯುವುದು ಹೇಡಿ ಮನಸು;
ಬಿಡು ವಿಧಿಯ ಹುಸಿಮಾತ, ತೊಡು, ತೋರು ಎದೆಗಾರಿಕೆಯ;
ಸಿದ್ಧಿಸದೆ ಹೋಯಿತೆ? ಪಟ್ಟ ಪ್ರಯತ್ನದಲೆ ತಣಿದು ಸುಖಿಸು!

೫೬ :: ಬರೀ ಮಂತ್ರಿಸಿದರೆ ಮಾವಿನಕಾಯಿ ಉದುರದು

ಯೇಷಾಂ ಬಾಹುಬಲಂ ನಾಸ್ತಿ,
ಯೇಷಾಂ ನಾಸ್ತಿ ಮನೋಬಲಮ್।
ತೇಷಾಂ ಚಂದ್ರಬಲಂ ಏವ,
ಕಿಂ ಕುರ್ಯಾದ್ ಅಂಬರಸ್ಥಿತಮ್॥056

ಸೋಮದೇವ (ಕಾಲ ೯೫೯), ಯಶಸ್ತಿಲಕ ಚಂಪೂ ೩:೫೪

ಬಾಹುಬಲವಿಲ್ಲದಗೆ, ಗಟ್ಟಿಮನಸೇ ಇಲ್ಲದ ಜಡಗೆ,
ಏನೂ ಮಾಡದೆ ನಿಟ್ಟುಸಿರ ಬಿಡುತ ಕಾಲ ತಳ್ಳುವವಗೆ,
ಹೇಳಿ- 'ಕೊರಗದಿರು, ತೆಗೆ; ಹೊಳೆವುದಿಲ್ಲವೆ ತಲೆಗೆ-
ಮಿನುಗು ತಾರೆಯೆ ತುಂಬುವುದೆ, ಛೆ, ಧೈರ್ಯ ಎದೆಗೆ'?

೫೭ :: ನಟರಿಗೇ ಚಪ್ಪಾಳೆ, ಪರದೆ ಎಳೆಯುವರಿಗಲ್ಲ

ರಥಸ್ಯೇಕಂ ಚಕ್ರಂ ಭುಜಗಯಮಿತಾ ಸಪ್ತತುರಗಾಃ
ನಿರಾಲಂಬೋ ಮಾರ್ಗರ್, ಚರಣರಹಿತಃ ಸಾರಥಿರ್ ಅಪಿ।
ರವಿರ್ ಗಚ್ಛತಿ ಅನ್ತಂ ಪ್ರತಿದಿನಂ ಅಪಾರಸ್ಯ ನಭಸಃ
ಕ್ರಿಯಾಸಿದ್ಧಿರ್ ಭವತಿ ಮಹತಾಂ ನ ಉಪಕರಣೇ॥057

ಬಲ್ಲಾಳ ಸೇನ, ಭೋಜಪ್ರಬಂಧ ೧೬೯

ರಥಕ್ಕೊಂದೇ ಚಕ್ರ, ಎತ್ತೆತ್ತಲೊ ಎಳೆಯುತಿವೆ ಕುದುರೆಗಳೇಳು ನಿತ್ಯ
ತಳವೇ ಇಲ್ಲದ ಜಾಡು, ಸಾರಥಿಯೋ, ಪಾಪ, ಕಾಲಿಲ್ಲ, ಹೆಳವ;
ಅನುದಿನವೂ ಸುತ್ತಿ ಬರುವ ದಿಗಂತ ಇತ್ತಣಿಂದತ್ತ ಆದಿತ್ಯ-
ಕೆಲಸ ಕೈಗೂಡಿತೆನೆ ನೆರವಿನಿಂದಲ್ಲ; ತನ್ನದೇ ಕೆಚ್ಚು ಕಸುವಿನಿಂದ!

೫೮ :: ಚೆನ್ನೆಯರ ಬಲೆಯೊಂದು ಭಾವಜನು ಬೀಸಿದ ಬೇಂಟೆ

ಧನುಃ ಪೌಷ್ಪಂ, ಮೌರ್ವೀ ಮಧುಕರಮಯೀ ಚಂಚಲದೃಶಾಂ
ದೃಶಾಂ ಕೋಣೋ ಬಾಣಃ, ಸುಹೃದ್ ಅಪಿ ಜಡಾತ್ಮಾ ಹಿಮಕರಃ
ತಥಾಪಿ, ಏಕೋ ಅನಂಗಸ್ ತ್ರಿಭುವಂ ಅಪಿ ವ್ಯಾಕುಲಯತಿ
ಕ್ರಿಯಾಸಿದ್ಧಿಃ ಸತ್ತ್ವೇ ಭವತಿ ಮಹತಾಂ ನ ಉಪಕರಣೇ58

-ಬಲ್ಲಾಳ ಸೇನ, ಭೋಜಪ್ರಬಂಧ ೧೭೧

ಹೂಗಳ ಬಿಲ್ಲು, ಸಿಡಿಮಿಡಿವ ತುಂಬಿಗಳು ಮುತ್ತಿಡುವ ಹೆದೆ ರಂಗ;
ಕುಡಿನೋಟಗಳೆ ಬಾಣ, ನೀರ ಹೂ ಬಂಧುರ ಚಂದಿರನ ಅನುಬಂಧ;
ಏನಿರಲಿ ಮೂಲೋಕವನೆ ತಲ್ಲಣಿಸಿ, ಕುಣಿಸಿ, ತಣಿಸುವ ಅನಂಗ-
ಕೆಲಸ ಕೈಗೂಡಿತೆನೆ ನೆರವಿನಿಂದಲ್ಲ; ತನ್ನದೇ ಕೆಚ್ಚು ಕಸುವಿನಿಂದ!

೫೯ :: ದಿನವೂ ಸಾಯುವುದಕ್ಕಿಂತ ವೀರಮರಣವೇ ಲೇಸು

ವರಂ ತೇಜಸ್ವಿನೋ ಮೃತ್ಯುಃ, ನ ತು ಮಾನಾವಖಂಡನಂ।
ಮೃತ್ಯುಃ ತತ್ ಕ್ಷಣಿಕೋ ದುಃಖಂ, ಮಾನಭಂಗೋ ದಿನೇ ದಿನೇ॥59

ಶಾಙ್ರ್ಗಧರ ಪದ್ಧತಿ ೧೭೪೩

ಬಂದೆರಗೆ ಸಾವಿನ ಹದ್ದು, ಹೊಡೆತ ತಂದೀತೊಮ್ಮೆ ನೋವು;
ಅಡಿಗಡಿಯ ಮಾನಭಂಗಕ್ಕಿಂತ ಸಾವೇ ಲೇಸು, ಅಲ್ಲವೇನ?
ಕ್ಷಣಕಾಲ ನುಂಗಿ ಸಹಿಸಿ ಅಸು ನೀಗೆ ಮುಗಿಯಿತದರ ಕಥೆ,
ಅವಮಾನ ತಾಳಿ, ಸಾಹಸಿ ಅನುದಿನ ಸಾವ ಬಯಸುತಾನ?

೬೦ :: ಸಾಹಸಿಯ ನೆರಳಾಗಿ ವಿಪತ್ತು ಬೆಂಬಿಡದು

ತೇಜಸ್ವಿನೋ ಹಿ ಪುರುಷಸ್ಯ ಸಂಪದ್ ಉದ್ಯೋತನ ಪ್ರತಿ-
ಪಕ್ಷಭೂತಾ ವಿಪದಪಿ ನ ಚ್ಛಾಯೇವ ಪರಿಹರತಿ ಪಾಶ್ರ್ವಮ್॥60

ವಿಜ್ಝಿಕಾ, ಕೌಮುದೀ ಮಹೋತ್ಸವ

ತೇಜಸ್ವಿಯಾದವಗು ಆಪತ್ತಿನರಸಿ ಅ ಸರಸಿಗೂ
ಬಿಡಿಸಲಾಗದ ಗುಟ್ಟು, ಏನೋ ಗಟ್ಟಿ ನೆಂಟು;
ಇತ್ತ ಬೆಳಗಿ ಸಂಪತ್ತ ಪ್ರಜ್ವಲಗೊಳಿಸುತಿರೆ, ನಗುತ
ವಿಪತ್ತೋ ನೆರಳಾಗಿ ಪಕ್ಕದಲತ್ತ ನಿಲುವುದುಂಟು.

೬೧ :: ಆಸೆ ಎಂಬ ಬಿಸಿಲುಗುದುರೆಗೇಕೆ ದಾಸನಾದೆ?

ಅಕೃತಸ್ಯ ಆಗಮೋ ನಾಸ್ತಿ,
ಕೃತೇ ನಾಶೋ ನ ವಿದ್ಯತೇ।
ಅಕಾಮಾದ್(?) ಏವ ಲೋಕೋಽಯಂ,
ತೃಷ್ಣೆ, ದಾಸೀ ಕೃತಸ್ ತ್ವಯಾ॥61

-ಜಲ್ಹಣ, ಸುಭಾಷಿತ ಸೂಕ್ತಿಮುಕ್ತಾವಳಿ

ಏನೂ ಮಾಡದೆ ಹಾಗೇ ಇರಲು ಬಾರದು ಏನೂ;
ಪಡಬೇಕು ಶ್ರಮ, ಎಂದೂ ತಾರದದು ನಷ್ಟವಷ್ಟೆ;
ಇಷ್ಟವಿರಲಿ ಬಿಡಲಿ, ಹೀಗೇಕೆ ಮತ್ತೆ, ಓ ಆಸೆ, ಏಕೆ,
ಜನರೆಲ್ಲರನು ಆಳಾಗಿ ಮಾಡಿಕೊಂಡೆ, ಎ¯ಗೆ ದುಷ್ಟೆ?

೬೨ :: ನಮ್ಮ ನಡುವಿನ ರಾಕ್ಷಸರು

ಸಾಕ್ಷರ ವಿಪರೀತಶ್ ಚೇತ್ ರಾಕ್ಷಸಾ ಏವ ಕೇವಲಂ।
ಸರಸೋ ವಿಪರೀತಶ್ ಚೇತ್ ಸರಸತ್ವಂ ನ ಮುಂಚತಿ॥62

ಸುಭಾಷಿತ ರತ್ನಭಾಂಡಾಗಾರ ೧೫೮:೨೦೯

ಓದಿಬರೆದವರವರು, ಬುದ್ಧಿಜೀವಿಗಳು ಸಾಕ್ಷರ ರು;
ತಿರುಗುಮುರುಗಾಯಿತೋ ಅವರೇ ರಾಕ್ಷಸ ರು, ಕೂಡ.
ಹಾಗಲ್ಲ, ಬದಲಾಗುವುದಿಲ್ಲ ಸರಸ ಜನ ಸಾತ್ವಿಕರು
ಹಿಂದುಮುಂದಾದರೂ ಒಂದೇ ಎಂದೆಂದೂ, ನೋಡ!

೬೩ :: ತನ್ನಿ ಚಿನ್ನದ ಹೂವ ನೆಲದೊಳಗಣ ಬಳ್ಳಿಯಿಂದ ಕಿತ್ತು

ಸುವರ್ಣಪುಷ್ಪಾಂ ಪೃಥಿವೀಂ ಚಿನ್ವನ್ತಿ ಪುರುಷಾಸ್ ತ್ರಯಃ।
ಶೂರಶ್ ಚ, ಕೃತವಿದ್ಯಶ್ ಚ, ಯಶ್ ಚ ಜಾನಾತಿ ಸೇವಿತುಮ್॥63

ಮಹಾಭಾರತ, ಉದ್ಯೋಗ ಪರ್ವ ೭೫:೭೪

ನೆಲ ಬಗೆದು, ಅಡಗಿಹ ಆಳದ ಬೆಡಗಿನ ಬಳ್ಳಿಯಿಂದ
ಹೊರತೆಗೆಯಲಾದೀತೆ ಆಯ್ದು ಚಿನ್ನದ ಹೂವ ಬಿಡಿಸಿ?
ಹೌದು, ಮೂವರು ಮಾತ್ರ ಮಾಡಬಲ್ಲರೀ ಸಾಹಸವ-
ಶೂರ, ಕಡು ಜಾಣ, ಸೇವೆಯಲಿ ನುರಿತ ಶ್ರಮಿಕ ಸರಸಿ.
