೧೩ಹೆಣ್ಣು-ಗಂಡು

೯೨ :: ಚಲುವೆ

ಪತ್ನೀಂ ಮನೋರಮಾಂ ದೇಹಿ, ಮನೋಚಿತ್ತಾನುಸಾರಿಣೀಂ।
ತಾರಿಣೀಂ ದುರ್ಗಸಂಸಾರ-ಸಾಗರಸ್ಯ ಕುಲೋದ್ಭವಾಮ್॥092

-ಮಾರ್ಕಂಡೇಯ ಪುರಾಣ, ದೇವಿಮಹಾತ್ಮ್ಯ

ಮನಕೆ ಮುದವನು ಕೊಡುವ ಮಡದಿಯನೆ ನೀ ತೋರು,
ನನ್ನ ಇಂಗಿತವರಿತು ನಡೆವವಳು ಇರಲಿ ಅವಳು;
ನೋಡಿ ಕೇಳಿ ಕಲಿವ ಒಳಿತು ಪರಿಸರದಿ ಹುಟ್ಟಿ ಬೆಳದಿರಲಿ,
ಬಾಳಸಂಗಾತಿಗಳಾಗಿ ಜೊತೆ ನಡೆಯೆ ನಮಗೆ ದಡ ಕಾಣಲಿ.

೯೩ :: ತುಂಬಿದ ಕೊಡದ ಬೆಡಗಿ, ತುಡುಗಿ!

ಇಯಂ ಸುಸ್ತನೀ ಮಸ್ತಕ-ನ್ಯಸ್ತ-ಕುಂಭಾ!
ಕುಸುಂಭಾರುಣಂ ಚಾರು ಚೈಲಂ ವಸಾನಾ।
ಸಮಸ್ತಸ್ಯ ಲೋಕಸ್ಯ ಚೇತಃ ಪ್ರವೃತ್ತಿಂ।
ಗೃಹೀತ್ವಾ ಘಟೇ ನ್ಯಸ್ತ ಯಾತೀವ ಭಾತಿ॥093

ಜಗನ್ನಾಥ, ಭಾಮಿನೀ ವಿಲಾಸ

ತುಂಬಿದೆದೆಯ ಲವಂಗಿ ನಸುಗೆಂಪು ಪತ್ತಲವ
ಉಟ್ಟು ಹೊರಟಿಹಳೊಂದು ಕೊಡವ ಹೊತ್ತು;
ಜಗದ ಮನಸನ್ನೆಲ್ಲ ಸೆಳೆದು ಕೊಡದಲಿ ತುಂಬಿ
ಕೊಂಡೊಯ್ಯುತಿಹಳೇನೋ ಎನಿಸಿತೀ ಹೊತ್ತು!

೯೪ :: ಜಿಂಕೆಗಣ್ಣಿನ ಹುಡುಗಿ

ನ ಯಾಚೇ ಗಜಾಲೀಂ, ನ ವಾ ವಾಜಿರಾಜಿಂ,
ನ ವಿತ್ತೇಷು ಚಿತ್ತಂ ಮದೀಯೇ ಕದಾಚಿತ್।
ಇಯಂ ಸುಸ್ತನೀ ಮಸ್ತಕ-ನ್ಯಸ್ತ-ಕುಂಭಾ।
ಲವಂಗೀ ಕುರಂಗೀ-ದೃಗ್ ಅಂಗೀಕರೋತು॥094

ಜಗನ್ನಾಥ, ಭಾಮಿನೀ ವಿಲಾಸ'

ಆನೆಗಳ ಪಡೆ ಬೇಡ, ಕುದುರೆಗಳ ದಂಡೂ ಬೇಡ;
ಧನಕನಕದತ್ತ ಸ್ವಲ್ಪವೂ ನನ್ನ ಮನ ಹರಿಯದು.
ತನ್ನದೇ ಕಲಶಗಳ ತುಂಬು ಭಾರವ ಮರೆತು,
ತಲೆಯ ಮೇಲಿನ್ನೊಂದು ಕೊಡವ ಹೊತ್ತು,
(ಕಂಕುಳಲಿ ಮತ್ತೊಂದಿಟ್ಟು, ಬಳುಕಿ ನಡೆವ ಲತಾಂಗಿ)
ಕುರಂಗನಯನೆ ಲವಂಗಿಯ ಸಂಗ ನಾ ಬಯಸುವೆ;
(ಆ ಅನಂಗನ ರಾಣಿ) ನನ್ನನಂಗೀಕರಿಸಲಿ'!

೯೫ :: ಗೌರಿಯ ಪಣ

ತಥಾ ಸಮಕ್ಷಂ ದಹತಾ ಮನೋಭವಂ ಪಿನಾಕಿನಾ,
ಭಗ್ನ ಮನೋರಥಾ ಸತೀ।
ನಿನಿಂದ ರೂಪಂ ಹೃದಯೇನ ಪಾರ್ವತಿ,
ಪ್ರಿಯೇಷು ಸೌಭಾಗ್ಯಫಲಾ ಹಿ ಚಾರುತಾ॥095

ಕಾಳಿದಾಸ, ಕುಮಾರಸಂಭವ ೫:೧

ತನ್ನ ಮನದರಕೆಗಳೆನೆಲ್ಲ ತನ್ನೆದುರೇ ಸುಟ್ಟನೇ,
ಹುಟ್ಟಡಗಿಸಿದನೆ ಈ ಉರಿಗಣ್ಣ- ಎನುತ ಪರಿತಪಿಸಿ,
ಬಲು ಚೆಲುವೆ ತಾನೆಂಬ ಹಮ್ಮು ಇದ್ದುದು ಸಹಜ;
ಮನಸಾರೆ ಅದ ಹಳಿದು ಈಗ ಹೊರನಡೆದಳು.
ತನ್ನೊಲವು ಆ ಎಲ್ಲೆ ಮೀರಿ ಇರುವುದ ತೋರಿ,
ಶಿವನ ಗೆಲ್ಲುವೆನೆಂದು ಗೌರಿ ಪಣ ತೊಟ್ಟಳು!

೯೬ :: ಸುಟ್ಟರೂ ಮತ್ತೆ ಹುಟ್ಟಿದವ

ದೃಶಾ ದಗ್ಧಂ ಮನಸಿಜಂ ಜೀವಯನ್ತಿ ದೃಶೈವ ಯಾಃ।
ವಿರೂಪಾಕ್ಷಸ್ಯ ಜಯನ್ತೀಸ್ ತಾಸ್ ಸ್ತುವೇ ವಾಮಲೋಚನಾಃ॥096

ಕಾಶ್ಮೀರದ ವಲ್ಲಭದೇವ (ಸಂ.), ಸುಭಾಷಿತಾವಳಿ

ಮುಕ್ಕಣ್ಣ ಉರಿಗಣ್ಣ ಬಿಟ್ಟು ಸುಟ್ಟಿರಬಹುದು
ಮತ್ತೆ ಮತ್ತೆ ಹುಟ್ಟಿಸುವರಲ್ಲ ಮನಸಿಜನ ಇವರು;
ಚೆಲುವ ಕಂಗಳ ನೋಟ ಬೀರಿ ಉಸಿರೂಡಿಸುವವರ
ಹಣೆಗಣ್ಣನವನನೇ ಗೆಲಿದವರ ನಾ ಕೊಂಡಾಡುವೆ !

೯೭ :: ಹೇಗೆ ನಡೆದರೂ ಸೋಲೇ

ಸಾ ದೃಷ್ಟಾ ಯೈರ್ ನ ವಾ ದೃಷ್ಟಾ
ಮುಷಿತಾ ಸಮಮ್ ಏವ ತೇ।
ಹೃದಯಂ ಹೃತಂ ಏಕೇಷಾಂ,
ಅನ್ಯೇಷಾಂ ಚಕ್ಷುಷೋಽಫಲಮ್॥097

ಪ್ರಭಾಕರ ದೇವ; ಶಾಙ್ರ್ಗಧರ ಪದ್ಧತಿ

ನಿಜ, ನನ್ನ ಈ ಮಾತ ನಂಬಿ, ಸ್ವಲ್ಪ;
ಅವಳ ವಿಚಾರವೇ ಹಾಗೆ, (ಪಗಡೆಯಾಟದ ಕೊನೆಗೆ.
ಹೇಗೆ ಪಡೆ ನಡೆಸಿದರೂ ರಾಜ ಕುತ್ತನೆದುರಿಸಿದಂತೆ);
ನೋಡಿದವರೂ ಅಷ್ಟೆ, ಅವಳ ನೋಡದವರೂ ಅಷ್ಟೆ-
ಇಬ್ಬರೂ ಮೋಸ ಹೋದರು, ಕೊನೆಗೆ ಕೊನೆಗೇ!
ನೋಡಿದವರೆಲ್ಲರ ಹೃದಯ ಆಗಿ ಹೋಯಿತು ಲೂಟಿ;
ನೋಡದಿದ್ದವರ ಪಾಡು ಬಿಡಿ, ಅಳಬೇಕೋ ಬೇಡವೋ,
ಆ ಕಣ್ಣಿದ್ದೂ ಕುರುಡಾದವರ ಕಹಿಕತೆಗಿಹುದೆ ಸಾಟಿ?

೯೮ :: ಭ್ರಮೆ ಇರಲಿ ಚಿರಕಾಲ

ಯದ್ ಏವ ರೋಚತೇ ಮಹ್ಯಂ,
ತದ್ ಏವ ಕುರುತೇ ಪ್ರಿಯಾ॥
ಇತಿ ವೇತ್ತಿ; ನ ಜಾನಾತಿ-
ತತ್ ಪ್ರಿಯಂ ಯತ್ ಕರೋತಿ ಸಾ॥098

ಸರಸ್ವತೀಕಂಠಾಭರಣ ೫:೭೪

ಏನು ಬೇಕೋ ನನಗೆ, ಅದನೇ ಮಾಡುವಳು, ಭೇಷ್-
ಸಂತಸ ಪಡುವ ಪೆದ್ದ ಗಂಡೇ, ಗೊತ್ತೇ ನಿನಗೆ?
ಅವಳೇನ ಮಾಡುವಳೋ ಅದನೆಲ್ಲ ಮೆಚ್ಚಿರುವೆ
ಹೀಗಾಗಿ ಹೋಗಿರಲು ಎಲ್ಲಿದೆ ಮದ್ದು ನಿನ್ನ ಭ್ರಮೆಗೆ?

೯೯ :: ಸೋತು ಗೆದ್ದವರಾರು?

ಅಕ್ಷಮೇವ ಪಣೀಕೃತೇ ಅಧರೇ ಕಾಂತಯೋರ್
ಜಯ-ಪರಾಜಯೇ ಸತಿ।
ಅತ್ರ ವಕ್ತುಂ ಯದಿ ವೇತ್ತಿ ಮನ್ಮಥಃ
ಕಸ್ ತಯೋರ್ ಜಯತಿ ಜೀಯತೇಽಪಿ ವಾ॥099

ವಲ್ಲಭದೇವ, ಸುಭಾಷಿತಾವಳಿ

ಸ್ಪರ್ಧೆಯಲಿ ಆಟಕ್ಕೆ ತುಟಿಯನ್ನೇ ಪಣವಾಗಿ
ಇರಿಸಿ ಕುಳಿತರು ಪಟ್ಟಾಗಿ ರಸಿಕ ನಲ್ಲೆ ನಲ್ಲ;
ಸೋತದ್ದು, ಗೆದ್ದದ್ದು ಯಾರು ಯಾರೆಂದರೆ-
ಯಾರಿಗೆ ತಾನೆ ಗೊತ್ತು, ಆ ಕಾಮನೇ ಬಲ್ಲ!

೧೦೦ :: ಮೋಡಿಮಾಡುವ ಆ ಕಮ್ಮಗೋಲ!

ಅನಂಗೇನ ಅಬಲಾಸಂಗಾಜ್ ಜಿತಾ ಯೇನ ಜಗತ್ರಯೀ।
ಸಾ ಚಿತ್ರಚರಿತಃ ಕಾಮಃ ಸರ್ವಕಾಮಪ್ರದೋ ಅಸ್ತು ವಃ॥100

ಕ್ಷೇಮೇಂದ್ರ, ಕಲಾವಿಲಾಸ

ದೇಹವಿಲ್ಲದ ಪ್ರಾಣಿ, ಅನಂಗ; ಅವನೊಡನೆ ಜೊತೆಗೂಡಿ
ಇಹರಾರು? ಅಬಲೆ ಹರಿಣಿ ಮುಗುದೆಯರು ನೆರವನೂಡಿ;
ಗೆಲ್ಲಲಿಲ್ಲವೇ, ಮನಸಿಜ, ಹೀಗಿದ್ದರೂ ಮೂಲೋಕದುದ್ದ ಅಗಲ?
ನಿಮ್ಮೆಲ್ಲರಾಸೆ ನನಸಾಗಿಸಲಿ ಮೋಡಿಮಾಡುವ ಆ ಕಮ್ಮಗೋಲ!

೧೦೧ :: ಓಡಿದಳು ರಾಗಿಣಿ ಸಂಧ್ಯೆ

ಚಂಚಚ್ ಚಂದ್ರಕರಸ್ಪರ್ಶ ಹರ್ಷ-ಉನ್ಮೀಲಿತ ತಾರಕಾ
ಅಹೋ ರಾಗವತೀ ಸಂಧ್ಯಾ ಜಹಾತು ಸ್ವಯಂ ಅಂಬರಂ॥101

ವಾಲ್ಮೀಕಿ ರಾಮಾಯಣ

ಕುಣಿವ ಚಂದಿರನ ಕೈತಾಗಿ
ಸಂತಸದಿ ಚುಕ್ಕೆಗಳು ಬೀಗಿದವು ಹಿಗ್ಗಿ;
ತನ್ನ ಬಣ್ಣದುಡುಗೆಯ ತೊಡೆದು, ಖೋಡಿ,
ಬಿಸುಟು ಓಡಿದಳು ಸಂಧ್ಯೆ, ಓ ನೋಡಿ, ನೋಡಿ!

೧೦೨ :: ಮಾತ್ಸರ್ಯ ಇಲ್ಲದ ಹೆಣ್ಣು ಇರುವಳೇನು?

ವಿಲೋಕ್ಯ ಸಂಗಮೇ ರಾಗಂ
ಪಶ್ಚಿಮಸ್ಯ ವಿವಸ್ವತಾ।
ಕೃತಂ ಕೃಷ್ಣಂ ಮುಖಂ ಪ್ರಾಚ್ಯ,
ನ ಹಿ ನಾರ್ಯೋ ವಿನಾಽಈರ್ಷಯಾ॥102

ಪಾಣಿನಿ, ಕವೀಂದ್ರವಚನ ಸಮುಚ್ಚಯದಲ್ಲಿ ಉಲ್ಲೇಖ

ಬೆಂಗದಿರ-ಸಂಗದಲಿ ಪಡುವಣದ ಜವ್ವನೆ ಪ್ರಮದೆ
ರಂಗೇರಿ ಬೀಗುವುದ ಕಣ್ಣಾರೆ ಕಂಡು,
ಮೂಡಣದ ಭಾಮಿನಿ ಮುನಿದು ಕಪ್ಪಾದಳು-
ಹೊಟ್ಟೆಯುರಿಯಿಲ್ಲದ ಹೆಣ್ಣು ಎಲ್ಲಿರುವಳು?!

೧೦೩ :: ಏನು ಬಿಗಿ ಈ ಕುಸ್ತಿಯ ಪಟ್ಟು?

ನ ತದ್ ದೃಢಂ ಬಂಧನಂ ಆಹುರ್ ಧೀರಾಃ
ಯದ್ ಆಯಸಂ ದಾರುಜಂ ವರ್ವಜಂ ಚ ।
ಸಂರಕ್ತರಕ್ತಾ ಮಣಿಕುಂಡಲೇಷು
ಪುತ್ರೇಷು ಧಾರೇಷು ಚ ಯಾ ಅಪ್ರೇಥಾಃ॥103

ಧಮ್ಮಪದ, ತೃಷ್ಣಾವರ್ಗ ೩೪೫

ಕಬ್ಬಿಣದ ಕಟ್ಟಿಗೆಯ ನೂಲುಹಗ್ಗದ ಕಗ್ಗ ಪಟ್ಟು
ಗಟ್ಟಿ ಬಂಧನವೆಂದಾರು ಬಗೆದಾರು ಜಟ್ಟಿಗಳು?
ಅವಕಿಂತ ಮಿಗಿಲಾಗಿ ಬಿಗಿ ನಗ ನಾಣ್ಯ ಕಡುಕದ ಮುತ್ತು
ಮಡದಿ ಮಕ್ಕಳ ಮೇಲೆ ಕವಿದ ಮೋಹದ ಮತ್ತು॥

೧೦೪ :: ತಿಳಿಯಲಾಗದ ಗುಟ್ಟು

ಚಿಕುರ ನಿಚಯೇ ಯತ್ ಕೌಟಿಲ್ಯಂ, ವಿಲೋಚನ ಯೋಶ್ ಚ ಯಾ
ತರಲತರತಾ, ಯತ್ ಕಾಠಿಣ್ಯಂ ತಥಾ ಕುಚಕುಂಭಯೋಃ
ವಸತಿ ಹೃದಿ ತದ್ ಯಾಸಾಂ ಪಿಂಡೀ ಭವನ್ ನನು ತಾ ಇಮಾ
ಗಹನ ಹೃದಯಾ ವಿಜ್ಞಾಯಂತೇ ನ ಕೈಶ್ಚನ ಯೋಶಿತಃ॥104

ಕಲ್ಹಣ, ರಾಜತರಂಗಿಣೀ ೮:೩೬೫

ಕುಡಿ ನೋಟದ ಸ್ಥಿರತೆ, ಮುಂಗುರುಳ ತೂಗಲೆತ,
ದುಂಡುದೊನ್ನೆಯ ಎದೆಯ ಕಾಠಿಣ್ಯ, ಹಿಡಿತ;
ಏನಿದೇನಿದು, ನಾರಿ ನಿನ್ನ ಮನದಾಳದ ನಿಲುವ
ಅರಿಯಬಲ್ಲರು ಹೇಳು, ಯಾರಿಗಿದು ಗೊತ್ತ?

೧೦೫ :: ಅವರವರಿಗವರದೇ ಹಿರಿ ಚಿಂತೆ

ಕನ್ಯಾ ವರಯತೇ ರೂಪಂ, ಮಾತಾ ವಿತ್ತಂ, ಪಿತಾ ಶ್ರುತಂ
ಬಾಂಧವಾಃ ಕುಲಂ ಇಚ್ಛನ್ತಿ, ಮಿಷ್ಟಾನ್ನಂ ಇತರೇ ಜನಾಃ॥105

ನೈಷಧ ಚರಿತಂ ೧೦:೧; ಸುಭಾಷಿತ ರತ್ನ ಸಮುಚ್ಚಯ

ಆಸ್ತಿ ಹಣ ಉಳ್ಳವನೆ?- ತಾಯಿಯದದೊಂದೇ ಚಿಂತೆ;
ಓದಿದವನೇ ನೀನು? ಕೇಳುವನು ತಂದೆ ಬಂದ ವರಗೆ;
ಪ್ರೇಮಿಸಿದ ಹುಡುಗಿಗೋ ಹುಡುಗನನುರೂಪತೆ ಬೇಕು
ನೆಂಟರಿಗೆ ಸಂಬಂಧದ ಘನತೆ ಅದೆ ಮುಖ್ಯ, ಅದೆ ಲಕ್ಷ್ಯ-
ಪೊಗದಸ್ತಾಗಿದ್ದರೆ ಸಾಕು ಊಟ ಉಳಿದವರಿಗೆ !

೧೦೬ :: ಮುಗಿಲ ಮೂಲೆಯಲಿ ನವಿಲ ಕುಣಿತ

ಅನೇಕ ರಾಗ ಸಂಕೀರ್ಣಂ, ಘನಲಗ್ನಂ ಅಪಿ ಕ್ಷಣಾತ್।
ಮಾನುಷಂ ಯೌವನಂ ಕಷ್ಟಂ ನಶ್ಯತಿ ಇಂದ್ರಧನುರ್ ಯಥಾ॥106

ವಡ್ಡಾರಾಧನೆ, ಸನತ್ಕುಮಾರನ ಕತೆ, ಪುಟ ೧೧೮

ಬಣ್ಣಗಳ ಚೆಲ್ಲಾಡಿ, ತುಂತುರು ಮಳೆಯೊಡಗೂಡಿ,
ಬಾನಲಿ ಈಗದು ಮೂಡಿ ಇನ್ನಿಲ್ಲವಾಗುವ ಮೋಡಿ;
ಕಾಮನ ಬಿಲ್ಲಿನ ತೆರದಿ ಹೂವಿನ ಹೆಣ್ಣಿನ ಹರೆಯ
ಎಷ್ಟು ಕ್ಷಣವಿದ್ದೀತು, ಛೆ! -ನೀವೇ ಹೇಳಿ, ನೋಡಿ!

೧೦೭ :: ಬಯಕೆಯ ಬಲೆ ಬೇರೆ, ಸುಖದ ಅನುಭವ ಬೇರೆ

ದಧತಿ ತಾವದ್ ಅಮೀ ವಿಷಯಾಃ ಸುಖಮ್।
ಸ್ಫುರತಿ ಯಾವದ್ ಇಯಂ ಹೃದಿ ಮೂಢತಾ।
ಮನಸಿ ತತ್ವವಿದಾಂ ತು ವಿವೇಚಕೇ
ಕ್ವ ವಿಷಯಾಃ, ಕ್ವ ಸುಖಂ, ಕ್ವ ಪರಿಗ್ರಹಾಃ॥107

ಶಿಲ್ಹಣ, ಶಾಂತಿಶತಕ ೨:೫

ಮುಸುಕಿ ಮೂಢತೆ ನಮ್ಮ ಹೃದಯವನು ಆವರಿಸಿರೆ
ಈ ವಿಷಯಸುಖಗಳೇ ಎಲ್ಲ ಎನಿಸಬಹುದು, ನೀರೆ;
ಆದರಳೊಳಗಣ ತಿರುಳ ಬಲ್ಲವರ ಮಾತೇ ಬೇರೆ:
ವಿಷಯಗಳೆತ್ತಣವೋ, ಬಯಕೆ ಬಲೆಯೆಲ್ಲೊ, ಸುಖವೆಲ್ಲೋ,
ಪಡೆದು ಅನುಭವಿಸುವುದಂತು ಇನ್ನು ಎಲ್ಲೆಲ್ಲೋ!'

೧೦೮ :: ಸುಗ್ಗಿ ಲೂಟಿಯಾದೀತು ಪುಂಡರ ದಂಡು ನುಗ್ಗಿ

ಅಕಂಟಕ ಪುಷ್ಪಮಹೀ ವೇಶಯೋಷಿರ್ ಅಮಾತೃಕಾ
ಮಂತ್ರಿಹೀನಾ ರಾಜಶ್ರೀರ್ ಭುಜ್ಯತೇ ವಿಟ ಚೇಟಕೈಃ॥108

ಕ್ಷೇಮೇಂದ್ರ, ಸಮಯಮಾತೃಕಾ

ಸುತ್ತ ಮಣ್ಣಿನ ದಿಣ್ಣೆ, ಹಣ್ಣಿನ ತೋಟಕ್ಕಿಲ್ಲ ಮಾಲಿ, ಮುಳ್ಳ ಬೇಲಿ;
ತಾಯಿ ಇಲ್ಲದ ಬೆಲೆವೆಣ್ಣು ಹರೆಯದ ಚೆಲುವೆ ಮುಗ್ಧೆ ಬೆಣ್ಣೆ;
ತುಂಬಿ ಉಕ್ಕಿದೆ ಬೊಕ್ಕಸ ಕಣಜ, ರಾಜ್ಯದ ಮಂತ್ರಿಗಳ ಜಾಗ ಖಾಲಿ
ಈ ಪಾಟಿ ಇರೆ ಸುಗ್ಗಿ ಲೂಟಿಯಾಗದೆ ಪುಂಡರ ದಂಡು ನುಗ್ಗಿ?!

೧೦೯ :: ಉರಿಯನಾರಿಸುವುದು ತುಪ್ಪದಿಂದಲ್ಲ

ನ ಜಾತು ಕಾಮ ಕಾಮಾನಾಂ ಉಪಭೋಗೇನ ಶಾಮ್ಯತೇ।
ಹವಿಷಾ ಕೃಷ್ಣವತ್ಮ್ರ್ಯಾ ಇವ ಭೂಯ ಏವ ಅಭಿವರ್ಧತೇ॥109

ವಾಯುಪುರಾಣ ೯೩:೯೫

ಕಾಮದನುಭವದಿಂದ ಕಾಮ ಮಣಿಸುವೆ ಎನುವೆ
ಉಂಡು ತಣಿಸುವ ಒಡಲ ಹಸಿವು ದಾಹ ಅದಲ್ಲ;
ಉರಿಯ ಆರಿಸಹೊರಟು ತುಪ್ಪವನೇ ಸುರಿಯುತ್ತಿರೆ
ಬೆಂಕಿ ಭುಗಿಲೆಂದೇಳಲದು ನೆರವಾಯಿತಲ್ಲ.
