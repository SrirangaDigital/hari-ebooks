೯. ದೃಢತೆ-ದೂರದೃಷ್ಟಿ

೬೪ :: ಶ್ರೇಯ-ಪ್ರೇಯ

ಶ್ರೇಯಶ್ಚ ಪ್ರೇಯಶ್ಚ ಮನುಷ್ಯಂ ಏತಸ್।
ತೌ ಸಂಪರೀತ್ಯ ವಿನಕ್ತಿ ಧೀರಃ।
ಶ್ರೇಯೋ ಹಿ ಧೀರೋ ಅಭಿ ಪ್ರೇಯಸೋ ವೃಣೀತೇ।
ಪ್ರೇಯೋ ಮಂದೋ ಯೋಗಕ್ಷೇಮಾತ್ ವೃಣೀತೇ॥064

ಕಠೋಪನಿಷತ್ ೨:೧

ನಮ್ಮ ಜೀವನ-ಪಥದಿ ಒಂದಲ್ಲ ಒಂದು ದಿನ
ಆ ಕವಲು ದಾರಿಯ ಬುಡವ ಸಂಧಿಸಲೇ ಬೇಕು;
ಆಗ ಶ್ರೇಯ-ಪ್ರೇಯವು ಬಂದು ಕೈ ಮುಗಿದು ಕೇಳುವುವು:
'ಇತ್ತ ಬಾ, ಇತ್ತ ಬಾ, ನನ್ನತ್ತ ಬಾ, ಬಾ 'ರೆಂದು;
ದೂರದೃಷ್ಟಿಯ ಧೀರ'ಶ್ರೇಯಸ್ಸ ' ಬಯಸಿದರೆ,
ನಡೆವ ಮಂದಮತಿ ಬರಿ ಇಹದ ಸುಖ'ಪ್ರೇಯಸ್ಸಿ 'ನೆಡೆಗೆ!

೬೫ :: ಮುಂದಾಲೋಚನೆ, ಮಂದಾಲೋಚನೆ

ಅನಾಗತ ವಿಧಾತಾ ಚ ಪ್ರತ್ಯುನ್ಪನ್ನಮತಿಸ್ ತಥಾ
ದ್ವೌ ಇಮೌ ಸುಖಂ ಏಧೇತೇ ಯದ್ ಭವಿಷ್ಯೋ ವಿನಶ್ಯತಿ॥065

ಪಂಚತಂತ್ರ, ಮಿತ್ರಭೇದ ೩೪೭, ಯದ್ಭವಿಷ್ಯ-ಮತ್ಸ್ಯನ ಕತೆ

ಮುಂದೆ ಎಂದೋ ನಡೆವುದಕೆ ಇಂದೇ ಯೋಜನೆ ಹೂಡಿ,
ಈಗ ಬಂದೆರಗಿದುದಕೆ ತಕ್ಕ ಪರಿಹಾರ ಹುಡುಕಿ ನಡೆವ-
ಇಬ್ಬರೇ ಭಾಜನರು ನೆಮ್ಮದಿಗೆ; ಸುಮ್ಮನೆ, ಕಾಲವ ದೂಡಿ,
ಬೆಂಕಿ ಬಿದ್ದಾಗ ಬಾವಿ ತೋಡಿದರಾಯ್ತು- ಎನ್ನುವವ ಕೆಡುವ ॥

೬೬ :: ನೋಡುವಡೆ ನೋಡಿ ಗಗನದತ್ತ

ಧರ್ಮಂ ಚರತ, ಮಾಽಽಧರ್ಮಂ; ಸತ್ಯಂ ವದತ ನಾನೃತಮ್।
ದೀರ್ಘಂ ಪಶ್ಯತ, ಮಾ ಹೃಸ್ವಂ; ಪರಂ ಪಶ್ಯತ, ಮಾಽಪರಮ್॥066

ಮಹಾಭಾರತ

ನಡೆ ಧರ್ಮಮಾರ್ಗದಲಿ, ಬೇಡ ಬೇರೆಡೆ ಒಂದೂ ಹೆಜ್ಜೆ;
ನುಡಿಯುತಿರು ದಿಟವನ್ನೇ, ಸಟೆಯನಾಡಲೆ ಬೇಡ, ಸಲ್ಲ;
ಉತ್ತಮಿಕೆ ಗುರಿಯಾಗೆ, ಕೀಳಿನೊಡನೇಕೆ ಸಂಧಾನದ ಒಜ್ಜೆ ;
ನೋಟ ದೂರದ್ದಿರಲಿ, ಸನಿಹ ಕಾಣುವುದಷ್ಟೇ ಎಲ್ಲ ಅಲ್ಲ;

೬೭ :: ನಡುಹಗಲಿನ ಕತ್ತಲೆ

ಶೋಕೋ ನಾಶಯತೇ ಧೈರ್ಯಂ,
ಶೋಕೋ ನಾಶಯತೇ ಶ್ರುತಂ।
ಶೋಕೋ ನಾಶಯತೇ ಸರ್ವಂ,
ನಾಸ್ತಿ ಶೋಕಸಮೋ ರಿಪುಃ॥067

ವಾಲ್ಮೀಕಿ ರಾಮಾಯಣ ೨:೬೨:೧೫

ಎದೆಯುಡುಗಿ ನಡುಗುವುದು, ತಳ ಕುಸಿದ ಭಾಸವದು;
ತಿಳಿವಿಗಿರುಳಿನ ಮಬ್ಬು ಥಟ್ಟನೆ ಮುಸುಕ ಬಹುದು;
ಕುಗ್ಗಿಸುತ ಎಲ್ಲ ಇದ್ದರೂ ಇಲ್ಲಿ ಏನಿಲ್ಲವೆನಿಸುವುದು-
ಶೋಕ ಕಡು ಹಗೆಯಾಗಿ ಮಣಿಸದದು ಏನು ಇಹುದು?

೬೮ :: ಕುಗ್ಗದೆಲೆ ನುಗ್ಗುವರು ನೀಡಿ ಎದೆಮುಂದೆ

ನಿಂದನ್ತು ನೀತಿ ನಿಪುಣಾಃ, ಯದಿ ವಾ ಸ್ತುವನ್ತು,
ಲಕ್ಷ್ಮೀ ಸಮಾವಿಶತು, ಗಚ್ಛತು ವಾ ಯಥೇಷ್ಟಮ್;
ಅದ್ಯೈವ ವಾ ಮರಣಂ ಅಸ್ತು, ಯುಗಾಂತರೇ ವಾ-
ನ್ಯಾಯಾತ್ ಪಥಃ ಪ್ರವಿಚಲನ್ತಿ ಪದಂ ನ ಧೀರಾಃ068

ಭರ್ತೃಹರಿ ನೀತಿ ಶತಕ ೮೧

ನಿಂದಿಸಲಿ ಹುಳುಕು ಹುಡುಕುತ ಜರಿವುದಕೆ ಇರುವ ಜನ,
ಹಾಡಿ ಹೊಗಳಲಿ ಏರಿಸೇರಿಸಿ ಹತ್ತಿಸಲಿ ನಿನ್ನ ಹೊನ್ನ ಮೇನ;
ಹರಿದಿರಲಿ ಹಣದ ಹೊಳೆ ಸತತ; ಇಲ್ಲ, ಬತ್ತಲಿ ನೀರ ಪೂರ;
ಕಥೆ ಇಲ್ಲಿಗೇ ಮುಗಿಯಲಿ; ಇಲ್ಲ, ಸಾಗಲಿ ಸಾರ್ಥ ದೂರ ದೂರ-
ಸರಿ ದಾರಿ ಬಿಡುವನೆ, ನಡೆವನೆ, ತಿರುವಿನ ಜಾಡ, ಧೀರ?

೬೯ :: ಆಕಾಶವೇ ಬೀಳಲಿ ಮೇಲೆ...

ನಿಮಿತ್ತಂ ಆಸಾಧ್ಯ ಯದ್ ಏವ ಕಿಂಚನ,
ಸ್ವಧರ್ಮ ಮಾರ್ಗಂ ವಿಸೃಜಂತಿ ಬಾಲಿಶಾಃ।
ತಪಃ ಶ್ರುತ ಧ್ಯಾನ ಧನಾಸ್ ತು ಸಾಧವೋ
ನ ಯಾಂತಿ ಕೃಚ್ಛ್ರೇ ಪರಮೇಽಪಿ ವಿಕ್ರಿಯಾಮ್॥069

ಸುಭಾಷಿತ

ಕುಂಟುನೆಪ ಸಿಕ್ಕರೆ ಸಾಕು, ಬಿಟ್ಟೋಡಿ ಬಿಡಬಹುದು
ಹಿಡಿದ ಕಡುಕಷ್ಟದ ಜಾಡ ಹಸುಗೂಸ ಮನಸಿನವರು;
ಶ್ರದ್ಧೆ ಕಲಿಕೆ ಅರಿವೇ ಆಸ್ತಿಯಾಗುಳ್ಳವರು ಸಾಧಕರು
ಜಡಿದರೂ ಕಷ್ಟಗಳ ಮಳೆ, ಇಟ್ಟಡಿಯ ಹಿಂದೆ ಇಡರು.

೭೦ :: ಬಿಸಿಲುಗುದುರೆಯನೇರಿ ಹೊರಟೆ ಎಲ್ಲಿಗೆ ನೀನು?

ಯೋ ಧ್ರುವಾಣಿ ಪರಿತ್ಯಜ್ಯ, ಅಧ್ರುವಂ ಪರಿಷೇವತೇ।
ಧ್ರುವಾಣಿ ತಸ್ಯ ನಶ್ಯನ್ತಿ, ಅಧ್ರುವಂ ನಷ್ಟಂ ಏವ ಚ॥07೦

ಗರುಡಪುರಾಣ ೧೦೮

ಸ್ಥಿರವನಿಲ್ಲಿಯೇ ಬಿಟ್ಟು ಅಸ್ಥಿರವ ಬೆಂಬತ್ತುವಿರಾ,
ಹತ್ತಿ ಅ ಬಿಸಿಲುಗುದುರೆಯ ಕನಸಿನ ಸವಾರಿಯಲ್ಲಿ?
ಕೈಯ ದಂಟೂ ಕಳೆದು, ಕನ್ನಡಿಯ ಗಂಟೂ ಸಿಗದೆ,
ಭ್ರಮೆ ನಿರಸನವಾಗಿ ಉಳಿದೀತು ತಳಮಳದ ಕೊಳ್ಳಿ!
