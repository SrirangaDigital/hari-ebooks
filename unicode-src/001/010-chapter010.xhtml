೧೦ ವಿಧಿವಿಲಾಸ

೭೧ :: ದುರ್ಬಲರೇ ವಿಧಿಗೆ ಬಲಿ

ಅಶ್ವಂ ನೈವ ಗಜಂ ನೈವ, ವ್ಯಾಘ್ರಂ ನೈವ ಚ ನೈವ ಚ।
ಅಜಾಪುತ್ರಂ ಬಲಿಂ ದದ್ಯಾತ್, ದೈವೋ ದುರ್ಬಲಘಾತಕಃ॥071

ಸಮಯೋಚಿತಪದ್ಯಮಾಲಿಕಾ

ಕುದುರೆಯನ್ನೇನಲ್ಲ, ಆನೆಯನ್ನಂತೂ, ಬಿಡಿ, ಅಲ್ಲ
ಹುಲಿಯನ್ನ? ಛೇ, ಖಂಡಿತಾ ಅಲ್ಲವೇ ಅಲ್ಲವಯ್ಯಾ;
ಬಲಿ ಕೊಡಲು ಹೇಳುವರು ಪಾಪ ಕುರಿಮರಿಯನ್ನ-
ಬೆನ್ನ ಹತ್ತೀತು ವಿಧಿ ದುರ್ಬಲರ, ಅಲ್ಲವೇನಯ್ಯಾ!

೭೨ :: ಹಗಲುಗನಸಿನ ಸಾರ್ಥದ ಪಥಿಕ

ಅತ್ರ ವರ್ಷಂ ವಸಿಷ್ಯಾಮಿ ಅತ್ರ ಹೇಮಂತ ಗ್ರೀಷ್ಮಷು।
ಇತಿ ಬಾಲೋ ವಿಚಿಂತಯತಿ, ಅಂತರಾಯಂ ನ ಊಹ್ಯತಿ॥072

ಧಮ್ಮಪದ, ಮಾರ್ಗವರ್ಗ ೨೮೬

ಇಲ್ಲಿ ಮಳೆಗಾಲವ ಕಳೆವೆ
ಹೇಮಂತ ಗ್ರೀಷ್ಮದಲುಳಿವೆ-
ಸಾರ್ಥದಿ ಹೊರಟವ ಕಂಡನೊಂದು ಹಗಲುಗನಸ।
ಮುಂದೆ ಇನ್ನು ನಡೆವುದದೇನು
ಊಹಿಸದು ಕೂಸಿನ ಮನಸು-
ನದಿಯ ದಾಟದ ಪಥಿಕನ ಬೆನ್ನ ಬದಿ ವಿಧಿ ನಕ್ಕಿತು!

೭೩ :: ಚಕ್ರ ಉರುಳಿತು, ಮೇಲಿನದು ಕೆಳಗಿಳಿಯಿತು

ಹೃತಂ ರಾಜ್ಯಂ, ಪ್ರಿಯಾ ದಗ್ಧಾ, ಭ್ರಷ್ಟಃ ಸೂನುರ್, ಅಬಾಂಧವಃ,
ಜಾತೋ ಅಸ್ಮಿ ಏಕಾಕೀ, ಅಪಾಥೇಯೋ,
ಲುಠನ್ ಭಿಕ್ಷಾಭುಜೋ ಅಂಗನೇ-
ಇತಿ ಏಕೈಕಂ ಚ, ನಿರ್ಧಾಯ ದುಃಖಂ ನ ಈಕ್ಷಿಷ್ಟ ಸೋ ಅಪರಂ
ಪ್ರಾಪ್ತ ಅವಸಾದಂ ಶೋಚಂತಂ ಆತ್ಮತುಲ್ಯಂ ಕಥಾಸು ಅಪಿ ॥073

ಕಲ್ಹಣ, ರಾಜತರಂಗಿಣೀ ೭:೧೬೪೮-೪೯

ರಾಜ್ಯ ಹೋಯಿತು, ಕೈಹಿಡಿದಾಕೆ ಇನ್ನೆಲ್ಲಿ, ಈಗವಳು ಹಿಡಿ ಬೂದಿ!
ಮಗ ತೊರೆದ, ಅಗಲಿದರು ಬಂಧುಗಳು ಹಿಡಿದು ತಮ್ಮ ಹಾದಿ;
ಏಕಾಕಿ, ಜೊತೆಯಿಲ್ಲ, ಹೊಟ್ಟೆಹೊರೆಯಲು ಮನೆಮನೆಗಲೆತ ತಿರುಪೆ
-ಯಾರಿಗೂ ಈ ಗತಿ ಬೇಡ, ಬಾಳ ಕತೆ ಹೀಗೆ, ಗೋಳು ಚಿಂದಿ!

೭೪ :: ವಿಧಿ ಹಾಕಿದ ಕಲ್ಲು

ಯದ್ ಉಲ್ಲಾಸಾಯ ಸಂರಂಭೋ
ಧೀರೈರ್ ವಿಸ್ತಾರ್ಯತೇ ಮಹಾನ್
ಕೃತ್ಯಂ ಹಿನಸ್ತಿ ತದ್ ದೈವಂ ಅತಿ ಅಲ್ಪೇನೈವ ವಸ್ತುನಾ॥074

ಕಲ್ಹಣ, ರಾಜತರಂಗಿಣೀ ೭:೯೧೭

ತೊಡಗಿ ಹೊರಟಿರಬಹುದು ಉತ್ಸಾಹದೊಡಗೂಡಿ
ಭಾರಿ ಭಾರಿ ಯೋಜನೆಗಳ ಹೂಡಿ, ಮಾಡಿ;
ಕಲ್ಲು ಬೀಳಲೂಬಹುದು, ಒಂದು ಹಿಡಿ ಮರಳೇ ಸಾಕು,
ವಿಧಿ ನೋಡಿ, ನಗುತಲಿದೆ, ಎಂಥ ಖೋಡಿ!

೭೫ :: ವಿಧಿಯೊಡನೆ ಸಂಧಾನ

ಕುಷ್ಠಾರ್ತಾಂಘ್ರಿಯುಗಃ ಶಿಖೀ
ಬಹುಪದಂ ಗೃಹ್ಣಾತಿ ಧಾವನ್ ಅಹಿಂ
ಭಾನುಃ ಪಾದಸಹಸ್ರಬಾಕ್ ಪ್ರತಿಪದಂ
ಸಂಚಾರ್ಯತೇ ಅನೂರುಣಾ॥
ವಂಚ್ಯತೇ ಬಲಿನೋಽಪಿ ಯಲ್ ಲಘುಬಲೈಃ
ಸಾಮಥ್ರ್ಯಹೀನೈಶ್ಚ ಯದ್ ಭ್ರಾಮ್ಯಂತೇ
ಪರಿಪೂರ್ಣವೃತ್ತಯ -ಇದಂ ದೈವಸ್ಯ ಲೀಲಾಯಿತಮ್॥075

ಕಲ್ಹಣ, ರಾಜತರಂಗಿಣೀ ೭:೯೫೯

ಕಾಲೆರಡಕ್ಕೂ ರೋಗಬಡಿದಿದೆ, ಇರಲಿ ಬಿಡಿ,
ನವಿಲೋಡಿ ಹಾವನ್ನ ಹಿಡಿಯದೇನು?
ಮುನ್ನಡೆಸುವುದಾರು ಕಿರಣಗಳೊಡೆಯನ, ಹೇಳಿ
ತೊಡೆಮುರಿದ ಸಾರಥಿ ಅರುಣನೇನು?
ಶಕ್ತಿ ಇದ್ದರೇನಂತೆ ಲಘುಬಲರು ಯುಕ್ತಿಯಲಿ
ಮೀರಿಸರೇ ಅತಿಬಲರ ಕಪಟತನದಿ?
ಬಿದ್ದರೇ ಗೊಂದಲದಿ ಎಲ್ಲ ಬಲ್ಲವರೂನು
ವಿಧಿಯಾಟ ಅರಿಯದೆಯೆ ಹೋದರೇನು?

೭೬ :: ವಿಧಿ ಬರೆದು ಲೇಖಣಿ ಪಕ್ಕಕ್ಕಿಟ್ಟಾಯ್ತು

ಭವಿತವ್ಯಂ ಭವತಿ ಏವ, ಕರ್ಮಣಾಂ ಏಷ ನಿಶ್ಚಯಃ
ವಿಪತ್ತೌ ಕಿಂ ವಿಷಾದೇನ, ಸಂಪತ್ತೌ ವಿಸ್ಮಯೇನ ಕಿಮ್॥076

ವಡ್ಡಾರಾಧನೆ, ಸನತ್ಕುಮಾರನ ಕತೆ, ಪುಟ ೧೦೯

ಆಗುವುದು ಆಗಲೇಬೇಕು; 'ಹೇಗೆ ಎಲ್ಲಿ ಯಾರಿಂದೆಂ' ದು
ವಿಧಿ ಬರೆದು, ತನ್ನ ಲೇಖಣಿಯ ಎಂದೋ ಕೆಳಗಿರಿಸಿದೆ;
ಬೆಂಬಿಡದ ಸಿರಿ ನೆರಳೋ, ದುಗುಡದುರೆ ಹೊರೆ ಇರುಳೋ
ಸೋಜಿಗವ ಬೆರಗ ಪಡಲೇನಿಹುದಿಲ್ಲಿ, ಅರ್ಥವಿಹುದೆ?

೭೭ :: ತಾನೊಂದು ಬಗೆದರೆ ದೈವ ಬೇರೆ ಬಗೆದೀತು

ರಾತ್ರಿರ್ ಗಮಿಷ್ಯತಿ ಭವಿಷ್ಯತಿ ಸುಪ್ರಭಾತಮ್,
ಭಾಸ್ವಾನ್ ಉದೇಷ್ಯತಿ ಹಸಿಷ್ಯತಿ ಪಂಕಜಶ್ರೀಃ ।
ಇತ್ಥಂ ವಿಚನ್ತಯತಿ ಕೋಶಗತೇ ದ್ವಿರೇಫೇ,
ಹಾ, ಮೂಲತಃ ಕಮಲಿನೀಂ ಗಜ ಉಜ್ಜಹಾರ!॥077

ಸೂಕ್ತಿಮುಕ್ತಾವಲೀ ೧೯:೫

ಇರುಳು ಕಳೆವುದು ಜಾರಿ, ಮೂಡುವುದು ಮುಂಜಾವು;
ನೇಸರುದಯಿಸೆ ಬಿರಿವುದು ತಾವರೆ ಮುಗುಳುನಗೆಯ ಬೀರಿ;
ಹೊರ ಬರುವೆನಿನ್ನೇನು ಎನುತಿರೆ ಮೊಗ್ಗಲಡಗಿದ ದುಂಬಿ-
ಬುಡ ಸಹಿತ ಕಿತ್ತೊಗಿಯಿತೇ ಹಾ ಕಮಲದ ಗಿಡವ ಮದ್ದಾನೆ!

೭೮ :: ವಿಧಿ ದೈವಾಯತ್ತ

ದೈವಂ ಫಲತಿ ಸರ್ವತ್ರ,
ನ ವಿದ್ಯಾ ನ ಚ ಪೌರುಷಮ್।
ಸಮುದ್ರಮಥನಾಲ್ ಲೇಭೇ,
ಹರಿರ್ ಲಕ್ಷ್ಮೀಂ, ಹರೋ ವಿಷಮ್॥078

ಸುಭಾಷಿತ ರತ್ನಭಾಂಡಾಗಾರ ೯೧:೧೦

ಎಲ್ಲೆಡೆಯೂ ಜಯಮಾಲೆ ಯಾರ ಕೊರಳಿಗೆ ಕೊನೆ ಕೊನೆಗೆ?
ವಿದ್ಯೆಗೇ? ಪೌರುಷಕೇ? ಯಾರ ನಿರ್ಧಾರ?- ಮಗನೆ;
ಕಡಲು ಕಡೆದಾಯ್ತಲ್ಲ, ದೈವ ಯಾರಿಗೇನು ಕೊಟ್ಟಿತು ಗೊತ್ತೆ?
ಹರಿಗೆ ಸಿಕ್ಕಳು ಲಕ್ಷ್ಮಿ, ದಕ್ಕಿದ್ದು ಹರಗೆ ವಿಷವೇ ತಾನೆ?
