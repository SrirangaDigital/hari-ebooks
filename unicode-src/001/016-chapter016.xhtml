೧೬ದೇವರು

೧೩೨ :: ಆ ಬೆಳಕು ಈ ಬೆಳಕು ಎಲ್ಲ ಆ ಹೆಬ್ಬೆಳಕೇ

ನ ತತ್ರ ಸೂರ್ಯೋ ಭಾತಿ ನ ಚಂದ್ರತಾರಕಂ,
ನೇಮೋ ವಿದ್ಯುತೋ ಭಾನ್ತಿ ಕುತೋಯಂ ಅಗ್ನಿಃ ॥
ತ್ವಂ ಏವ ಭಾಂತಂ ಅನುಭಾತಿ ಸರ್ವಂ,
ತಸ್ಯ ಭಾಸಾ ಸರ್ವಂ ಇದಂ ವಿಭಾತಿ॥132

ಕಠೋಪನಿಷತ್ ೨:೨:೧೪

ಅಲ್ಲಿ ರವಿ ಮೂಡುವುದಿಲ್ಲ, ಚಂದ್ರ ಹೊಳೆಯುವುದಿಲ್ಲ,
ತಾರೆಗಳು ಮಿನುಗುವುದಿಲ್ಲ ಮಿಂಚುಗಳ ಝಳಪು ಇಲ್ಲ:
ಇನ್ನೆಲ್ಲಿ ಅಗ್ನಿಯ ಸೊಲ್ಲು? ಪ್ರತಿಫಲವವೆಲ್ಲಕೂ ಮೂಲ-
ಆ ಹೆಬ್ಬೆಳಕೆ ಬೆಳಗಿ ಬೆಳಗಿಸುತಿದೆ ತುಂಬಿ ಜಗವನೆಲ್ಲ!

೧೩೩ :: ಬೆನಕನ ನೆನಕೆ

ಶುಕ್ಲ-ಅಂಬರ-ಧರಂ ವಿಷ್ಣುಂ,
ಶಶಿವರ್ಣಂ ಚತುರ್ಭಜಮ್।
ಪ್ರಸನ್ನ-ವದನಂ ಧ್ಯಾಯೇತ್,
ಸರ್ವ ವಿಘ್ನ-ಉಪಶಾಂತಯೇ॥133

ಅನಾಮಕ ಕವಿಯ ಒಂದು ಜನಪ್ರಿಯ ಸ್ತೋತ್ರ

ಬಿಳಿಯ ಬಟ್ಟೆಯ ಧರಿಸಿ ಮೆರೆವನೆ,
ಚಂದ್ರನಂದದಿ ಸುಂದರಾಂಗನೆ,
ಪಾಶ ಅಂಕುಶಗಳನು ಹಿಡಿದಿಹ ನಾಲ್ಕು ಕೈಗಳುಳ್ಳವನೆ।
ನಗುಮೊಗನೆ ಗಣದೊಡೆಯ ಬೆನಕನೆ
ನಡೆವ ದಾರಿಯ ನಡುವೆ ಇರುತಿಹ
ಅಡೆ ತಡೆಗಳೆಲ್ಲವನು ತೊಡೆಯುತ ನೀನೇ ಕಾಪಾಡು॥

೧೩೪ :: ತ್ರ್ಯಂಬಕನಾದುದು ಹೇಗೆ?

ತಿಸ್ರೋ ದೇವೀರ್ ಯದಾ ಚೈವ ಭಜತೇ ಭುವನೇಶ್ವರಃ।
ದ್ಯಾಂ ಆಪಃ ಪೃಥಿವೀಂ ಚೈವ ತ್ರ್ಯಂಬಕಶ್ಚ ತತಃ ಸ್ಮೃತಃ॥134

-ಮಹಾಭಾರತ, ದ್ರೋಣಪರ್ವ, ೨೦೨:೩೦

ಒಬ್ಬರಿಬ್ಬರು ಅಲ್ಲ, ಮೂರು ಲೋಕಮಾತೆಯರನ್ನ
ಕಾಪಿಡುವ ಈ ಸ-ಅಂಬ ಶಿವ, ಶಕ್ತಿಯೊಡಗೂಡಿ;
ನೀರು, ಭೂಮಿಯ ಜೊತೆಗೆ ಅನ್ತರಿಕ್ಷವ ಕಾಯ್ದು
ಶಂಕರನು ತ್ರ್ಯಂಬಕನಾದ, ಜಗದೀಶ ನೋಡಿ!

೧೩೫ :: ಕಳಿತ ಹಣ್ಣನಾಗಿಸು ಎನ್ನ

ತ್ರ್ಯಂಬಕಂ ಯಜಾಮಹೇ ಸುಗಂಧಿಂ ಪುಷ್ಟಿ ವರ್ಧನಂ।
ಉರ್ವಾರುಕಂ ಇವ ಬಂಧನಾತ್ ಮೃತ್ಯೋರ್ ಮುಕ್ಷೀಯ
ಮಾ ಅಮೃತಾತ್॥135

ತೈತ್ತಿರೀಯ ಆರಣ್ಯಕ, ಮಹಾನಾರಾಯಣ ಉಪ ೫೬:೧

ತ್ರ್ಯಂಬಕನ ಭಜಿಸುವೆನು ಸುಗಂಧಿ ಪುಷ್ಟಿವರ್ಧನನನ್ನು,
ಮೃತ್ಯು ಕೊಂಡೊಯ್ಯದ ರೀತಿ, ನೀ ನನ್ನ ಉಳಿಸೋ!
ಅಮೃತದ ಸೆಲೆಯಿಂದ ಬೇರ್ಪಡದೆ ಇರುವಂತೆ,
ತೊಟ್ಟು ಕಳಚಿದ ಕಳಿತ ಹಣ್ಣಿನಂದದಿ ಬಿಡಿಸಿ ಇರಿಸೋ!

೧೩೬ :: ಕಣ್ಣಲ್ಲಿ ಬೆಳಕಿನ ಉಸಿರು

ಅಸುನೀತೇ ಪುನರ್ ಅಸ್ಮಾಸು ಚಕ್ಷುಃ।
ಪುನಃ ಪ್ರಾಣಂ ಇಹಿ ನೋ ಧೇಹಿ ಭೋಗಂ।
ಜ್ಯೋಕ್ ಪಶ್ಯೇಮ ಸೂರ್ಯಂ ಉತ್ ಚರನ್ತಂ।
ಅನುಮತೇ ಮೃಡ(ಳ)ಯಾನ ಸ್ವಸ್ತಿ॥136

-ಋಗ್ವೇದ ೧೦:೫೯:೬

ನಮ್ಮ ಈ ಕಣ್ಣಲಿ ಮತ್ತೆ ಬೆಳಕ ಸಿರಿಯನೆ ಕೂಡು,
ಉಸಿರೂಡು, ದೇವರೇ, ಸುಖದ ಅನುಭವ ನೀಡು;
ಮೇಲೆ ಮೇಲೇರುವ ರವಿಯ ನೋಡುತಲೇ ಚಿರಕಾಲ
ನಾವು ಬಾಳುವ ತೆರದಿ ಹರಸಿ, ನಮ್ಮ ಕಾಪಾಡು!

೧೩೭ :: ಸೂರ್ಯ ಜಗತ್ತಿನ ಕಣ್ಣು

ಉದ್ ಉ ತ್ಯಂ ಜಾತವೇದಸಂ ದೇವಂ ವಹನ್ತಿ ಕೇತವಃ।
ದೃಶೇ ವಿಶ್ವಾಯ ಸೂರ್ಯಂ।
ಚಿತ್ರಂ ದೇವಾನಾಂ ಉದಗಾದ್ ಅನೀಕಂ
ಚಕ್ಷುರ್ ಮಿತ್ರಸ್ಯ ವರುಣಸ್ಯ ಆಗ್ನೇಃ।
ಆಪ್ರಾಃ ದ್ಯಾವಾ ಪೃಥಿವೀ ಅನ್ತರಿಕ್ಷಗ್‍ಮ್।
ಸೂರ್ಯ ಆತ್ಮಾ ಜಗತಸ್ ತಸ್ಥುಷಶ್ ಚ॥137

ತೈತ್ತಿರೀಯ ಸಂಹಿತಾ ೧:೪:೪೩

ಮಿತ್ರ, ವರುಣರ ಕಣ್ಣು, ಅವನೆ ಅಗ್ನಿಗೂ ಕಣ್ಣು,
ಸಮಸ್ತ ಶಕ್ತಿಸಮುದಾಯಕ್ಕೆ ಲೋಕಕ್ಕೇ ಕಣ್ಣು;
ಜಗವೆಲ್ಲವೀ ಜ್ಞಾನಪ್ರಚೋದಕನ ಭವ್ಯ ರವಿಯ
ತೇಜಃಪುಂಜನ ದಿವ್ಯ ದರ್ಶನವ ಪಡೆಯಲೆನೆ
ಕಿರಣಗಳು ಹೊತ್ತನು ಹೊತ್ತು ಮೇಲೆ ಮೇಲೇರಿಸಿವೆ-
ದಿವಿ-ಬುವಿ-ಆಗಸವನಾವರಿಸಿದನು ಚರಾಚರಾಗಳಾತ್ಮ!

೧೩೮ :: ನಿನ್ನ ಸೇರುವ ಆಸೆ

ಉದ್ ವಯಂ ತಮಸಸ್
ಪರಿ ಪಶ್ರ್ಯನ್ತೋ ಜ್ಯೋತಿರ್ ಉತ್ತಮಮ್।
ದೇವಂ ದೇವತ್ರಾ ಸೂರ್ಯಂ
ಅಗನ್ಮ ಜ್ಯೋತಿರ್ ಉತ್ತಮಮ್॥138

ತೈತ್ತಿರೀಯ ಸಂಹಿತಾ ೪:೧:೭

ಕತ್ತಲಾಚೆಯ ಬೆಳಕ ಅತ್ತ ಎತ್ತರವನೇರಿದನ,
ಉತ್ತಮನ ಚಂಡಕಿರಣನ ಆ ಪರಂಜ್ಯೋತಿಸ್ವರೂಪನನ್ನ,
ನೋಡಿ, ಕಣ್ಣಲಿ ತುಂಬ ತುಂಬಿಕೊಂಡವನನ್ನ,
ಬೇಡುವೆವು- ಪೂರೈಸು ಕೊನೆಗೆ ನಿನ್ನ ಸೇರುವ ನಮ್ಮಾಸೆಯನ್ನ!

೧೩೯ :: ಅವನೊಬ್ಬನೇ!

ಏಕೋ ಹಿ ರುದ್ರೊ, ನ ದ್ವಿತೀಯಾಯ ತಸ್ಥುರ್,
ಯ ಇಮಾಂಲ್ ಲೋಕಾನ್ ಅಶೀತ ಈಶಾನೀಭಿಃ।
ಪ್ರತ್ಯಂಗ್ ಜನಾಂಸ್ ತಿಷ್ಠತಿ ಸಂಚುಕೋಚ,
ಅನ್ತಕಾಲೇ ಸಂಸೃಜ್ಯ ವಿಶ್ವಾ ಭುವನಾನಿ ಗೋಪಾ॥139

ಶ್ವೇತಾಶ್ವತರ ಉಪನಿಷತ್ ೩:೨

ಎರಡಲ್ಲ, ಬೇರಿಲ್ಲ, ಒಂದೇ ರುದ್ರ, ಅವ ಒಬ್ಬನೇ;
ಹುಟ್ಟು, ಇರುವು, ಅಳಿವು ಮೂರನೂ ನಿರ್ವಹಿಸುತ್ತಿರುವ,
ಮೊದಲು ಕೊನೆಗೂ ಜಗವ ತನ್ನಲೇ ಅಡಗಿಸಿಕೊಳುವ,
ಏಕೈಕ ಸರ್ವ ಶಕ್ತಿ 'ಅದು', ರುದ್ರ ಅವನೊಬ್ಬನೇ!!

೧೪೦ :: ಎದೆಯಾಳದಲಿ ಇವ ಕುಳಿತ!

ಏಕೋ ದೇವಃ, ಸರ್ವ ಭೂತೇಷು ಗೂಢಃ;
ಸರ್ವವ್ಯಾಪೀ, ಸರ್ವಭೂತ-ಅನ್ತರಾತ್ಮಾ।
ಕರ್ಮಾಧ್ಯಕ್ಷಃ, ಸರ್ವ-ಭೂತ-ಅಧಿವಾಸಃ;
ಸಾಕ್ಷೀ ಚೇತಾ, ಕೇವಲೋ ನಿರ್ಗುಣಶ್ಚ॥140

ಶ್ವೇತಾಶ್ವತರ ಉಪನಿಷತ್ ೬:೧೧

ಆ ದೇವ ತಾನೊಬ್ಬನೇ; ಎಲ್ಲ ಚರಾಚರಗಳಲಡಗಿ, ಹುದುಗಿ,
ಎಲ್ಲೆಡೆಯು ಹರಡಿರುತ, ಎಲ್ಲರೆಲ್ಲರ ಎದೆಯ ಆಳದಲಿ ಕುಳಿತ.
ಸುತ್ತ ತಾ ನೋಡುವವ, ನಾವೆಸಗುವೆಲ್ಲದಕೆ ಕಣ್ಣಿಟ್ಟ ಅಧ್ಯಕ್ಷ,
ನಡೆ ನುಡಿ ಮನದ ತುಡಿತಕ್ಕೆ ಮಿಡಿತಕ್ಕೆ ಸಾಕ್ಷಿ ಹೀಗೊಬ್ಬನೇ,
ನಾಮ-ರೂಪದ ಎಲ್ಲೆಗಳ ಮೀರಿರುವಾತ ಅವನೊಬ್ಬನೇ!

೧೪೧ :: ದೇವನೊಬ್ಬನೆ, ಅವನ ನಾಮ ಹಲವು

ಆಕಾಶಾತ್ ಪತಿತಂ ತೋಯಂ, ಯಥಾ ಗಚ್ಛತಿ ಸಾಗರಮ್।
ಸರ್ವ-ದೇವ ನಮಸ್ಕಾರಃ ಕೇಶವಂ ಪ್ರತಿಗಚ್ಛತಿ॥141

-ಅನಾಮಕ ಕವಿಯ ಜನಪ್ರಿಯ ಸ್ತೋತ್ರ

ಕೆಳ ಬಿದ್ದು ಬಾಂದಳದಿಂದ, ಹರಿದು ತೊರೆ ನದಿಯತ್ತ,
ಕೊನೆಗೊಮ್ಮೆ ಸೇರುವುದು ಸಾಗರವ ನೀರು;
ಈ ಪರಿಯೆ ನಮ್ಮೆಲ್ಲ ನಮನಗಳು ಸಾಗುವುವು-
'ಅವನ'ತ್ತ; ದೇವನೊಬ್ಬನೆ, ಕರೆಯೆ ನಾಮ ನೂರು!

೧೪೨ :: ಶಿವ ಶಿರವ ತೋಯಿಸುವೆ

ಆಪಾತಾಲ-ನಭಃ-ಸ್ಥಲಾಂತ ಭುವನ ಬ್ರಹ್ಮಾಂಡಂ ಆವಿಃಸ್ಫುರತ್।
ಜ್ಯೋತಿಃಸ್ಫಾಟಿಕ-ಲಿಂಗ-ಮೌಳಿವಿಲಸದ್ ಪೂರ್ಣೇನ್ದುವಾನ್ತಾಮೃತೈಃ।
ಅಸ್ತೋಕಾಪ್ಲುತಂ ಏಕಂ ಈಶಾನಂ ಅನಿಶಂ
ರುದ್ರಾನುವಾಕಾನ್ ಜಪನ್
ಧ್ಯಾಯೇದ್ ಈಪ್ಸಿತ-ಸಿದ್ಧಯೇ ಅದ್ರುತಪದಂ
ವಿಪ್ರೋ ಅಭಿಷಿಂಚೇತ್ ಶಿವಮ್॥142

ಧ್ಯಾನಶ್ಲೋಕ, ರುದ್ರಾಧ್ಯಾಯ

ಪಾತಾಳದಿಂದೊಡೆದು ಆಗಸದ ತುದಿ ಮುಟ್ಟಿ, ಆವರಿಸಿ ಜಗದಗಲ,
ಬ್ರಹ್ಮಾಂಡವನೇ ಬೆಳಗುತಿಹ, ಶುದ್ಧತೇಜೋಮಯಸ್ಫಟಿಕ
ಲಿಂಗವನೆ ಹೃದಯದಾಳದಲೆನ್ನ ನೆನೆ ನೆನೆಯುತಿರುವೆ;
ತೋಯಿಸುವೆ ಪೂರ್ಣಚಂದ್ರನ, ಸುಧೆಯ ಬಿಂದುಗಳಿಂದ
ಎಡೆಬಿಡದೆ ನಾನಿಂತು ಈ ನೀರ ಧಾರೆಯನ್ನೆರೆವೆ-
ರುದ್ರಾನುವಾಕನು ಜಪಿಸಿ, ಬಿನ್ನವಿಸಿ ಹರಕೆಗಳ, ಶಿವನ ಶಿರದಿ!

೧೪೩ :: ನೀವವನ ನೋಡಿಲ್ಲ?

ಅಸೌ ಯಸ್ ತಾಮ್ರೋ ಅರುಣ ಉತ ಬಭ್ರುಸ್ ಸುಮಂಗಲಃ। .. ..
ಅಸೌ ಯೋ ಅವಸರ್ಪತಿ ನೀಲಗ್ರೀವೋ ವಿಲೋಹಿತಃ।
ಉತ ಏನಂ ಗೋಪಾ ಅದೃಶನ್ನ್ ಅದೃಶನ್ ಉದಹಾರ್ಯಃ।
ಉತೈನಂ ವಿಶ್ವಾ ಭೂತಾನಿ ಸ ದ್ರಷ್ಟೊ ಮೃಡಯಾತಿ ನಃ॥143

ಶತರುದ್ರೀಯ ೭-೮

ನೋಡಿಲ್ಲ? ಯಾರವರು?
ತಲೆಯೆತ್ತಿ ನೋಡಿ ನಿನಗೆ, ಕೈಮುಗಿಯದವರಾರು?
ಹೊಳೆಯಿಂದ ನೀರ ಹೊತ್ತಾರೆ ಹೊತ್ತು ತರುವ ತೆತ್ತಿಗರೆ,
ಮತ್ತೆ ಹೆಂಗಸರೆ, ತುರುಗಾಹಿ ಹುಡುಗರೆ,
ಹೊಲಗದ್ದೆತೋಟದಲಿ ಕೆಲಸ ಮಾಡುವ ಜನರೆ,
ಹುಟ್ಟುತಲೆ ತಾಮ್ರದ ಕೆಂಪು,
ಥಟ್ಟನೆ ಬಂಗಾರ ಬಣ್ಣಕೆ ತಿರುಗಿದವನೂ ನೀನೆ,
ಕೊನೆಗೆ ನಸುಗೆಂಪಿನ ಕೆನೆ,
ಏನು ಸೋಜಿಗ ನಿನದು ಶುಭಕರನೆ, ಶಿವನೇ!?

೧೪೪ :: ಅಲ್ಲೆಲ್ಲ ನಾನಿರುವೆ!

ಯದ್ ಯದ್ ವಿಭೂತಿಮತ್ ಸತ್ತ್ವಂ
ಶ್ರೀಮದ್ ಊರ್ಜಿತಂ ಏವ ವಾ।
ತತ್ ತದ ಏವ ಅವಗಚ್ಛ ತ್ವಂ
ಮಮ ತೇಜೋ ಅಂಶ ಸಂಭವಮ್॥144

ಭಗವದ್ಗೀತೆ ೧೦:೪೧

ಅಸಮಾನ್ಯ ಪ್ರಭೆ, ಶಕ್ತಿ ಎಲ್ಲೆಲ್ಲಿ ತೋರುವುದೋ,
ಎಲ್ಲಿ ಸುಡುಗತ್ತಲನು ತೂರಿ ಬರುವುದೋ ಬೆಳಕು,
ಸುಖ ದುಃಖ ನೊರೆ-ಅಲೆಯ ಈ ಕಡಲ ಒಡಲಿನಲಿ
ಬಿಡುಗಡೆಯ ಆ ಕಡೆಗೆ ನಡೆಯೆ ಸೇತುವೆಯೊಂದ
ಹೂಡಿ, ಬೆಂಗಾಡಿನ ನಡುವೆ ಹೆದ್ದಾರಿ ತೋಡುವನೋ,
ಅಡ್ಡ ಕಳೆ ಕೊಳೆ ಮುಳಿವನ್ನ ಕುಟ್ಟಿ ಹುಡಿ ಹುಡಿ ಮಾಡಿ
ಹತ್ತಲೆತ್ತರ ಬೆಟ್ಟ ಮೆಟ್ಟಲನು ಕಟ್ಟುವನೋ,
ಬೀಸುವುದೋ ಎಲ್ಲಿ ಶಾಂತಿಯ ಕಂಪು, ನೆಮ್ಮದಿಯ ತಂಪು -
ಅಲ್ಲೆಲ್ಲ ದೈವಾಂಶ ತೇಜದ ಬೀಜ ಮೊಳೆವುದ ಕಾಣ!

೧೪೫ :: ನೆಲೆ ನಿಲ್ಲು ಸ್ಥಿರವಾಗಿ

ವಾಂಛಿತದಾಯಿನಿ, ಚಂಚಲಾ ತ್ವಂ- ಇತಿ
ಕಿಂಚಿದ್ ಅಸ್ತಿ ಭಯಂ ಅಜ್ಞ-ಜನಾನಾಮ್।
ತ್ವಂ ಚ ಮದೀಯೇ ಗೃಹೇ ಸತತಂ
ನಿಶ್ಚಂಚಲಾ ಭವ ಪ್ರಸನ್ನಮುಖೀ॥
-ತ್ವಾಂ ವನ್ದೇ ಮಾತರಮ್ ॥145

ಶ್ರೀ ಪ್ರಸನ್ನ ತೀರ್ಥರ ಲಕ್ಷ್ಮೀಸ್ತುತಿ

ಕೇಳಿದ್ದ ಕೊಡುವವಳೆ, ಚಂಚಲಳು ನೀನೆಂದು
ಕೊಂಚ ಭಯ ಪಡುವವರು ತಿಳಿಯದವರು;
ಇರಲಿ, ನೀ ಬಂದೆನ್ನ ಹೃದಯದಲಿ ಮನೆ ಮಾಡು,
ನಿಶ್ಚಂಚಲೆಯೆ ಆಗಿ ನೆಲಸು, ಸತತ ಹರಸು!

೧೪೬ :: ಎಲ್ಲಿರುವೆ ನೀ ಲಕ್ಷ್ಮಿ?

ಯಾ ಶ್ರೀಃ ಸ್ವಯಂ ಸುಕೃತೀನಾಂ ಭವನೇಷು,
ಅಲಕ್ಷ್ಮೀಃ ಪಾಪಾತ್ಮನಾಂ, ಕೃತಧಿಯಾ ಹೃದಯೇಷು ಬುದ್ಧಿಃ।
ಶ್ರದ್ಧಾ ಸತಾಂ, ಕುಲಜನ-ಪ್ರಭವಸ್ಯ ಲಜ್ಜಾ,
ತಾಂ ತ್ವಾಂ ನತಾಃ ಸ್ಮ, ಪರಿಪಾಲಯ, ದೇವಿ, ವಿಶ್ವಮ್ ॥146

ಮಾರ್ಕಂಡೇಯ ಪುರಾಣ, ದೇವೀ ಮಾಹಾತ್ಮ್ಯ ೪:೫

ಪುಣ್ಯಕಾರ್ಯಗಳ ಎಸಗುವರ ನೆಲೆಯಲ್ಲಿ ಸಿರಿ, ಸಂಪತ್ತು;
ಪಾಪಿಗಳ ಸೆರೆಯಲ್ಲಿ ದಾರಿದ್ರ್ಯದಾಪತ್ತು, ದೈನ್ಯ ವಿಪತ್ತು;
ಸುಮನಸರ ಹೃದಯದಲಿ ಬೆಳಗುವ ಬುದ್ಧಿಯ ಮಹತ್ತು;
ಸಜ್ಜನರ ಶ್ರದ್ಧೆಯ ಸೊತ್ತು,
ಕುಲೀನ ನಾರಿಯ ಸಹಜ ಲಜ್ಜೆಯ ಒತ್ತು-
ಹೀಗೆ ಬಗೆ ಬಗೆ ರೂಪದಲಿರುವೆ;
ಲಕ್ಷ್ಮಿ, ಜಗವ ಕಾಪಾಡು, ಹೊತ್ತು!

೧೪೭ :: ಒಲಿವೆ ಯಾರಿಗೆ ನೀನು, ಹೇಳು, ಸಿರಿಯೆ?

ಕಾನಿ ಇಹ ಭೂತಾನಿ ಉಪಸೇವಸೇ ತ್ವಂ?
ಸಂ-ತಿಷ್ಠಸೇ ಕಾನ್ ಇವ, ಸೇವಸೇ ತ್ವಂ?
ಕಾನಿ ತ್ರಿಲೋಕೇಶ್ವರ-ಭೂತಕಾನ್ತೇ,
ತತ್ತ್ವೇನ ಮೇ ಬ್ರೂಹಿ, ಮಹರ್ಷಿ-ಕನ್ಯೇ॥147

ಮಹಾಭಾರತ, ಅನುಶಾಸನ ಪರ್ವ ೧೧:೪

ಯಾರು ಯಾರಿಗೆ ಒಲಿದು ನೀ ನಲಿವೆ, ಸಿರಿಯೆ?
ಯಾರು ಯಾರಲ್ಲಿರುವೆ ? ನಿಲ್ಲುವೆಯೆ ನೆಲಸಿ?
ಜೀವ-ಜಡ-ಜಗ-ರಕ್ಷಣೆಯ ಪೂರಕದ ಓ ತತ್ತ್ವವೇ,
ಲಕ್ಷ್ಮಿಯೆ, ನಿನ್ನ ಮನದಿಂಗಿತವ ತಿಳಿಯ ಹೇಳೆ!

೧೪೮ :: ಇದ್ದರೂ ನಲಿವುದು ಎಲ್ಲಿ?

ದಕ್ಷಾಣಾಂ ಸಾಹಸೀನಾಂ ಚ ಸ್ವ-ವಿಶ್ವಾಸ-ಮಹಾತ್ಮನಾಂ
ಕ್ರೋಧ-ಮೋಹ-ಮಾತ್ಸರ್ಯಾನಾಂ ವರ್ಜಿತಾನಾಂ ಚ ಶತ್ರುತಾಂ।
ಕಾರ್ಯೇಷು ಕೌಶಲಂ ಯತ್ರ ಲಾವಣ್ಯ ಸಮುಪಸ್ಥಿತಂ
ಲಕ್ಷ್ಮೀರ್ ವಸತಿ ಸಂತುಷ್ಟಾ ಹೃದಯೇ ಅನಿಶಂ ಧ್ರುವಮ್॥148

ಹರಿಹರೇಶ್ವರ

ಆತ್ಮವಿಶ್ವಾಸಿಗಳಲ್ಲಿ, ದಕ್ಷರಲ್ಲಿ, ಸಾಹಸಿಗಳಲಿ,
ಕಾರ್ಯಕುಶಲರಲಿ, ಜಿತೇಂದ್ರಿಯರಲಿ, ಕೃತಜ್ಞರಲಿ,
ಕ್ರೋಧ-ರಹಿತರಲಿ, ಮೋಹ-ಮರೆತವರಲ್ಲಿ,
ಮದವ ತುಳಿದವರಲ್ಲಿ, ಸೌಂದರ್ಯ ನೆಲೆವೀಡಾದಲ್ಲಿ,
ಶುಚಿ ಇದ್ದ ಕಡೆಯಲ್ಲಿ, ಗುಣದ ಖಣಿಯಾದವರಲ್ಲಿ
-ಲಕ್ಷ್ಮಿ ಇರಲು ಸಂತೋಷಿಸುವಳು.

೧೪೯ :: ಕರೆದವರ ಎದುರಿಗೆ ಬರುವೆ

ಅನನ್ಯಾಶ್ಚಿನ್ತಯನ್ತಾನಾಂ ತು ತ್ವಂ ಪ್ರಪನ್ನಶರಣಾರ್ಥಿನಾಂ।
ಭಕ್ತಾನಾಂ ಹೃದಯೇ ತಿಷ್ಠನ್ ಯದ್ ಭಾವಂ ತದ್ ಪ್ರಕಾಶಸಿ॥149

ಹರಿಹರೇಶ್ವರ

ನಂಬಿ ಮೊರೆ ಹೊಕ್ಕವರ, ತುಂಬಿ ಭಕ್ತಿಯನೊಂದೆ
ಬೇರೆಲ್ಲದಕೆ ಬರಿದಾಗಿಸಿಕೊಂಡವರ ಹೃದಯದಲ್ಲಿ,
ಓಗೊಡುತ ಭಾವಪರವಶ ಕರೆಗೆ ನೆನೆವ ಮನದಲ್ಲಿ
ಬೇಕೆಂದ ರೂಪವ ತಾಳಿ ನೀ ಮೈದೋರುವೆ!

೧೫೦ :: ಗಣಗಳೊಡೆಯನೆ ಬಾ!

ಓಂ ಗಣಾನಾಂ ತ್ವಾ ಗಣಪತಿಂ ಹವಾಮಹೇ।
ಕವಿಂ ಕವೀನಾಂ ಉಪವಶ್ರಮಸ್ತಮಮ್।
ಜ್ಯೇಷ್ಠರಾಜಂ ಬ್ರಹ್ಮಣಾಂ।
ಬ್ರಹ್ಮಣಸ್ಪತಿ, ಆ ನಃ ಶೃಣ್ವನ್ ಊತಿಭಿಃ ಸೀದ ಸಾದನಮ್॥150

ಋಗ್ವೇದ ೨:೨೩:೧

ಗಣಪತಿಯೆ, ಗಣಗಳೊಡೆಯನೆ ಬಾರ, ಇತ್ತ ಸ್ವಾಗತ ನಿನಗೆ,
ಕಡುಜಾಣ ಜಾಣರಲಿ, ಎಲ್ಲೆಡೆ ಎಲ್ಲರಲಿ ಹೆಸರಾಂತನಿರುವೆ।
ಹಿರಿಯರಸ ನೀ ಹಾಡು ಹೊಗಳಿಕೆ ಬಿನ್ನಪವದೆಲ್ಲದಕು,
ಕೇಳು ಬಾ, ನೆರವೀಯೆ, ಇಲ್ಲಿ ಕುಳ್ಳಿರು ನೀನು, ಪೂಜೆಗೊಳ್ಳು॥

೧೫೧ :: ಎಲ್ಲಿ? ದೇವರು ಎಲ್ಲಿ?

ಸ್ವಾಧೀನಂ ಸಂ-ಅತಿಕ್ರಮ್ಯ ಮಾತರಂ ಪಿತರಂ ಗುರುಮ್।
ಅಸ್ವಾಧೀನಂ ಕಥಂ ದೈವಂ ಪ್ರಕಾರೈರ್ ಅಭಿರಾಧ್ಯತೇ?॥151

-ವಾಲ್ಮೀಕಿ, ರಾಮಾಯಣ, ಬಾಲಕಾಂಡ ೩೦:೩೩

ಕಂಡಿಲ್ಲವೆನ್ನುವಿಯಾ ದೇವರನು ನಾನೆಲ್ಲೂ?
ಕಣ್ಣೆದುರೇ ಇಹರಲ್ಲ ತಾಯಿ, ತಂದೆ, ಗುರು;
ಇವರ ಮಾತನು ಮೀರಿ ನಡೆವುದಾದರೂ ಏಕೆ?
ಕೈಗೆ ಎಟುಕದ ಬೇರೆ ದೇವರಾರಾಧನೆ ಬೇಕೆ?

೧೫೨ :: ದೀಪರೂಪಳೆ ಬಾ ನೋಡು!

ಭೋ ದೇವಿ, ದೀಪ ರೂಪಸ್ ತ್ವಂ,
ಕರ್ಮಸಾಕ್ಷೀ ಹಿ ಅವಿಘ್ನಕೃತ್।
ಯಾವತ್ ಪೂಜಾ ಸಮಾಪ್ತಿಸ್ ಸ್ಯಾತ್,
ತಾವತ್ ತ್ವಂ ಸುಸ್ಥಿರಾ ಭವ॥152

ದೇವತಾರ್ಚನ ವಿಧಿ

ಬೆಳಕ ಬೀರುವ ದೀಪ, ನೀನು ದೇವಿ ಸ್ವರೂಪ,
ಬಾರೆ, ಬಾ, ನಾ ಮಾಡುವುದೆಲ್ಲವನು, ಕಣ್ಣಾರೆ ನೋಡು!
ತುಂಬು ಕರುಣೆಯ ತೋರು, ಸ್ಥಿರವಾಗಿ ನೀ ನೆಲಸು, 'ಅಡೆತಡೆಗಳಿಲ್ಲದೆ ಪೂಜೆ ಕೊನೆಗೊಳ್ಳಲೆ'ದು ಹರಸು!

೧೫೩ :: ಕರ್ಪೂರದಂದದಿ ಬೆಳ್ಳಿಬೆಳ್ಳಗಿನ ಖಣಿಯೇ!

ಕರ್ಪೂರಗೌರಂ ಕರುಣಾವತಾರಂ
ಸಂಸಾರಸಾರಂ ಭುಜಗೇಂದ್ರಹಾರಮ್।
ಸದಾವಸಂತಂ ಹೃದಯಾರವಿಂದೇ
ಭವಂ ಭವಾನೀಸಹಿತಂ ನಮಾಮಿ॥153

ದೇವತಾರ್ಚನ ವಿಧಿ

ಕರುಣೆಯೇ ಒಡಲಾಗಿ ಬಂದೆದುರು ನಿಂದವನೇ, ಬಂಧುರನೇ,
ಬೆಳ್ಳಿಬಿಳುಪಿನ ಕರ್ಪೂರದಂದದ ಸುಂದರನೇ, ಸಾಂಬಶಿವನೇ
ಸಂಸಾರ ಸಾರ ಹಾವ ಕೊರಳ ಸರ ತೊಟ್ಟವನೇ, ಗಟ್ಟಿಗನೇ,
ನನ್ನೆದೆಯ ಕಮಲದಲಿ ನೆಲಸಿಹನೇ, ಶಂ-ಕರನೇ ನಮನ

೧೫೪ :: ಹರಿ ಹರ ವಿರಂಚಿ -ಎಲ್ಲವೂ ನೀನೆ!

ದೀಪಂ ಜ್ಯೋತಿ ಪರಬ್ರಹ್ಮಾ, ದೀಪಂ ಜ್ಯೋತಿ ಜನಾರ್ದನ।
ದೀಪೇನ ಹರತೇ ಪಾಪಂ, ತಸ್ಮಾತ್ ದೀಪಂ ನಮೋಸ್ತು ತೇ॥154

ದೇವತಾರ್ಚನ ವಿಧಿ

ಹುಟ್ಟು, ಉಳಿವು, ಅಳಿವಿನ ರೂಪ, ದೀಪವೇ, ಮುನ್ನ;
ಅರಿಯದೆಯೇ ನಾ ಎಸಗಿರಬಹುದು ತಪ್ಪು ಕೆಲವನ್ನ;
ತಿಳಿದೋ ತಿಳಿಯದೆಯೋ ಆದ ಆಗುವ ಎಲ್ಲ ನನ್ನ
ತಪ್ಪುನೆಪ್ಪುಗಳನೊಪ್ಪಿ ಮನ್ನಿಸಿ ತೋರು ಬೆಳಕನ್ನ
