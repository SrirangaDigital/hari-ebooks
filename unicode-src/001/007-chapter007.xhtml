೭ಔದಾರ್ಯ-ಕೃತಜ್ಞತೆ

೪೫ :: ತನ್ನಂತೆ ಪರರ ಬಗೆ

ಆತ್ಮವತ್ ಸರ್ವಭೂತಾನಿ, ಪರದ್ರವ್ಯಾಣಿ ಲೋಷ್ಟವತ್
ಸ್ವಭಾವಾದ್ ಏವ ನೋ ಭೀತ್ಯ ಯಃ ಪಶ್ಯತಿ ಸ ಪಶ್ಯತಿ॥045

-ಸಂಗ್ರಹ ಯೋಗವಾಸಿಷ್ಠ ೧೬:೯ ಪುಟ ೧೬೧

ಕಾಣಬಲ್ಲರೋ ಯಾರು ತನ್ನಂತೆ ಪರರನ್ನು,
ಮಣ್ಣಾಗಿ ಕಡೆಗಣಿಸಿ ತನ್ನದಲ್ಲದವನು;
ಯಾರಿಗೂ ಅಂಜಿಯೇನಲ್ಲ ಯಾರ ತಣಿಸಲೂ ಅಲ್ಲ-
ಸಹಜವಾಗಿಯೇ ಇರಬಲ್ಲ ಅವನೆಲ್ಲ ಬಲ್ಲ!

೪೬ :: ಎಲ್ಲರೂ ನಮ್ಮವರೆ, ನಮ್ಮ ಮನೆಯವರೇ

ಅಯಂ ನಿಜೋ, ಪರೋ ವಾ -ಇತಿ ಗಣನಾ ಲಘುಚೇತಸಾಂ।
ಉದಾರಚರಿತಾನಾಂ ತು ವಸುಧಾ ಏವ ಕುಟುಂಬಕಮ್॥046

ಹಿತೋಪದೇಶ, ಸಂಧಿ ೧೩೧

ಇವರು, ಬಿಡಿ, ನಮ್ಮವರು; ಆ ಅವರೇ? ಬೇರೆ -
ಎಂದೆಲ್ಲ ಬೇರೆಣಿಸಿ, ಪರಿಗಣಿಪ ಮನಸು ಕಿರಿದು;
ಎಲ್ಲರೂ ನಮ್ಮವರೇ, ಇಡೀ ಜಗವೇ ನನ್ನ ಮನೆ-
ಎಂದು ಕಾಣುವ ಬಗೆವ ಹೃದಯ ತುಂಬ ಹಿರಿದು.

೪೭ :: ಪಾಪ-ಪುಣ್ಯದ ಚಹರೆ

ಶ್ಲೋಕಾರ್ಧೇನ ಪ್ರವಕ್ಷ್ಯಾಮಿ ಯದ್ ಉಕ್ತಂ ಗ್ರಂಥ ಕೋಟಿಭಿಃ।
ಪರೋಪಕಾರಾಯ ಪುಣ್ಯಾಯ, ಪಾಪಾಯ ಪರಪೀಡನಂ॥047

ವ್ಯಾಸ, ಮಹಾಭಾರತ

ಕೋಟಿ ಗ್ರಂಥಗಳಲ್ಲಿ ಹೇಳಿರುವ ಮಾತನ್ನೇ
ಒಂದರ್ಧ ಶ್ಲೋಕದಲ್ಲಿ ಹೇಳುವೆನು ಕೇಳಿ;
ಇತರರಿಗುಪಕರಿಸುವುದೇ ಪುಣ್ಯ, ನಿಜದ ನಿಜ;
ಪರರ ನೋಯಿಸುವುದೆಲ್ಲಾ ಪಾಪಕೇಳಿ!

೪೮ :: ಪರರಿಗೆಂತೋ ಅಂತೆ ತಾನು ತನಗೂ

ಶ್ರೂಯತಾಂ ಧರ್ಮಸರ್ವಸ್ವಂ ಶ್ರುತ್ವಾ ಚೈವ ಅವಧಾರ್ಯತಾಂ।
ಆತ್ಮನ; ಪ್ರತಿಕೂಲಾನಿ ಪರೇಷಾಂ ನ ಸಮಾಚರೇತ್॥048

ಪಂಚತಂತ್ರ ಕಾಕ-ಉಲೂಕೀಯ ೧೦೨

ಇಲ್ಲಿ ಕೇಳಿರಿ ಜನರೆ, ಇದೆ ಎಲ್ಲ ಧರ್ಮದ ಸಾರ;
ಕೇಳಿ, ನಿಮ್ಮ ನಡೆ ನುಡಿಗಳಲಿದನು ಅಳವಡಿಸಿಕೊಳ್ಳಿ.
ನಿಮಗೇ ಹೀಗಾಗುವುದೇನೂ ಬೇಡ, ಅದು ಸಲ್ಲದೆನಿಸಿದರೆ
ಬೇರೆಯವರ ಬಳಿಯೂ ಆ ರೀತಿ ನೀವು ನಡೆಯ ಬೇಡಿ!

೪೯ :: ತಮಗಾಗಿಯೇ ಎಲ್ಲವೂ ಅಲ್ಲ

ಪಿಬಂತಿ ನದ್ಯಃ ಸ್ವಯಂ ಏವ ನ ಅಂಭಃ
ಸ್ವಯಂ ನ ಖಾದಂತಿ ಫಲಾನಿ ವೃಕ್ಷಾಃ।
ನ ಅದಂತಿ ಸಸ್ಯಂ ಖಲು ವಾರಿವಾಹಾಃ
ಪರೋಪಕಾರಾಯ ಸತಾಂ ವಿಭೂತಯಃ॥049

ಸುಭಾಷಿತ ರತ್ನಭಾಂಡಾಗಾರ ೪೯:೭೦

ನದಿ ತನ್ನ ನೀರನೆಂದಾದರು ತಾನೇ ಕುಡಿದುದುಂಟೇ?
ತಾ ಬಿಟ್ಟ ಹಣ್ಣುಗಳ ಮರ ತಿನುವುದೇನು?
ತಾನೆ ತಿನ್ನಲು ಬೆಳೆಯ, ಮೋಡ ಸುರಿವುದೆ ಮಳೆಯ?
ಪರರ ನೆರವಿಗೆ ಎಂದೇ ಸಜ್ಜನರ ಸವಿಜೇನು!

೫೦ :: ಮರೆಯುವರೆ ಉಪಕಾರವನು

ಪ್ರಥಮ ವಯಸಿ ಪೀತಂ ತೋಯಂ ಅಲ್ಪಂ ಸ್ಮರನ್ತಃ,
ಶಿರಸಿ ನಿಹಿತ ಭಾರಾ ನಾರಿಕೇಳಾ ನರಾಣಾಮ್।
ದದತಿ ಜಲಂ ಅನಲ್ಪಂ ಆಸ್ವಾದಂ ಆಜೀವಿತಾನ್ತಂ,
ನ ಹಿ ಕೃತಂ ಉಪಕಾರಂ ಸಾಧವೋ ವಿಸ್ಮರನ್ತಿ॥050

ವಿಕ್ರಮ ಚರಿತ ೯೧

ಎಳೆಯತನದಲಿ ಕುಡಿದ ನೀರನನವರತ ನೆನೆಯುತ ಮತ್ತೆ,
ತಲೆಯಲದ ಹೊತ್ತು ತೆಂಗು ಬಾನೆತ್ತರಕೇರಿದರು ತಾನು,
ಇರುವರೆಗೆ ಕೊಡುತಲಿದೆ ಕೊಡ ಕೊಡ ತುಂಬಿ ಸವಿಜಲವ-
ಇನಿತಾದರೇನು ಪಡೆದ ನೆರವ ಸಜ್ಜನರು ಮರೆವರೇನು?

೫೧ :: ನೀನಿಚ್ಛಿಸಿದಂತೆ ನನ್ನ ಬಾಳು

ಯದ್ ಯದ್ ಕರ್ಮಂ ಅಭೀಪ್ಸಿತಂ ತು
ಅಕರವಂ ಸರ್ವಂ ಹಿ ತ್ವಯಾ ಏವ ಇಚ್ಛಯಾ;
ಮನ್ಯೇ ಸಾಧಿತ ಧ್ಯೇಯಂ ಅಖಿಲಂ
ತ್ವತ್ ಪ್ರಸಾದ ಕೃಪಯಾ ಧಿಯಾ-
ಅನುಗ್ರಹಿತೋಽಸ್ಮಿ, ಹೇ ಭಗವನ್,
ಕೃತಜ್ಞತಾಂ ಮಾಂ ಸ್ವೀಕುರು॥051

-ಹರಿಹರೇಶ್ವರ

ನಾನೇನ ಬಯಸಿದೆನೋ ಅದನೆಲ್ಲ ಮಾಡಿದೆನೆ?
ಇಲ್ಲ, ನೀನಿಚ್ಛಿಸಿದಂತೆ ಕುಣಿದು, ಮೆರೆದೆ;
ಏನೆಲ್ಲ ಸಾಧಿಸಿದೆ, ಅದು ಅಲ್ಪವಿರಬಹುದು-
ಅದು ನಿನ್ನ ಕರುಣೆ ಪ್ರಸಾದ; ಹೆಮ್ಮೆ ನನಗೆ.
ಅನುಗ್ರಹಿಸಿ ಕಾಯ್ದೆಯೈ ಓ ದೇವ, ಮಾದೇವ,
ನನ್ನ ನೆನಕೆಗಳನ್ನು ಸ್ವೀಕರಿಸಿದೋ!

೫೨ :: ಮೌಲ್ಯಾಧಾರಿತ ಬದುಕಿನ ಬಯಕೆ

ಇಚ್ಛಾ ಜ್ಞಾನ ಕ್ರಿಯಾ ಶಕ್ತಿಂ ಅಮಲಂ
ಅದದಾಃ ತ್ವಂ ಮೇಧಾಂ, ಯತ್ನತೋ
ಮೌಲ್ಯಾಧಾರಿತ ಜೀವನಂ ಯಾಮಿ ಅಹಂ,
ಅನುಭಾವ್ಯ ನಿಶ್ರೇಯಸಂ ಸಂಪ್ರತಿ
ಅನುಗ್ರಹಿತೋಽಸ್ಮಿ, ಹೇ ಭಗವನ್,
ಕೃತಜ್ಞತಾಂ ಮಾಂ ಸ್ವೀಕುರು॥052

-ಹರಿಹರೇಶ್ವರ

ಇಚ್ಛಾ ಜ್ಞಾನ ಕ್ರಿಯೆಯ ಮೂರೂ ಶಕ್ತಿಗಳೊಡನೆ
ನಾ ಬೆಳೆದೆ- ನೀನವನಿತ್ತು ಕೈ ಹಿಡಿದು ನಡೆಸೆ;
ಮೌಲ್ಯಾಧಾರಿತ ಬದುಕ ಬಾಳಲಾಶಿಸಿ ಇಟ್ಟ
ಹೆಜ್ಜೆಗಳ ಒಜ್ಜೆಗಳೋ ಬಿಡುಗಡೆಯ ಕಡೆಗೆ.
ಅನುಗ್ರಹಿಸಿ ಕಾಯ್ದೆಯೈ, ಓ ದೇವ, ಮಾದೇವ,
ನನ್ನ ಕೃತಜ್ಞತೆಗಳನು ಸ್ವೀಕರಿಸಿದೋ! ॥೫॥
