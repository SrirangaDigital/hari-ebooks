೧೯ಮಿಣುಕು-ತುಣುಕು

೧೬೭ :: ಚೆನ್ನುಡಿಯ ಸವಿಗಿಲ್ಲ ಸಾಟಿ ಏನೂ

ದ್ರಾಕ್ಷಾ ಮ್ಲಾನಮುಖೀ ಜಾತಾ, ಶರ್ಕರಾ ಚ ಅಶ್ಮತಾಂ ಗತಾ।
ಸುಭಾಷಿತರಸಂ ದೃಷ್ಟ್ವಾ ಸುಧಾ ಭೀತಾ ದಿವಂ ಗತಾ॥167

ಸುಭಾಷಿತ ರತ್ನಭಾಂಡಾಗಾರ ೨೯:೧೦

ಚೆನ್ನುಡಿಯ ರಸದೆದುರು ಏನು ಆಯಿತು ಗೊತ್ತ?-
ಮುಖ ಮುದುಡಿ ಸಿಹಿ ದ್ರಾಕ್ಷಿ ಹುಳಿಗೊಂಡಿತು;
ಜಾರಿ ಸಕ್ಕರೆಯ ಸವಿ ಬರಿ ಮಣ್ಣ ಹುಡಿಯೆನಿಸಿ,
ಹೆದರಿ ಸುಧೆ ಸಗ್ಗಕೆ ಜಗ್ಗನೆ ಹಾರಿ ಮರೆಯಾಯಿತು!

೧೬೮ :: ಹೀಗೇಕೆ ಜೇನ್ನುಡಿಯ ಗತಿಯಾಯ್ತು?

ಬೋದ್ಧಾರೋ ಮತ್ಸರಗ್ರಸ್ತಾಃ,
ಪ್ರಭವಃ ಸ್ಮಯ ದೂಷಿತಾಃ।
ಅಬೋಧೋ ಅಪಹತಾಶ್ಚ ಅನ್ಯೇ,
ಜೀರ್ಣಂ ಅಂಗೇ ಸುಭಾಷಿತಮ್॥168

ಭರ್ತೃಹರಿ ನೀತಿ ಶತಕ ೧:೧

ತಿಳಿದವರು ಇಲ್ಲವೇನೆಂದಲ್ಲ, ಸಹಿಸರವರಿಗೆ ಹೊಟ್ಟೆಯುರಿ;
ಉಳ್ಳವರು ಇಲ್ಲವೇನೆಂದಲ್ಲ, ಅವರ ಒಣಜಂಬಕೆಣೆಯಿಲ್ಲ;
ಉಳಿದವರ ಪಾಡು, ಪಾಪ, ಅರೆಬರೆ ತಿಳಿವಳಿಕೆಯ ಹಿಡಿಶಾಪ-
ಮುಪ್ಪಡರಿ ಚೆನ್ನುಡಿ ಹೀಗೆ ಯಾರ ಕಣ್ಣಿಗೂ ಬೀಳದಾಯ್ತೆ?

೧೬೯ :: ಮಾತು ಬಲ್ಲವನೆಲ್ಲ ಊರ ಗೆಲ್ಲಬಲ್ಲ

ಧತ್ತೇ ಶ್ರಿಯಂ, ಸೃಜತಿ, ಕೀರ್ತಿ ಅಘಂ ಲುನೀತೇ,
ಮಿತ್ರತ್ವಂ ಆನಯತಿ ಹನ್ತ, ವಿರೋಧಿನೋಽಪಿ।
ಯಾತಿ ಅಧ್ನಭಿಃ ಪ್ರತಿಪದಂ ಸುಮನೋನುಕೂಲೈರ್
ಗೌ: ಕಾಮಧುಕ್ ಕಂ ಇವ ನ ಅಪಹರತಿ ಅನರ್ಥಮ್॥169

ಕಲ್ಹಣ, ರಾಜತರಂಗಿಣೀ ೭:೭೮೯

ಮಾತೇ ಮುತ್ತು, ಸಂಪತ್ತ ಹೊತ್ತು ಬರುವುದು ಅದೇನೇ,
ಮನ್ನಣೆಯ ತಾ ತಂದು, ತೊಳೆವುದು ಪಾಪದ ಕರಿ ನಿಶಾನೆ
ಹಗೆ ಹೊಗೆಯ ನಡುವೆಯೂ ಗೆಳೆತನದ ಬೆಸುಗೆಯ ಹೂಡಿ,
ಅಡಿಗಡಿಗೆ ಬಲ್ಲವರು ಮೆಚ್ಚುವ ತೆರದಿ ಮೆರೆವೀ ಕಾಮಧೇನು
-ಗೋಮಾತೆ ಯಾವ ತಪ್ಪನು ತಾನೇ ಸರಿಪಡಿಸದಿರದು?

೧೭೦ :: ಕಳೆ ಕೊಳೆಯ ತೊಳೆವ ಹೊಳೆ

ಆಪಃ ಪುನನ್ತು ಪೃಥಿವೀಂ, ಪೃಥಿವೀ ಪೂತಾ ಪುನಾತು ಮಾಮ್।
ಪುನನ್ತು ಬ್ರಹ್ಮಣಸ್ಪತಿರ್, ಬ್ರಹ್ಮ, ಪೂತಾ ಪುನಾತು ಮಾಮ್॥170

ತೈತ್ತಿರೀಯ ಆರಣ್ಯಕ ೧೦:೨೩:೧; ಮಹಾನಾ ಉಪ. ೩೦:೧

ಸುಪವಿತ್ರ ಜಲ ನೆಲವ ಶುದ್ಧಿಗೊಳಿಸಲಿ ಹರಿದು,
ಈ ನೆಲದ ಮೇಲಡಿಯಿಟ್ಟು ನಾ ಪುನೀತನಾದೆ!
ಎಲ್ಲ ದೈವಾಂಶಗಳ ಪಾವಿತ್ರ್ಯ ಇನ್ನು ಎಡೆಬಿಡದೆನ್ನ
ಒಳಹೊರಗಿನೆಲ್ಲ ಕಳೆ ಕೊಳೆಯ ತೊಳೆದಿರಿಸಲಿ!

೧೭೧ :: ನೀರೆ, ಮನ್ನಿಸು ಬಾರೆ

ಯದ್ ಉಚ್ಛಿಷ್ಟಂ ಅಭೋಜ್ಯಂ ಯದ್ ವಾ ದುಷ್ಚರಿತಂ ಮಮ।
ಸರ್ವಂ ಪುನನ್ತು ಮಾಂ ಆಪೋ, ಅಸತಾಂ ಚ ಪ್ರತಿಗ್ರಹಂ॥171

ಬೌಧಾಯನ ಬ್ರಹ್ಮಕರ್ಮ ಸಮುಚ್ಚಯ

ಎಂಜಲಾಗಿದ್ದರೂ ತಿಂದೆ, ತಿನ್ನಬಾರದನೂ ಮೆದ್ದೆ,
ಅರಿವಿರದೆ ತಪ್ಪು ತಪ್ಪಾಗಿ ತುಂಬ, ಹುಂಬ, ನಾ ನಡೆದೆ;
ಕೆಟ್ಟದರ ಕೆಟ್ಟವರ ಒಡನಾಟ ಸಹಿಸಿ ಸುಮ್ಮನೆ ಇದ್ದೆ-
ನನ್ನ ಮನ್ನಿಸಿ, ನೀರೆ, ನೀ ಶುಚಿಗೊಳಿಸು ಮುದದೆ!

೧೭೨ :: ಬಿಟ್ಟು ಕಳಚಿದೆ, ಈಗ ಈಗಲೇ ಹೊರಟೆ

ಯೇ ಅಪ್ಸು ಅನ್ತರ್ ಅಗ್ನಯಃ, ಪ್ರವಿಷ್ಟಾ, ಗೋಹ್ಯ,
ಉಪಗೋಹ್ಯೋ, ಮಯೂಖೋ, ಮನೋಹಾಸ್,
ಖಲೋ, ವಿರಜಸ್, ತನೂದೂಷುರ್, ಇಂದ್ರಿಯಹಾ-
ತಾನ್ ವಿಜಹಾಮಿ। ಯೋ ರೋಚನಸ್ ತಂ ಇಹ ಗೃಹ್ಣಾಮಿ॥''
(ಇತಿ ಮನ್ತ್ರೇಣ ಆಪೋ ಗೃಹೀತ್ವಾ)
ಓಂ ತೇನ ಮಾಂ ಅಭಿಷಿಂಚಾಮಿ, ಶ್ರಿಯಂ ಯಶಸೇ,
ಬ್ರಹ್ಮಣೇ ಬ್ರಹ್ಮವರ್ಚಸಾಯ॥172

ಪಾರಸ್ಕರ ಗೃಹ್ಯಸೂತ್ರ ೨:೬:೮-೧೦

ಈ ನೀರ ಒಳಹೊಕ್ಕ, ಮುಚ್ಚಿ ಅಡಗಿದ ಬಡಬ,
ಪ್ರಖರ ತೀಕ್ಷ್ಣದ ಕಿರಣ, ಮನಮುರಿವ ಬರಸಿಡಿಲು,
ತಡೆಯಲಾರದ ನೋವು, ಒಡಲ ಹುಡಿಮಾಡುವದ
ಇಂದ್ರಿಯಗಳೆಲ್ಲವ ಸುಡುವ ಬಗೆ ಬಗೆಯ ಅಗ್ನಿಗಳ
ಬಿಟ್ಟು ಕಳಚಿದೆ, ನಾನಿದೋ ಈಗ ಈಗಲೇ ಹೊರಟೆ.
ಹೊಳೆ ಹೊಳೆವ ಬೆಳಕನ್ನ ಹಿಡಿದು ಮುಂದಕೆ ನಡೆವೆ.
ಸಿರಿ ಯಶಸ್ಸಿನ ಬಯಕೆ, ಬ್ರಹ್ಮವರ್ಚಸ್ಸಿನ ಹರಕೆ,
ಆ ಬ್ರಹ್ಮನನೆ ತಲುಪುವಾಕಾಂಕ್ಷೆಯಿಂ ನೀರ ಸಿಂಪಡಿಸಿಕೊಳುವೆ!

೧೭೩ :: ಉತ್ತರಿಸಿ ಪ್ರಶ್ನೆಗಳ

ಕೋ ಧರ್ಮಃ? ಭೂತದಯಾ,
ಕಿಂ ಸೌಖ್ಯಂ? ಅರೋಗಿತಾ।
ಜಗತಿ [ಜನ್ತೋಃ] ಕಃ ಸ್ನೇಹಃ? ಸದ್ಭಾವಃ,
ಕಿಂ ಪಾಂಡಿತ್ಯಂ? ಪರಿಚ್ಛೇದಃ॥173

ನಾರಾಯಣ ಪಂಡಿತ, ಹಿತೋಪದೇಶ ೨:೧೪೯

''ಧರ್ಮವಾವುದು? ಹೇಳಿ.''
''ದಯೆ ತೋರೆ ಎಲ್ಲ ಜೀವಿಗಳಲ್ಲಿ'';
''ಸೌಖ್ಯವೆಲ್ಲಿದೆ? ಹೇಳಿ.''
''ಪ್ರಾಣಿಗಳು ಅನುಭವಿಪ ಆರೋಗ್ಯದಲ್ಲಿ!''
''ಸ್ನೇಹವೆಂದರದೇನು?''
''ಸದ್ಭಾವ ನಮ್ಮೊಡನೆ ಕೂಡಿ ನಡೆವವರಲ್ಲಿ'';
''ಪಾಂಡಿತ್ಯ ಗುರುತಿಸುವುದದೆಂತು?''
''ಒಗಟೊಡೆದು ತಿಳಿವ ಬೆಳಗುವಲ್ಲಿ!''

೧೭೪ :: ಅವನಂತೆ ಅವರೂನು!

ರಾಜ್ಞಿ ಧರ್ಮಿಣಿ ಧರ್ಮಿಷ್ಠಾಃ, ಪಾಪೇ ಪಾಪಾಃ, ಸಮೇ ಸಮಾಃ।
ರಾಜಾನಂ ಅನುವರ್ತನ್ತೇ, ಯಥಾ ರಾಜಾ ತಥಾ ಪ್ರಜಾಃ॥174

-ಬಲ್ಲಾಳಸೇನ, ಭೋಜಪ್ರಬಂಧ ೪೪

ಅವ ನ್ಯಾಯಿ ಧರ್ಮಿಷ್ಠ, ಅವರೂ ಸರಿ-ನಡತೆಯವರೇ ಸೈ;
ಅವ ಪಾಪಿ ದುರ್ಬುದ್ಧಿ, ಪಾಪಮಾರ್ಗಿಗಳಂತೆಯೇ ಅವರೂ;
ಅತ್ತಲಿತ್ತಲೂ ಇರದ ನಡುವಿನವ, ಮಂದಿಗೂ ನಡು ಜಾಡೆ
ಒಡೆಯನನ್ನೆ ಅನುಸರಿಸುವರು ಜನ ಕಣ್ಣುಮುಚ್ಚಿ,
''ಹೇಗೆ ರಾಜನೋ ಹಾಗೆ, ಹಾಗೇ ಪ್ರಜೆ ಅವನ ಮೆಚ್ಚಿ!''

೧೭೫ :: ಯಾರು ಹೆಚ್ಚು?

ಮಾತಾ ಗುರುತರಾ ಭೂಮೇಃ, ಪಿತಾ ಉಚ್ಚತರಶ್ಚ ಖಾತ್।
ಮನಃ ಶೀಘ್ರತರಃ ವಾಯೋಃ, ಚಿಂತಾ ಬಹುತರೀ ನೃಣಾಂ॥175

-ವ್ಯಾಸ, ಮಹಾಭಾರತ, ವನಪರ್ವ ೩೧೩:೬೦

ತಾಯಿಗಿಂತ ದೊಡ್ಡದೆಲ್ಲಿ? ಭೂಮಿಗಿಂತ ತೂಕವೇ?
ತಂದೆಗಿಂತ ಮೇಲೆ ಮುಗಿಲೆ? ಆ ಎತ್ತರಕೇರದು;
ಗಾಳಿಗಿಂತ ವೇಗವಾಗಿ ಹರಿದಾಡುವ ಲಘು ಮನಸು-
ಚಿಂತೆಗಿಂತ ಮಿಗಿಲಾಗಿ ಎಲ್ಲಿಹುದು, ನಿಜ ಇದು!

೧೭೬ :: ಇದ್ದರೂ ಅಷ್ಟೇ, ಇಲ್ಲದಿದ್ದರೆ ಏನು?

ತುಷ್ಟೇ ಸತಿ ನ ಲಾಭಾಯ, ದುಷ್ಟೇ ನಾಶಾಯ ನೈವ ಚ
ಪ್ರಜ್ವಾಲಿತಾನಿ ಶಶ್ಪಾಣಿ ನ ಅಂಗಾರಾಯ ನ ಭಸ್ಮನೇ॥176

ಸಮಯೋಚಿತ ಪದ್ಯಮಾಲಿಕಾ

ಒಲಿದರೂ ಲಾಭವೇನು ಆಗದು, ಇಲ್ಲ;
ಇಲ್ಲ ಮುನಿದರೂ ಅಂತಹದೇನೂ ಅದರಿಂದ ನಷ್ಟ;
ಧಗಧಗಿಸಿ ಉರಿದು ಸುಟ್ಟರೂನು ಹುಲ್ಲು
ಆದೀತೇ ವಿಭೂತಿ? ಇದ್ದಿಲಂತೂ ಆಗದದು ಸ್ಪಷ್ಟ॥

೧೭೭ :: ಅದೂ ಅಲ್ಲ, ಇದೂ ಅಲ್ಲ

ನ ಸಂಧ್ಯಾಂ ಸಂಧತ್ತೇ, ನಿಯಮಿತ ನಮಾಜಂ ನ ಕುರುತೇ;
ನ ವಾ ಮೌಂಜೀಬಂಧಂ ಕಲಯತಿ, ವಾ ಸುನ್ನತವಿಧಿಂ।
ನ ರೋಜಾಂ ಜಾನೀತೇ, ವ್ರತಂ ಅಪಿ ಹರೇರ್ ನೈವ ಕುರುತೇ;
ನ ಕಾಶೀ ಮಕ್ಕಾ ವಾ ಶಿವ ಶಿವ, ನ ಹಿಂದುರ್ ನ ಯವನಃ ॥177

ಸಮಯೋಚಿತ ಪದ್ಯಮಾಲಿಕಾ

ಸಂಧ್ಯಾವಂದನೆ? ಇಲ್ಲ, ನಿಯಮಿತ ನಮಾಜೂನು ನೀ ಮಾಡಲ್ಲ;
ಮುಂಜಿ ಆಯಿತೋ? ಇಲ್ಲ, ಸುನ್ನತಿ ವಿಧಿ ನಡೆಯಿತೋ ಗೊತ್ತಿಲ್ಲ;
ಉಪವಾಸ ಮಾಡಿದೆ? ಇಲ್ಲ; ರೋಜಾ ಹಿಡಿದಿದ್ದೆಯಾ ನೆನೆಪಿಲ್ಲ;
ಹೋದೆಯಾ ಕಾಶಿಗೆ? ಇಲ್ಲ; ಮೆಕ್ಕಾ ಮದೀನಾಕ್ಕೆ ಇನ್ನೂ ಆಗಿಲ್ಲ;
-ನೀನೆಂಥ ಬ್ರಾಹ್ಮಣನೋ, ಪಕ್ಕಾ ಮುಸಲ್ಮಾನಂತೂ ಅಲ್ಲವಲ್ಲ!

೧೭೮ :: ಏನನಾದರೂ ಮಾಡು, ಹೆಸರಂತು ಗಳಿಸು!

ಘಟಂ ಭಿಂದ್ಯಾತ್, ಪಟಂ ಛಿಂದ್ಯಾತ್,
ಕುರ್ಯಾದ್ ವಾ ಗಾರ್ದಭಸ್ವನಮ್।
ಏನಕೇನ ಪ್ರಕಾರೇಣ ಪ್ರಸಿದ್ಧಪುರುಷೋ ಭವೇತ್॥178

ಸಮಯೋಚಿತ ಪದ್ಯಮಾಲಿಕಾ

ಒಡೆ ಸಿಕ್ಕ ಮಡಕೆಯನು, ಹರಿ ಬಟ್ಟೆ ಕಿತ್ತೆಸೆದು,
ಕೂಗು ಕತ್ತೆಯ ತೆರದಿ, ಅಥವಾ, ಪೇಟೆಯಲ್ಲಿ;
ಏನನಾದರೂ ಮಾಡು, ಮಗನೆ, ಎಲ್ಲರಂತಿರಬೇಡ,
ಜನರ ಕಣ್ಣಿಗೆ ಬೀಳು, ಹೆಸರಂತು ಗಳಿಸು!

೧೭೯ :: ಹಳೆಯದೆಂದು ಹಳಿಯಬೇಡಿ

ನವ ವಸ್ತ್ರಂ, ನವ ಛತ್ರಂ, ನವ್ಯಾ ಸ್ತ್ರೀ, ನೂತನಂ ಗೃಹಂ।
ಸರ್ವತ್ರ ನೂತನಂ ಶಸ್ತಂ, ಸೇವಕಾನ್ನಂ ಪುರಾತನೇ॥179

ನೀತಿ ಪ್ರದೀಪ ೧೪

ಉಡುಗೆ, ಕೊಡೆ ಹೊಸದಿರಬೇಕು, ಹೊಸ ಮನೆಯದೇ ಶೋಭೆ,
ಹೊಸ ಜೋಡಿ ಹಕ್ಕಿಗಳ ರಸಿಕತೆ ಎಲ್ಲರ ಕಣ್ಕಚ್ಚು ಮೆಚ್ಚು;
ಎಲ್ಲರ ಬಯಕೆ: ಇರಲಿ ಎಲ್ಲ ಹೊಸತೇ, ಠಾಕು ಠೀಕು; ಆದರೂ,
ಕೆಲಸಕ್ಕೆ ಅನುಭವಿಯಾಳು, ಅನ್ನಕ್ಕೆ ಹಳೆ ಅಕ್ಕಿ ಬೇಕೇ ಬೇಕು!

೧೮೦ :: ಸಸಾರ - ತಾತ್ಸಾರ

ಅತಿ ಪರಿಚಯಾದ್ ಅವಜ್ಞಾ,
ಸಂತತ ಗಮನಾತ್ ಅನಾದರೋ ಭವತಿ।
ಮಲಯೇ ಭಿಲ್ಲಪುರಂಧ್ರೀ ಚಂದನತರುಕಾಷ್ಠಂ
ಇಂಧನಂ ಕುರುತೇ॥180

ಶಾಙ್ರ್ಗಧರಪದ್ಧತಿ ೮೩

ಮತ್ತೆ ಮತ್ತೆ ಬರುವರೆ? ಇಕ್ಕು ನೂಕು ತಕ್ಕೊ-ಮಣೆಯುಪಚಾರ;
ಹೆಚ್ಚು ಪರಿಚಿತರೇನು? ಉರಳಿ ದಿನ, ಆದರದ ಜಾರು, ತಾತ್ಸಾರ;
ಸುತ್ತ ಬೆಳೆದಿರೆ ಗಂಧದ ಮರ, ಅತ್ತ ಕಾಡ ಮನೆಯೊಡತಿಗೆ ಸಸಾರ
-ಇಕ್ಕುವಳು ಒಲೆಯುರಿಗೆ ಸೌದೆಯೊಲು ಚಂದನದ ತುಂಡನ್ನೆ ನೇರ.

೧೮೧ :: ದೂರವಿದ್ದರೇ ಚೆಂದ

ದೂರಸ್ಥಾ ಪರ್ವತಾ ರಮ್ಯಾ,
ವೇಶ್ಯಾ ಚ ಮುಖಮಂಡನೇ।
ಯುದ್ಧಸ್ಯ ವಾರ್ತಾ ರಮ್ಯಾ ಚ,
ತ್ರೀಣಿ ರಮ್ಯಾಣಿ ದೂರತಃ॥181

ಸಮಯೋಚಿತ ಪದ್ಯಮಾಲಿಕಾ

ದೂರದಲ್ಲಿಹ ಬೆಟ್ಟ, ಏನು ಎತ್ತರ, ನುಣ್ಣಗೆ, ಆಹಾ! ಎಷ್ಟು ಭವ್ಯ;
ಅತ್ತ ಬದಿ ಹೊಲ ಹಸಿರೋ ಹಸಿರು, ಕಣ್ತಣಿಪ ದಿವ್ಯ;
ತನ್ನವಳಲ್ಲದ ಬೆಡಗಿಯದೇನು ಸಿಂಗಾರ, ವೈಯಾರ, ಲಾವಣ್ಯ-
ಸುದ್ದಿ ಯುದ್ಧದೂ ಕೂಡ ಹಾಗೆ ದೂರದ್ದೇ ರಮ್ಯ!

೧೮೨ :: ಯಾವ ಯಾವುದಕ್ಕೆ ಎಷ್ಟು ಎಷ್ಟು ಬೇಕು?

ಏಕಸ್ ತಪೋ ದ್ವಿರ್ ಅಧ್ಯಾಯೀ,
ತ್ರಿಭಿರ್ ಗೀತಂ, ಚತುಷ್ ಪಥಮ್।
ಸಪ್ತ ಪಂಚ ಕೃಷೀಣಾಂ ಚ,
ಸಂಗ್ರಾಮೇ ಬಹುಭಿರ್ ಜನೈಃ॥182

ಸುಭಾಷಿತ ರತ್ನಭಾಂಡಾಗಾರ ೧೫೮:೨೧೧

ಧ್ಯಾನ, ಭಕ್ತಿ, ತಪಸ್ಸು ಒಬ್ಬರೇ ಒಬ್ಬರ ಸಾಧನೆಯಷ್ಟೆ;
ಇಬ್ಬರಿದ್ದರೆ ಒಳಿತು ಓದು ಬರಹ ಕಲಿಕೆ ಅಧ್ಯಯನಕೆ;
ಹಾಡಿಗೋ ಮೂವರ ನೆರವು, ದಾರಿಗೆ ನಾಲ್ವರು ಸಾಕು;
ಕೃಷಿಗೈದಾರು ಜನ, ಯುದ್ಧಕೆಷ್ಟಿದ್ದರೂ ಅಷ್ಟೂ ಬೇಕು!

೧೮೩ :: ಯಾರು ಮಿಗಿಲಾಗಿಹರು ಉದರವೋ, ಹೃದಯವೋ?

ತ್ವಾಂ, ಉದರ, ಸಾಧು ಮನ್ಯೇ,
ಶಾಕೈರ್ ಅಪಿ ಯದ ಅಸಿ ಲಬ್ಧ-ಪರಿತೋಷಮ್।
ಹತ ಹೃದಯಮ್! ಹಿ,
ಅಧಿಕಾಧಿಕ-ವಾಂಛಾಶತ-ದುರ್ಭರಂ ನ ಪುನಃ॥183

ಶಿಲ್ಹಣ ಶಾಂತಿಶತಕ ೧:೨೪

ಓ ಹೊಟ್ಟೆ, ನೀನೇ ಮೇಲು; ನಿನಗೆ ಬೇರೇನೂ ಬೇಕಿಲ್ಲ;
ಏನಿಲ್ಲದಿರೆ ಅಷ್ಟಿಷ್ಟು ಶಾಕಾಹಾರದಿಂದಲೇ ನೀನಲ್ಪ ತೃಪ್ತ;
ನೋಡು ಹೃದಯವ ಸ್ವಲ್ಪ? ಅದು ನಿನ್ನಂತೇನೂ ಅಲ್ಲ-
ನೂರಾರು ಭಾರಿ ಅಭಿಲಾಷೆ ಆಕಾಂಕ್ಷೆಯದಕೆ, ತಣಿಸಲಳವಲ್ಲ!

೧೮೪ :: ಎಮ್ಮೆ, ನಿನಗೆ ಸಾಟಿಯಿಲ್ಲ

ತ್ವಂ ವಾಚಂಯಮತಾಂ ದಧಾಸಿ ಸಹಸೇ ಶೀತಾತಪೌ ನಿಶ್ಚಲಂ
ಕಿಂಚಿನ್ನೈವ ಚ ಯಾಚಸೇ ಭುವಿ ಯಥಾಜಾತಾತ್ಮನಾ ತಿಷ್ಠಸಿ।
ಪ್ರಾಣಾಕರ್ಷಕೃತಾ ಯಮೇನ ನಿಯಮೇನಾರೂಢ ಏವಾಸಿ ಚ
ಪ್ರಾಯಃ ಕಾಸರಸಾರ್ವಭೌಮ ನಿತರಾಂ
ಯೋಗೀವ ಸಂದೃಶ್ಯಸೇ॥184

ವಾಂಛೇಶ್ವರ (ಕುಟ್ಟ ಕವಿ), ಮಹಿಷ ಶತಕ

ನೀನೋ ಮಾತನಾಡದ ಮೌನಿ, ಸಹಿಸಿರುವೆ
ಗಾಳಿ ಮಳೆ ಚಳಿ ಬಿಸಿಲನೊಂದೊಂದನೂ;
ಬೇಡ ಏನೂ ಬೇಡವೆನುತಲೇ ಬಟ್ಟೆಗೆಟ್ಟವನಾಗಿ
ಹುಟ್ಟಿದೋಪಾದಿಯಲೆ ಇರುವೆ ನೀನು;
ಉಸಿರೂಡಿ ಒಳಸೆಳೆವ ನಿಯಮಗಳ ಯಮನೊಡನೆ
ನಿರತನಿಹೆ ನೀ ಹಗಲಿರುಳು ಯಾವಾಗಲೂ-
ನೀನಲ್ಲದಿನ್ನಾರು ಹೇಳು, ಓ ಕೆಸರೊಡೆಯ, ಮಹಿಷನೆ,
ಹೀಗಿರಲು ನಿಜವಾಗಿ ಬೇರೆ ಯೋಗಿ?

೧೮೫ :: ಕೆಳಗಿಹೆನೆಂಬ ಕೀಳರಿಮೆ ಬೇಡ!

ನಿತರಾಂ ನೀಚೋಽಸ್ಮಿ- ಇತಿ
ತ್ವಂ ಖೇದಂ, ಕೂಪ, ಮಾ ಕದಾಪಿ ಕೃಥಾಃ।
ಅತ್ಯಂತ ಸ-ರಸ-ಹೃದಯೋ ಯತಃ
ಪರೇಷಾಂ ಗುಣಗ್ರಹಿತಾ ಅಸಿ॥185

ಪಂಡಿತ ಜಗನ್ನಾಥ, ಭಾಮಿನಿ ವಿಲಾಸ ೭

ತೀರ ಕೆಳಗಿಹೆನೆಂದು ಅಳುಕಿ ಕೊರಗದಿರು, ಓ ಬಾವಿ,
ಇರಬಹುದು ನೆಲದಿಂದ ನೀನು ಆಳದಲಿ, ತಳದಲ್ಲಿ;
ಆದ್ರ್ರಹೃದಯಿಯೋ ನೀನು, ಗುಣಗ್ರಾಹಿ ಸದಯ,
ಹಗ್ಗವೆಸೆದವರ ಕೊಡ ಕೊಡ ತುಂಬಿ ನೀಡುವೆ ನೀರನಲ್ಲಿ!
