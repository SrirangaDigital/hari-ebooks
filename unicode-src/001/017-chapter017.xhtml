೧೭ಭಕ್ತಿ-ಸಾಯುಜ್ಯ

೧೫೫ :: ನೆನವುದಾರನ ಹೇಳಿ, ಅವನನಿಲ್ಲದಾರನ್ನ?

ಅಂತೇ ಸಂತೋಷದಃ ವಿಷ್ಣುಂ ಸ್ಮರೇತ್ ಹಂತಾರಂ ಆಪದಾಂ।
ಶರತಲ್ಪಗತೇ ಭೀಷ್ಮಃ ಸಸ್ಮಾರ ಗರುಡಧ್ವಜಂ॥155

-ಕ್ಷೇಮೇಂದ್ರ,ಚಾರುಚರ್ಯಾ

ಜಾತ್ರೆ ಮುಗಿಯಿತೆಂದಾಗ, ಸಂತೆ ತಾ ಕೊನೆಗೊಂಡಾಗ,
ಸದ್ದುಗದ್ದಲವಡಗಿ, ಬೆಂಕಿ ನುಣ್ಣಗೆ ಆರಿ ತಣ್ಣಗಾದಾಗ-
ನೆನವುದಾರನ ಹೇಳಿ,ಅವನ, ಆ ಅವನನಲ್ಲದಿನ್ನಾರನ್ನ?
ಆಪತ್ತುಗಳ ಕಿತ್ತೊಗೆದು ನೆಮ್ಮದಿಯ ತಂಪ ಸೂಸಿದವನನ್ನ-
ಶರತಲ್ಪದಲ್ಲಿ ಮಲಗಿ ಕುರುಕುಲದಜ್ಜ ನೆನೆದ ಭಗವಂತನನ್ನ!

೧೫೬ :: ಏನಿದ್ದರೂ ಇಲ್ಲದಿದ್ದರೂ ಒಲಿದವರಿಗೊಂದೇ

ತ್ವಯಿ ಪ್ರಸನ್ನೇ ಕಿಂ ಗುಣೇನ,
ತ್ವಯಿ ಅಪ್ರಸನ್ನೇ ಕಿಂ ಗುಣೇನ?
ರಕ್ತೇ ವಿರಕ್ತೇ ಚ ವರೇ ವಧೂನಾಂ,
ನಿರರ್ಥಕಃ ಕುಂಕುಮಪತ್ರಭಂಗಃ॥156

ಲೀಲಾಶುಕ, ಶ್ರೀಕೃಷ್ಣಕರ್ಣಾಮೃತ, ೨:೯೯

ನೀನೊಲಿದು ಇರುವಾಗ ಗುಣ ವಿಧಿ ವಿಧಾನದ ಹಂಗೆ?
ಒಲಿಯದಿರೆ ಎಷ್ಟಿದ್ದರೇನೇನು ಅವು ಎಲ್ಲ ವ್ಯರ್ಥ;
ಪ್ರೀತಿಯಿರೆ ಏನೂ ಬೇಡ; ಇಲ್ಲದಿರೆ ಏನಿದ್ದರೇನು ಬಿಡು-
ಪ್ರೇಮಿ-ಪ್ರೇಯಸಿಯರಲಿ ಶೃಂಗಾರ ಸಾಧನಕಿಲ್ಲ ಅರ್ಥ!

೧೫೭ :: ಆಗಲೂ ನಿನ್ನವನೇನೆ ನಾನು; ನೀನು ನನ್ನವನಾಗೆ!

ಸತಿ ಅಪಿ ಭೇದ ಅಪಗಮೇ, ನಾಥ,
ತವ ಅಹಂ ನ, ಮಾಮಕೀ ನಸ್ ತ್ವಮ್।
ಸಾಮುದ್ರೋ ಹಿ ತರಂಗಃ,
ಕ್ವಚನ ಸಮುದ್ರೊ ನ ತಾರಂಗಃ॥157

ಆಚಾರ್ಯ ಶಂಕರ, ವಿಷ್ಣು ಷಟ್ಪದಿ ೩

ನಿನ್ನ ಅನುಗ್ರಹದಿಂದ ಒಂದು ಚಂದದ ವೇಳೆ,
ಅಂದುಕೋ, ನಾನು ನಿನ್ನಲ್ಲೇನೆ ಸೇರಿಹೋದೆ;
ಆಗ ನನಗೂ ನಿನಗೂ ಏನು ಭೇದವೇ ಇರದ,
ಅಂದುಕೋ ಹಾಗೆಯೇ, ಸ್ಥಿತಿಯೊಂದು ಬಂದಿತೆಂದು;
ಆಗಲೂ ನಿನ್ನವನೇನೆ ನಾನು; ನೀನು ನನ್ನವನಾಗೆ!
ಅದು ಸಾಧ್ಯವೇ ಇಲ್ಲ, ಸಾಧ್ಯವಹುದೇ ಹೇಗೆ?-
ಸಮುದ್ರದಿಂದಲೇ ತರಂಗಗಳೆಲ್ಲ, ಹೌದು ನಿಜ;
ಅಲೆಗಳಿಂದಲೇ ಸಮುದ್ರವಾಯಿತೆನ್ನಬಹುದೇ?

೧೫೮ :: ಬಾ, ನೆನೆವೆ, ಮೈದೋರು!

ಭಕ್ತ್ಯಾ ತ್ವನ್ ಅನ್ಯಯಾ ಶಕ್ಯೋ ಹಿ
ಅಹಂ ಏವಂ ವಿಧೋ ಅರ್ಜುನ।
ಜ್ಞಾತುಂ ದ್ರಷ್ಟುಂ ಚ ತತ್ತ್ವೇನ ಪ್ರವೇಷ್ಟುಂ ಚ ಪರಂತಪ॥158

ಭಗವದ್ಗೀತೆ ೧೧:೫೪

ಏನು ಕೇಳಿದರೇನು? ಬರೆದೋದಿ ಹಾಡಿದರೇನು?
ಏನು ಧ್ಯಾನಿಸಿ ನೆನೆ ನೆನೆದು ಕುಳಿತರೇನು?
ನೀನಲ್ಲದೆ ಬೇರಿಲ್ಲ ತಿಳಿದ ಭಕ್ತಿಯೊಂದೇ ತಾನು
ಬಳಿಗೊಯ್ದು, ಇರಿಸಿ, ಕಾಣಿಸಿ ಅವನಲೇ ಸೇರಿಸದದೇನು?

೧೫೯ :: 'ನಾನಾದೆ ನೀ. ..'-ನೆನಲು ಕರುಣಿಸುವೆ ಸಾಯುಜ್ಯ!

ಭವಾನಿ, ತ್ವಂ ದಾಸೇ ಮಯಿ, ವಿತರ ದೃಷ್ಟಿಂ ಸಕರುಣಾಂ-
ಇತಿ ಸ್ತೋತುಂ ವಾಂಛನ್ ಕಥಯತಿ ಭವಾನಿ ತ್ವಮ್ -ಇತಿ ಯಃ ।
ತದೈವ ತ್ವಂ ತಸ್ಮೈ ದಿಶಸಿ ನಿಜ ಸಾಯುಜ್ಯಪದವೀಂ,
ಮುಕುನ್ದ-ಬ್ರಹ್ಮೇನ್ದ್ರ ಸ್ಫುಟ ಮುಕುಟ ನೀರಾಜಿತ ಪದಾಮ್॥159

ಆಚಾರ್ಯ ಶ್ರೀ ಶಂಕರ, ಸೌಂದರ್ಯ ಲಹರಿ ೨೨

'ನಾನಾದೆ ನೀ ಕಣ್ಣಾರೆ ದಯೆ ತೋರೆ, ನಿನ್ನಯ ದಾಸ, ಭವಾನಿ'-
ಎಂದು ಸ್ತುತಿಸಲು ತೊಡಗಿ, 'ನಾನಾದೆ ನೀ.. ..'- ನೆಂದು ನುಡಿಯೆ,
ಒಡನೆ ಕರುಣಿಸಿ ಬಿಡುವೆ ಎನಗೆ ಸಾಯುಜ್ಯ ಪದವಿಯ, ಕಾಣುವೆನು
ದೇವಗಣ ತಲೆಬಾಗೆ ಮುಕುಟದಾರತಿಗೊಳುವ ನಿನ್ನಡಿಯ, ತಾಯೆ!

೧೬೦ :: ಇಲ್ಲೇ ಅವನಿರುವಾಗ ಎಲ್ಲೆಲ್ಲೋ ಹುಡುಕುವುದೇಕೆ?

ಲಲ್ಲಾಹಂ ನಿರ್ಗತಾ ದೂರಂ ಅನ್ವೇಷ್ಟುಂ ಶಂಕರಂ ವಿಭುಂ
ಭ್ರಾಂತ್ವಾ ಲಬ್ಧೋ ಮಯಾ ಸ್ವಸ್ಮಿನ್
ದೇಹೇ ದೇವೋ ಗೃಹೇಸ್ಥಿತಃ॥160

ಲಲ್ಲೇಶ್ವರಿ, ಲಲ್ಲಾವಾಕ್ ೩, ಭಾಸ್ಕರನ ಸಂಸ್ಕೃತ ಛಾಯೆ

ಹಗಲಿರುಳು ಹುಡುಕುತ್ತಾ ಎತ್ತಲೆತ್ತಲೋ ಸುತ್ತಿ
ಅವನ ನಾ ನೆನೆದೇನೆ ಬಳಲಿ ಬಸವಳಿದೆ;
ಕೊನೆಗೊಮ್ಮೆ ಲಲ್ಲಳಿಗೆ ಇಲ್ಲೇ ಮನೆಯಲೇ ಸಿಕ್ಕ
ಎಲ್ಲ ಬಲ್ಲವ ಶುಭ ಸಮಯ ಒದಗಿ ಬರಲು!
