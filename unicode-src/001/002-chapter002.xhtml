೨ಜಾಣ್ಮೆ -ವಿದ್ಯೆ

೪ :: ಏನು ವರ ಬೇಕು?

ಅಂಧಂ ದರಿದ್ರಂ ಅಪಿ ಪ್ರಿಯಯಾ ವಿಹೀನಂ,
ವೀಕ್ಷ್ಯ ಈಶ್ವರೋ ವದತಿ- ಯಾಚ ವರಂ ತ್ವಂ ಏಕಮ್।
ನೇತ್ರೇ ನ, ನ ಅಪಿ ವಸು, ನೋ ವನಿತಾಂ, ಸ ವವ್ರೇ;
ಛತ್ರ-ಅಭಿರಾಮ-ಸುತ ದರ್ಶನಂ- ಇತಿ ಉವಾಚ॥004

ಸುಭಾಷಿತ ರತ್ನಾಕರ

ಮನೆ, ಮಗನ ಹೆಂಡತಿಯ ಕಳಕೊಂಡ ಬಡ ಕುರುಡ
ಕಡು ತಪವ ಮಾಡಿದನು; ಶಿವ ಅವನಿಗೊಲಿದ.
ಒಂದು ವರವನು ಕೇಳು; ಬೇಡಿದ್ದ ಕೊಡುವೆ-ಎನೆ,
ಸಂಪತ್ತ ಕೇಳದೆಯೇ, ಹೆಂಡತಿಯ ಬೇಕೆನ್ನದೆಯೇ,
ಕಣ್ಣನ್ನೂ ಬೇಡದೇ ಹೋದ ಆ ಜಾಣ ಹೈದ-
ಮಗ ರಾಜನಾಗಿ ಮೆರೆವುದ ನಾ ಕಣ್ತುಂಬ ನೋಡಬೇಕೆಂದ!

೫ :: ಒಂದೇ ಕಲ್ಲಿಗೆ ಮೂರು ಹಕ್ಕಿ

ತಪಃ ಕುರ್ವನ್ತೀಂ ಭಗವಾನ್ ಪಪ್ರಚ್ಛ,
ತುಷ್ಟೋಽಸ್ಮಿ ಬಾಲೇ, ವರಂ ಏಕಂ ಯಾಚ|
ಪುತ್ರೈಶ್ ಚ ಪೌತ್ರೈಶ್ ಚ ಭರ್ತೃಸಮೇತಂ,
ಇಚ್ಛಾಮಿ ಭೋಜನಂ ಹಿರಣ್ಯಪಾತ್ರೇ||005

ಸುಭಾಷಿತ

ಮದುವೆಯಾಗದೆ ಉಳಿದು, ಬೆಳೆದ ಹುಡುಗಿಯು ಕುಳಿತು,
ತಪಗೈದು ಕಣ್ದೆರೆಯೆ, ದೇವರೇ ಎದುರು ಬರೆ,
ವರವನೊಂದನು ಬೇಡು, ಕೊಡುವೆ"-ಎಂದನಂತೆ;
ಜಾಣೆ, ಏನ ಕೇಳಿದಳಾಗ, ಊಹಿಸಿರಿ ಅವಳ ಮೊರೆ-
ನನ್ನ ಮಕ್ಕಳ ಜೊತೆಗೆ, ಅನುದಿನವೂ ಇನಿಯನ ಕೂಡೆ,
ಅನುಗ್ರಹಿಸು, ಬಂಗಾರ ಹರಿವಾಣದಿ ಅನ್ನ ತಿನ್ನುವಂತೆ!

೬ :: ಇರುವುದಾದರೂ ಏಕೆ?

ವಿದ್ಯಾ ವಿವಾದಾಯ, ಧನಂ ಮದಾಯ,
ಶಕ್ತಿಃ ಪರೇಷಾಂ ಪರಿಪೀಡನಾಯ|
ಖಲಸ್ಯ; ಸಾಧೋಃ ವಿಪರೀತಂ ಏತತ್-
ಜ್ಞಾನಾಯ, ದಾನಾಯ ಚ ರಕ್ಷಣಾಯ||006

ಭವಭೂತಿ, ಗುಣರತ್ನ 7

ಓದು ಒಣ ಜಗಳಕ್ಕೆ, ಹಣ ಸೊಕ್ಕಿ ಮೆರೆಯಲಿಕೆ,
ಶಕ್ತಿಯಿದೆ ಅಸಹಾಯಕರ ಪೀಡಿಸಲು- ಖಳಗೆ;
ಸಜ್ಜನರಿಗೋ ಉಪಯೋಗ ಇವುಗಳದೇ ಬೇರೆ-
ಜ್ಞಾನಕ್ಕೆ, ದಾನಕ್ಕೆ ಹಾಗೂ ರಕ್ಷಣೆಯ ಹೊಣೆಗೆ!

೭ :: ಬೆಳಕನ್ನೇ ಕತ್ತಲಾಗಿಸಿದ ಭೂಪ

ವಿದ್ಯಾ ವಿವಾದಾಯ, ಧನಂ ಮದಾಯ,
ಪ್ರಜ್ಞಾ ಪ್ರಕರ್ಷಃ ಪರವಂಚನಾಯ।
ಅತ್ಯುನ್ನತಿರ್ ಲೋಕ ಪರಾಭವಾಯ,
ಏಷಾಂ ಪ್ರಕಾಶಸ್ ತಿಮಿರಂ ಹಿ ತೇಷಾಮ್॥007

ಕ್ಷೇಮೇಂದ್ರ ದರ್ಪದಲನಂ ೨:೯

ಕಲಿತ ವಿದ್ಯೆಯ ಬಳಕೆ ಬರಿ ಒಣ ಕಲಹಕೆ ಸೀಮಿತವಾಯ್ತೆ ನಿಮಗೆ?
ಇದ್ದ ಐಸಿರಿ ಸಂಪತ್ತೆಲ್ಲ ಮತ್ತೇರಿಸಲೊತ್ತೆ ಇಟ್ಟಾಯ್ತೆ ಈಪಾಟಿ?
ಅತಿಜಾಣ್ಮೆ ಕೀಳು ಸಾಧನವಾಯ್ತೆ ಮಂದಮತಿಗಳ ಮೋಸಗೊಳಿಸೆ?
ಮೇಲ್ಮೆ ಗಳಿಸಿದಿರಿ; ಲೋಕವನೇ ಮಣಿಸಿ ಸೋಲಿಸಿದಿರಿ, ಸ್ವಾಮಿ-
ಬೆಳಕನ್ನೇ ಕತ್ತಲಾಗಿಸಿದ ಭೂಪರು, ಬಿಡಿ ಬಿಡಿ, ನಿಮಗಿಲ್ಲ ಸಾಟಿ!

೮ :: ಇಂಥವರ ಮುಂದೆಯೇ ತಲೆ ಚಚ್ಚಿಕೊಳಲೇಕೆ?

ಯಸ್ಯನ್ ನಾಸ್ತಿ ಸ್ವಯಂ ಪ್ರಜ್ಞಾ ಶಾಸ್ತ್ರಂ ತಸ್ಯ ಕರೋತಿ ಕಿಮ್।
ಲೋಚನಾಭ್ಯಾಂ ವಿಹೀನಸ್ಯ ದರ್ಪಣಃ ಕಿಂ ಕರಿಷ್ಯತಿ॥008

ಹೀತೋಪದೇಶ, ವಿಗ್ರಹ ಅಧ್ಯಾಯ ೧೦೫; ೩:೧೧೯

ತಾನೇ ವಿವೇಚಿಸಲಾರ, ತಿಳಿಹೇಳಿದರೆ? ಕಿಮ್ಮತ್ತು ಎಳ್ಳಷ್ಟಿಲ್ಲ;
ಆ ಶಾಸ್ತ್ರ, ಈ ಶಾಸ್ತ್ರ ಓದಿ ಬಂದ ಫಲವೇನು?
ನೆರವಿಗೇನೂ ಬರದು ಆ ಈ ಪುಸ್ತಕದ ಸೂತ್ರ ಪರಿಹಾರ-
ಕನ್ನಡಿಯಿಂದೇನುಪಯೋಗ ಕಣ್ಣಿಲ್ಲದವಗೆ, ಏನು?

೯ :: ಜಗವೆಲ್ಲ ಮಲಗಿರಲು ನೀನೆದ್ದು ಜಾಗು!

ಉತ್ತಿಷ್ಠತ, ಜಾಗ್ರತ, ಪ್ರಾಪ್ಯ ವರಾನ್ ನಿಬೋಧತ;
ಕ್ಷುರಸ್ಯ ಧಾರಾ ನಿಶಿತಾ ದುರತ್ಯಯಾ,
ದುರ್ಗಂ ಪಥಸ್ ತತ್ ಕವಯೋ ವದಂತಿ॥009

ಕಠೋಪನಿಷತ್ 3:14

ಏಳು, ಜಾಗ್ರತನಾಗು, ಎಚ್ಚರದಿಂದಿರುತಲೇ ಸಾಗು,
ಬಲ್ಲವರ ಬಳಿ ಸಾರಿ, ಮೇರು ತಿಳಿವಳಿಕೆ ಗಳಿಸು;
ಕತ್ತಿಯಲುಗಿನ ಮೇಲೆ ನಡೆವ ದಾರಿಯೋ ಕಡಿದು-
ಅಳಿವು ಉಳಿವಿನ ಪ್ರಶ್ನೆ ಎಂದೆದುರಿಸುತ ಸೆಣೆಸು!

೧೦ :: ನಾಳೆ ಬಪ್ಪುದು ನಮಗಿಂದೇ ಬರಲಿ!

ಪ್ರಾಪ್ಸ್ಯಾಮಿ ಯಾನ್ ಅದ್ಯ ಗುಣಾನ್
ಕೋ ಮೇ ಶ್ವಸ್ತಾನ್ ಪ್ರದಾಸ್ಯತಿ।
ಅಪಕ್ರಮಣಂ ಏವ ಅತಃ ಸರ್ವಕಾಮೈರ್ ಅಹಂ ವೃಣೇ ॥010

ವಾಲ್ಮೀಕಿ ರಾಮಾಯಣ, ಅಯೋಧ್ಯಾಕಾಂಡ ೩೪:೪೦

ಇಂದು ನನಗನಿಸುದುದೇನೆ ನಾಳೆಯೂ ಅನಿಸುವ ಹಾಗೆ
ಮನದ ನೆಲೆ ಒದಗಿಸಿಕೊಡುವವರಾರು? ನಾನರಿಯೆನೆ;
ನನ್ನ ಆಶಯವೆಲ್ಲ ಇಂದೇ ಈಗಲೇ ಹಣ್ಣಾಯ್ತೆಂದು-
ಮುನ್ನಡೆದು ಸಾಗುವುದೊಳಿತು; ಆರಯಿಸಿಹೆನು ಅದನೆ!

೧೧ :: ಇಂದು-ನಾಳೆ

ಭವಿಷ್ಯಂ ನ ಅನುಸಂಧತ್ತೇ, ನ ಅತೀತಂ ಚಿಂತಯತಿ-ಅಸೌ
ವರ್ತಮಾನ ನಿಮೇಶಂ ತು ಹಸನ್ನ್ ಏವ ಅನುವರ್ತತೇ॥011

ಯೋಗವಾಸಿಷ್ಠ

ನಾಳೆ ಏನಾದೀತೆಂದು ತಲೆ ಕೆಡಿಸಿಕೊಳಲೇಕೆ, ಬಿಡು
ನೆನ್ನೆ ಕಳೆದುದ ಚಿಂತಿಸುವುದೆ ಸುಮ್ಮನೇ ಹಾಗೆ?
ಇಂದು ಬಂದೊದಗಿದುದ ನಗುನಗುತ ಸ್ವೀಕರಿಸು-
ನೆನ್ನೆಗೇ ನೆನ್ನೆ, ನಾಳೆಗೆ ನಾಳೆ, ಇರಲಿ ಇಂದು ನಮಗೆ.

೧೨ :: ಆಪತ್ತಿಗೊದಗುವ ವಿದ್ಯೆ ಕಲಿತಿಲ್ಲವೇನಯ್ಯ?

ಆಪತ್ಕಾಲ-ಉಪಯುಕ್ತಾಸು ಕಲಾಸು ಸ್ಯಾತ್ ಕೃತಶ್ರಮಃ।
ನೃತ್ತ ವೃತ್ತಿರ್ ವಿರಾಟಸ್ಯ ಕಿರೀಟೀ ಭವನೇಽಭವತ್॥012

ಕ್ಷೇಮೇಂದ್ರ, ಚಾರುಚರ್ಯಾ ೭೨

ದಿನವೂ ದೋಣಿಯನೇರಿ ಹೊಳೆ ದಾಟುವಾತನಿಗೆ
ಈಜು ಬಾರದೆ ಇರಲು ಅವ ಶುದ್ಧ ಪೆದ್ದ;
ಬೇರೇನೂ ಬೇಡವೇ ಬೇಡ, ಚಿಂತೆ ಏನೂ ಇಲ್ಲ-
ಆಪತ್ತಿಗೊದಗುವ ವಿದ್ಯೆ ಗೊತ್ತಿಲ್ಲವೇ, ಹೆಡ್ಡ?

೧೩ :: ಬುದ್ಧಿಗೆ ಮದ್ದು

ಚಿತ್ತಂ ಕಾರಣಂ ಅರ್ಥಾನಾಂ, ತಸ್ಮಿನ್ ಸತಿ ಜಗತ್ ತ್ರಯಮ್।
ತಸ್ಮಿನ್ ಕ್ಷೀಣೇ ಜಗತ್ ಕ್ಷೀಣಂ, ತತ್ ಚಿಕಿತ್ಸ್ಯಂ ಪ್ರಯತ್ನತಃ ॥013

ಯೋಗವಾಸಿಷ್ಠ

ಚಿತ್ತವೇ ಮೂಲ, ನೋಡು, ಎಲ್ಲಕ್ಕೂನು,
ಮೂಲೋಕ ಹುದುಗಿದೆ ಅಡಗಿ ಅಲ್ಲೇ;
ಅದರೇಳುಬೀಳುಗಳೇ ಇದರ ಉಬ್ಬರ ಇಳಿತ,
ಬೇಕು ಬುದ್ಧಿಗೆ ಮದ್ದು, ಮುದ್ದು, ಗುದ್ದು!

೧೪ :: ಕುರುಡರು ಯಾರು?

ಅನ್ಧದ್ವಯೇ ಮಹಾನ್ ಅನ್ಧೋ, ವಿಷಯ-ಅನ್ಧೀಕೃತ-ಈಕ್ಷಣಃ।
ಚಕ್ಷುಷಾ ಅನ್ಧೋ ನ ಜಾನಾತಿ, ವಿಷಯಾನ್ದೋ ನ ಕೇನಚಿತ್॥014

>

-ಮಹಾಸುಭಾಷಿತ ಸಂಗ್ರಹ, ಸಂಪುಟ ೧

ಎರಡು ಬಗೆ ಕುರುಡರಲಿ ಮೇಲಾರು? ಕೀಳಾರು?
ಕಣ್ಣಿರದ ಅಂಧನೋ? ತಿಳಿವಿಗೇ ಕಂಗಾಳನೋ?
ಕಣ್ಣಿಲ್ಲದ ಕಪೋತಿ, ಏನೂ ನೋಡಲಾರನು ಅಷ್ಟೆ;
ವಿಷಯಾಂಧನೋ- ಕಣ್ಣಿದ್ದೂ ಕಂಡದ್ದ ಕಾಣನಲ್ಲ!

೧೫ :: ಕೊಂಬು ಬಾಲವಿಲ್ಲದ ಪಶುವೇ

ಸಾಹಿತ್ಯ ಸಂಗೀತ ಕಲಾವಿಹೀನಃ
ಸಾಕ್ಷಾತ್ ಪಶುಃ ಪುಚ್ಛ-ವಿಷಾಣ-ಹೀನಃ
ತೃಣಂ ನ ಖಾದನ್ ಅಪಿ ಜೀವಮಾನಸ್
ತದ್ ಭಾಗದೇಯಂ ಪರಮಂ ಪಶೂನಾಂ॥015

ಭರ್ತೃಹರಿ, ನೀತಿಶತಕ ೧೦

ಓದುಬರಹ ಹಾಡುಪಾಡುಗಳೊಂದೂ ಹಿಡಿಸದೆನ್ನುವ ಜನರು
ನಿಜವಾಗಿ ಪ್ರಾಣಿಗಳೇ; ಕೊಂಬು ಬಾಲ ಮಾತ್ರ ಅವರಿಗಿಲ್ಲ;
ಹುಲ್ಲನೇನೂ ಅವರು ತಿನ್ನರಲ್ಲಾ- ಎನ್ನುವಿರಾ, ಹೌದು, ನಿಜ,
ಅದೊಂದೆ, ನೋಡಿ, ಹಸುಕರುಗಳ ಪುಣ್ಯ, ಅಲ್ಲವೇನು?

೧೬ :: ನಂದಿತೇ ತಿಳಿವಿನ ಸೊಡರು?

ಪರ್ಯಾಪತತ್ ಕಾಲ-ಕರಸ್ಯ-ಭೋಗಿ
ಸಂದರ್ಶನೇನ ಇವ ಮತಿಪ್ರದೀಪಃ
ಕ್ಷಿಪ್ರಂ ಪ್ರಶಾಂತಿ ಉನ್ಮುಖತಾಂ ಉಪೈತಿ
ವಿನಾಶಕಾಲೇಷು ಶರೀರಭಾಜಾಮ್॥016

ಕಲ್ಹಣ, ರಾಜತರಂಗಿಣೀ ೭:೧೬೨೯

ಹಾವೊಂದ ಕೈಯಲಿ ಹಿಡಿದು ಯಮ ಪ್ರತ್ಯಕ್ಷನಾದ, ಎಂದು
ಭಾಸವಾಗುವುದೇನು? ನಂದಿತೇ ಅರಿವಿನ ಸೊಡರು?
ಇಲ್ಲದಿರೆ, ತಮ್ಮ ಅಳಿವಿಗೆ ತಾವೇ ಕಾರಣರೂ ಆಗುವರೇ,
ಛೇ! ಎಲ್ಲ ಬಲ್ಲವರೇಕೆ ಹೀಗೆ ಕಾಲು ಜಾರಿಯಾರು?

೧೭ :: ವಿದ್ಯೆ ಶಿಕ್ಷಣ ಮತ್ತು ಗುರು

ಶ್ಲಿಷ್ಟಾ ಕ್ರಿಯಾ ಕಸ್ಯಚಿದ್ ಆತ್ಮಸಂಸ್ಥಾ,
ಸಂಕ್ರಾಂತಿರ್ ಅನ್ಯಸ್ಯ ವಿಶೇಷಯುಕ್ತಾ।
ಯಸ್ಯ ಉಭಯಂ ಸಾಧು ಸ ಶಿಕ್ಷಕಾಣಾಂ,
ಧುರಿ ಪ್ರತಿಷ್ಠಾಪಯಿತವ್ಯ ಏವ॥017

ಕಾಳಿದಾಸ, ಮಾಳವಿಕಾಗ್ನಿಮಿತ್ರ ೧:೧೬

ಬಲ್ಲವರು ಕೆಲರಿಹರು ಉದ್ದಾಮ ಗುರು ಪಂಡಿತೋತ್ತಮರು,
ಇನ್ನು ಕೆಲರೋ ಕಲಿಸ ಬಲ್ಲರು ಕರಗತವನೆಲ್ಲ ತೋರಿ;
ಬೆರಳೆಣಿಕೆಯಲ್ಲಿದ್ದಾರು ಅಲ್ಲಲ್ಲಿ, ಮತ್ತೆ, ನಿಸ್ಸೀಮರೆಡರಲೂ
ಅಂಥವರೇ ಹತ್ತಿ ನಿಲ್ಲಬಲ್ಲರು ಉತ್ತುಂಗ ಶಿಖರಕೇರಿ!

೧೮ :: ಹೇಳ ಹೆಸರಿಲ್ಲದೆ ಬದುಕಿದರೆ ಅದು ಬಾಳೆ?

ಅಂಧಃ ಸ ಏವ ಶ್ರುತ ವರ್ಜಿತೋ ಯಃ,
ಶಠಃ ಸ ಏವ ಅರ್ತಿ ನಿರರ್ಥಕೋ ಯಃ।
ಮೃತಃ ಸ ಏವ ಅಸ್ತಿ ಯಶೋ ನ ಯಸ್ಯಾ,
ಧರ್ಮೋ ನ ಧೀರ್ ಯಸ್ಯ ಸ ಏವ ಶೋಚ್ಯಃ॥018

ಕ್ಷೇಮೇಂದ್ರ, ಚತುರ್‍ವರ್ಗ ಸಂಗ್ರಹ ೧:೮

ಓದಿ ಬರೆಯದೆ ಹೋದೆ, ಕಣ್ಣಿದ್ದೂ ಕುರುಡಾದೆ, ಜಡನಾದೆ;
ಇದ್ದರೂ ಹಂಚಿ ಮೆಲ್ಲದೆ ಕಳೆದೆ- ಎಂಥ ಬಲ್ಲಿದನೋ ನೀನು?
ಹೇಳ ಹೆಸರಿಲ್ಲದೆ ಬದುಕಿ, ತೆರೆ ಬೀಳೆ ಮರೆಯಾದರದು ಬಾಳೆ?
ಧರ್ಮಬುದ್ಧಿ ಇಲ್ಲದಿರೆಂಥ ಬಲ್ಲವನೋ ನೀ, ಮರುಗುತಿಹೆನು!
