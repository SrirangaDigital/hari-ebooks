  11. ೦೬ – :: ಪಶ್ಚಿಮಾಭಿಮುಖಕ್ಕೊಂದು ಪೂರ್ವಸ್ಮರಣೆ'ಈ ಇ ಆ ಅ'
  12. ೦೭ – :: ಕಥೆಯಾಗಿ ಹೋದ ಪ್ರಾತಃಸ್ಮರಣೀಯರು'ವೇದಜೀವಿ ಗುಂಡಾವಧಾನಿಗಳು'
  13. ೦೮ – :: ನವಿರು ಮನಸ್ಸಿನ ಬಂಧುರ ಅಭಿವ್ಯಕ್ತಿ ಜಯಪ್ಪ ಹೊನ್ನಾಳಿ
    1. ಹುಲ್ಲಿನ ಮನೆ:
    2. ಎಲ್ಲಿ ಹೋಯ್ತ ನಿನ್ನ ಪ್ರೀತಿ?

  14. ೦೯ – :: ತೇಜ ನಿನದಗಳ ಅಂಕುರಾರ್ಪಣ 'ಚುಟುಕು ವರ್ಷಿಣಿ'
    1. ತೇಜಸ್ವಿನಿ ಅರಸ್ ಅವರಿಗೆ ಸಂದ ಪ್ರಶಸ್ತಿಗಳು:
  15. ೧೦ – :: ತಿಳಿನಗೆಯ ಹಾಲ್ಗಡಲು 'ನಿಶ್ಶಬ್ದ ಸಂಗೀತ'
  16. ೧೧ – :: ಬೆಚ್ಚನೆಯ ಚಿಂತನೆಗಳ ಪ್ರಯೋಗದ ಕೆಚ್ಚು 'ಮೀನಿನ ಹೆಜ್ಜೆ' ಮತ್ತು 'ಮಾಯಾವಿ ಸರೋವರ'

  17. ೧೨ – :: ವಿದ್ವತ್‍ಪೂರ್ಣ ಸಮೀಕ್ಷೆಗಳ ಸಮಾಹಾರ 'ಸಂಸ್ತವ' ಮತ್ತು 'ತತ್ತ್ವಾರ್ಥಚಿಂತನ'
    1. 'ತತ್ತ್ವಾರ್ಥಚಿಂತನ'
  18. ೧೩ – :: ಕುಡಿಕೆಯಲಿ ಕಡಲನಡಗಿಸಿ ಇಡುವ ಬಯಕೆ'ವಚನ ಉತ್ಸವ'
  19. ೧೪ – :: ವ್ಯಾವಹಾರಿಕ ಮಿಥ್ಯೆಯಲ್ಲಿ ಆಧ್ಯಾತ್ಮಿಕ ತಥ್ಯ ದರ್ಶಿಸಿದವನ ನಡೆ 'ಕನಕದಾಸ'

  20. ೧೫ – :: ನೋಡಲು ಹಿಂಜರಿಯುವ ನಾಣ್ಯದಇನ್ನೊಂದು ಮುಖ'ಮಾಂಗಲ್ಯ'
  21. ೧೬ – :: ಅಭಿವ್ಯಕ್ತಿಯ ಬಂಧುರ ಹಂದರ 'ಭಾವನಮನ'
  22. ೧೭ – :: ಸಾಂಸ್ಕೃತಿಕ ಪರ್ಯಟನೆಯ ದಾಖಲಾತಿಗಳು 'ಹೆರೋದಾತನ ದಿನಚರಿ'
    1. ಪ್ರಸ್ತುತ ಗ್ರಂಥದ ಕಥಾ ವಸ್ತುವೇನು?
    2. ನನ್ನ ಪ್ರಶ್ನೆ: ಈ ಗ್ರಂಥವನ್ನು ನೀವು ಅನುವಾದಿಸಿದ್ದು ಯಾಕೆ?
    3. ನನ್ನ ಪ್ರಶ್ನೆ: ಬರೆಯ ತೊಡಗಿದಾಗ ನಿಮ್ಮ ಆಕಾಂಕ್ಷೆ ಏನಾಗಿತ್ತು?
    4. ನನ್ನ ಪ್ರಶ್ನೆ: ನೀವೇ ಗುರುತಿಸುವುದಾದರೆ, ನಿಮ್ಮ ಈ ಗ್ರಂಥದವೈಶಿಷ್ಟ್ಯಗಳೇನು?
    5. ನನ್ನ ಪ್ರಶ್ನೆ: ನೀವು ಈ ಪುಸ್ತಕವನ್ನು ಬರೆಯಲು ಏಷ್ಟು ಕಾಲ ತೆಗೆದುಕೊಂಡಿರಿ?
    6. ನನ್ನ ಪ್ರಶ್ನೆ: ಕತೆಗಳೆ ತುಂಬಿರುವ ಈ ಪುಸ್ತಕದಲ್ಲಿ ಅವುಗಳನ್ನು ಹೇಳಲು ನೀವು ಯಾವ ಶೈಲಿಯನ್ನ ಸೂಕ್ತವೆಂದು ಕಂಡುಕೊಂಡಿರಿ?

  25. ೨೦ – :: ಸೃಜನಶೀಲ ಪ್ರತಿಭೆಗೊದಗಿದ ಕಾಣ್ಕೆ 'ಯೇಸು ಮಹಾದರ್ಶನ'
