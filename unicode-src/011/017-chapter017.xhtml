ಅನುಬಂಧಗಳು:

೧ :: ಕವನಗಳ ಸಂದರ್ಭ ಸೂಚಿ

೧. ಪದುಮ: (ಪುಟ ೧)

ದಾವಣಗೆರೆಯ ಬಿಡಿಟಿ ಇಂಜಿನಿಯರಿಂಗ್ ಕಾಲೇಜಿನಲ್ಲಿ ಓದುತ್ತಿದ್ದಾಗ ಬೆಂಗಳೂರಿನಲ್ಲಿ ಬಂಧುಗಳೊಬ್ಬರ ಮನೆಗೆ ಹೋಗುವ ಸಂದರ್ಭ ಒದಗಿಬಂತು. ಅವರ ಮನೆಯಲ್ಲಿ ಪಾದರಸದಂತೆ ಆಟವಾಡಿಕೊಂಡಿದ್ದ ಒಬ್ಬಳು ಚಿಕ್ಕ ಹುಡುಗಿಯನ್ನ ನೋಡಿದಾಗ ಕಲ್ಪನೆಗಳ ಬೆನ್ನೇರಿ ಬರೆದ ಕವನ. ಬಹುಶಃ ಕಾಬೂಲಿವಾಲ ಪ್ರಜ್ಞೆ ಪ್ರಭಾವ ಬೀರಿದ್ದಿರಬಹುದು.

ಈ ಕವನ ಆಕಾಶವಾಣಿ, ಬೆಂಗಳೂರು ಕೇಂದ್ರದಿಂದ ೨೪ ಜುಲೈ ೧೯೫೬ ರಂದು ಪ್ರಸಾರವಾಯಿತು. ಹಲವಾರು ಬಾರಿ ಬೇರೆ ಬೇರೆ ಆಕಾಶವಾಣಿ ಕೇಂದ್ರಗಳಿಂದ ಮರುಪ್ರಸಾರವೂ ಆಯಿತು. ಮಾರ್ಚ್ ೬, ೧೯೬೦ ರ 'ಜನಪ್ರಗತಿ' ಸಾಪ್ತಾಹಿಕ ಪತ್ರಿಕೆಯಲ್ಲಿ ಪ್ರಕಟವಾಯಿತು.

೨. ವಾಣಿ: (ಪುಟ ೩)

ಶಿವಮೊಗ್ಗದಲ್ಲಿ ಪ್ರಾಥಮಿಕ ಶಾಲೆಯಲ್ಲಿ ಓದುತ್ತಿದ್ದಾಗ ತರಗತಿಯಲ್ಲಿ ಹುಡುಗಿಯರೂ ಇರುತ್ತಿದ್ದರು. ಆ ಬಾಲ್ಯದ ದಿನಗಳ ಒಬ್ಬಳು ಸಹಪಾಠಿಯ ನೆನಪಿನಲ್ಲಿ ಆಮೇಲೆ ಎಷ್ಟೋ ವರ್ಷಗಳ ನಂತರ ಬರೆದ ಕವನ. (ಏಪ್ರಿಲ್ ೧೯೫೩, ಶಿವಮೊಗ್ಗ.)

೩. ಹೃದಯ ತಂತಿ ಮಿಡಿಯಿತು: (ಪುಟ ೬)

ಆಪ್ತ ಸೇಹಿತನೊಬ್ಬ ಮುನಿಸಿಕೊಂಡು ಮಾತು ಬಿಟ್ಟಾಗ ನೊಂದು ಬರೆದ ಕವನ. (ಮೇ ೧೯೫೫, ದಾವಣಗೆರೆ.)

೪. ಸರಸವಿದೇನೆ ಶಾರದೆಯೇ: (ಪುಟ ೮)

ಇಂಜಿನಿಯರಿಂಗ್ ಕಾಲೇಜಿನಲ್ಲಿ ವ್ಯಾಸಂಗ ಮಾಡುತ್ತಿದ್ದಾಗ ನಾಟಕ ಸಾಹಿತ್ಯದ ಗೀಳು ಹಚ್ಚಿಕೊಂಡು ಓದನ್ನೂ ನಿರ್ಲಕ್ಷ್ಯ ಮಾಡಿ ಪರೀಕ್ಷೆಯಲ್ಲಿ ಎಡವಿದಾಗ ಬರೆದ ಕವನ. (ಮೇ ೧೯೫೭, ಶಿವಮೊಗ್ಗ.)

೫. ಇನಿಯಳೆ ಏಕೆ: (ಪುಟ ೯)

ಅರೆಬರೆ ತಿಳಿವಳಿಕೆಯ ಸ್ವಾನುಭವದ ಕೊರತೆಯ ಹದಿವಯಸ್ಸಿನ ಹುಡುಗನೊಬ್ಬನ ಧರ್ಮೋಪದೇಶ. (ಜನವರಿ ೧೯೫೬, ದಾವಣಗೆರೆ.)

೬. ಅರಳಿದೆ ಮೊಗದ ನಗೆ ಮಲ್ಲಿಗೆ: (ಪುಟ ೧೦)

ತುಂಬಾ ಸಂತೋಷವಾದಾಗ ಏನೇನೋ ಕನಸುಗಳನ್ನ ಕಾಣುತ್ತಾ, ಇಂಜಿನಿಯರಿಂಗ್ ಪರೀಕ್ಷೆ ಇನ್ನೂ ಮುಗಿಯುವುದರೊಳಗೆ ಕೆಲಸಕ್ಕೆ ಸೇರುವ ಅವಕಾಶ ಒದಗಿ ಬಂದಿತೆಂದು ತಿಳಿದಾಗ ಬರೆದ ಕವನ. (೧೯೫೮, ದಾವಣಗೆರೆ)

೭. ಪ್ರೇಮ ಘೋರ: (ಪುಟ ೧೨)

ಯಾವುದೋ ಒಂದು ದಾಸರ ಕೀರ್ತನೆಯನ್ನು ಅನುಸರಿಸಿ ಬರೆದ ಕವನ. (ಫೆಬ್ರವರಿ ೦೧, ೧೯೫೮ ದಾವಣಗೆರೆ.)

೮. ಕನ್ನಡದ ತಿಳಿನೀರ ಸರಸಿಯಲಿ: (ಪುಟ ೧೩)

ಕನ್ನಡ ನಾಡು ಒಂದಾದ ವಿಜಯೋತ್ಸವದ ನೆನಪಿನಲ್ಲಿ ಬರೆದ ಕವನ. (ನವಂಬರ್ ೧೯೫೮, ದಾವಣಗೆರೆ.)

೯. ವಿದ್ಯುತ್ ಲತಾಂಗಿ: (ಪುಟ ೧೫)

ಬೆಂಗಳೂರಿನ ಆಕಾಶವಾಣಿ ಕೇಂದ್ರದವರು ಬೇರೆ ಬೇರೆ ಊರುಗಳಿಗೆ ಹೋಗಿ ಅಲ್ಲಿನ ಯುವ ಪ್ರತಿಭೆಗಳನ್ನ ಗುರುತಿಸಿ, ಅವರನ್ನು ಪ್ರೋತ್ಸಾಹಿಸುವ ಒಂದು ಕಾರ್ಯಕ್ರಮ ಹಮ್ಮಿಕೊಂಡಿದ್ದರು. ಆ ಉದ್ದೇಶದಿಂದ ಅವರು ದಾವಣಗೆರೆಗೆ ಬಂದಾಗ ನಡೆಸಿದ ಪ್ರತಿಭಾ ಸ್ಪರ್ಧೆಯಲ್ಲಿ ಮನ್ನಣೆಗಳಿಸಿದ ಕವನ ಇದು.

ಇಂಜಿನಿಯರಿಂಗ್ ವಿದ್ಯಾರ್ಥಿಯಾಗಿದ್ದರಿಂದ ನನ್ನ ಬೇರೆ ಕವನಗಳನ್ನ ಬಿಟ್ಟು ವಿಜ್ಞಾನಕ್ಕೆ ಸಂಬಂಧಪಟ್ಟ ಈ ಕವನವನ್ನ ಆರಿಸಿಕೊಂಡರು ಅಂಥ ಕಾಣುತ್ತೆ. (ರಚನೆ: ಜನವರಿ ೧೯೫೯ ದಾವಣಗೆರೆ; ಬೆಂಗಳೂರು ಆಕಾಶವಾಣಿ ಕೇಂದ್ರದಲ್ಲಿ ಪ್ರಸಾರ: ಜುಲೈ ೨೪, ೧೯೫೯)

೧೦. ಮುಂಬದಿಯ ಸಾಲಿನಲ್ಲಿ ಕುಳಿತ ಸುಕುಮಾರಿ: (ಪುಟ ೧೭)

ನಮ್ಮ ಮನೆ ಇದ್ದುದು ಶಿವಮೊಗ್ಗದಲ್ಲಿ; ಓದುತ್ತಿದ್ದುದು ದಾವಣಗೆರೆಯ ಇಂಜಿನಿಯರಿಂಗ್ ಕಾಲೇಜಿನಲ್ಲಿ. ಸುಮಾರು ತಿಂಗಳಿಗೊಮ್ಮೆ ಊರಿಗೆ ಹೋಗಿ ಬರುತ್ತಿದ್ದೆ. ಬಸ್ಸಿನಲ್ಲಿ ಉಚಿತವಾಗಿ ಪ್ರಯಾಣ ಮಾಡಲು ಉದಾರ ದಾನಿಗಳಾದ ಉದಯ ಮೋಟಾರ್ ಸರ್ವೀಸ್ ನವರು ಅನುಮತಿ ಪತ್ರ ಕೊಟ್ಟಿದ್ದರು.

ಹೀಗೆ ಪ್ರಯಾಣ ಮಾಡುತ್ತಿದ್ದಾಗ ಒಮ್ಮೆ ಬಸ್ಸಿನಲ್ಲೇ ಕುಳಿತು ಗೀಚಿ, ಅಮೇಲೆ ತಿದ್ದಿದ ಕವನದ ರೂಪ. (ಜುಲೈ ೬, ೧೯೫೮.)

೧೧. ಕಿಡಿ-ನುಡಿ: (ಪುಟ ೧೮)

ವಾಲ್ಮೀಕಿ ರಾಮಾಯಣದ ರಚನೆಗೆ ಸಂಬಂಧಪಟ್ಟ ಒಂದು ಘಟನೆ ಹೀಗೆ ಆಗಿರಬಹುದು ಎಂದು ಊಹಿಸಿ ಬರೆದದ್ದು. ರಚನೆ ಮೇ ೧೯, ೧೯೫೬ ಶಿವಮೊಗ್ಗ.

ಕೆಲಕಾಲ ನಂತರ ಸುಮಾರು ೧೯೫೮ ರಲ್ಲಿ ಕೊನೆ ಮಾತುಗಳನ್ನು ಸೇರಿಸಿದ್ದು. ಆಮೇಲೆ ಬೆಂಗಳೂರಿನಿಂದ ಹೊರಬರುತ್ತಿದ್ದ 'ಜೀವನ' ಮಾಸಪತ್ರಿಕೆಯಲ್ಲಿ ೧೯೭೪ ರಲ್ಲಿ ಇದು ಪ್ರಕಟವಾಯಿತು.

೧೨. ನೇಹ-ನಲ್ಮೆ: (ಪುಟ ೨೦)

ಯಾವುದೋ ಒಂದು ಇಂಗ್ಲಿಷ್ ಕವಿತೆಯನ್ನು (ಅದರ ಹೆಸರು ಮರೆತು ಹೋಗಿದೆ) ಓದಿದ ಮೇಲೆ ಅದರ ಪ್ರಭಾವದ ಗುಂಗಿನಲ್ಲಿ ಬರೆದ ಕವನ. ರಚನೆ: ಮೇ ೧೩, ೧೯೫೭ ಶಿವಮೊಗ್ಗ.)

೧೩. ಮಿಂದು ತಾ ಬಂದಿಹಳು: (ಪುಟ ೨೨)

ಹದಿವಯಸಿನ ರಸಿಕತೆಯ ಹುಚ್ಚು ಕನಸು (ನವಂಬರ್ ೨೮, ೧೯೫೬ ದಾವಣಗೆರೆ)

೧೪. ಇನ್ನಿಲ್ಲವೆನ್ನಾಕೆ: (ಪುಟ ೨೪)

ವಿವಾಹವಾದ ಯುವಕನೊಬ್ಬ ತನ್ನ ನವ ವಧುವನ್ನು ಮದುವೆಯಾದ ಕೆಲದಿನಗಳಲ್ಲೇ ಕಳೆದುಕೊಂಡ ಎಂಬ ಸುದ್ದಿಯನ್ನು ಪತ್ರಿಕೆಗಳಲ್ಲಿ ಓದಿದಾಗ ಸ್ಫುರಿಸಿದ ಕವನ. (ಪದ್ಯ ಅಪೂರ್ಣ; ಬರೆದದ್ದು ಶಿವಮೊಗ್ಗ ಸುಮಾರು ೧೯೫೪)

೧೫. ಕರೆ: (ಪುಟ ೨೬)

ಇಂಜಿನಿಯರಿಂಗ್ ಕಾಲೇಜಿನಲ್ಲಿ ಅಧ್ಯಾಪಕನಾಗಿದ್ದಾಗ ಬರೆದ ಕವನ. ಆರ್.ವಿ. ಇಂಜಿನಿಯರಿಂಗ್ ಕಾಲೇಜು, ಬೆಂಗಳೂರು ವಾರ್ಷಿಕ 'ಸಂಸ್ಮರಣಿಕೆ'ಯಲ್ಲಿ ಪ್ರಕಟವಾಯಿತು. (೧೯೭೧, ಬೆಂಗಳೂರು)

೧೬. ತುಣುಕುಗಳು: (ಪುಟ ೩೦)

ಆಗೊಮ್ಮೆ ಈಗೊಮ್ಮೆ ಕುರುಕಿದ ಚುಟುಕುಗಳು. (೧೯೫೯)

೨ :: ಕವನಗಳ ಅಕಾರಾದಿ

ಅರಳಿದೆ ಮೊಗದ ನಗೆ ಮಲ್ಲಿಗೆ /೧೩

ಇನಿಯಳೇ ಏಕೆ? /೧೧

ಇನ್ನಿಲ್ಲವೆನ್ನಾಕೆ /೨೮

ಕನ್ನಡದ ತಿಳಿನೀರ ಸರಸಿಯಲಿ /೧೬

ಕರೆ /೩೦

ಕಿಡಿ-ನುಡಿ /೨೨

ತುಣುಕುಗಳು /೩೪

ನೇಹ-ನಲ್ಮೆ /೨೪

ಪದುಮ /೧

ಪ್ರೇಮ-ಘೋರ /೧೫

ಮಿಂದು ತಾ ಬಂದಿಹಳು, ನಿಂದಿಹಳು ಮನದನ್ನೆ! /೨೬

ಮುಂಬದಿಯ ಸಾಲಿನಲಿ ಕುಳಿತ ಸುಕುಮಾರಿ /೨೦

ವಾಣಿ /೦೪

ವಿದ್ಯುತ್ ಲತಾಂಗಿ /೧೮

ಸರಸವಿದೇನೇ ಶಾರದೆಯೇ /೧೦

ಹೃದಯ ತಂತಿ ಮಿಡಿಯಿತು /೮

೩ :: ಕವನಗಳ ಮೊದಲ ಸಾಲುಗಳ ಅಕಾರಾದಿ

ಅರಳಿದೆ ಮೊಗದ ನಗೆ ಮಲ್ಲಿಗೆ / ಪುಟ ೧೩

ಅರೆ, ಮಾಧು, ಮಾತುಲನ ಮನೆಯಿಂದ ನವವಧುವ/ ಪುಟ ೨೮

ಆ ತಳೋದರಿ ವಾಣಿ ನಮ್ಮೆಲ್ಲರೊಡನಾಡಿ! / ಪುಟ ೦೪

ಊರನ್ನು ಕಟ್ಟೋಣ, ನಾಡನ್ನೆ ಬೆಳಗೋಣ / ಪುಟ ೩೦

ಎನ್ನೊಲವು ಮೀಸಲಿದೆ; ನೇಹಿಗನ ಮಾರ್ದವದ / ಪುಟ ೨೪

ಓ ಸಹೃದಯೀ ಗೆಳೆಯ, ನಿತ್ಯ ನೂತನ / ಪುಟ ೩೫

ಕನ್ನಡದ ತಿಳಿನೀರ ಸರಸಿಯಲಿ / ಪುಟ ೧೬

ಕ್ರೌಂಚದಳುವನು ಕೇಳಿ ಎಚ್ಚರಿತು ಬೆಚ್ಚಿದನು / ಪುಟ ೨೨

ನಲುಮೆಯ ಒಲುಮೆಯ ಚಿಲುಮೆಯನು / ಪುಟ ೩೬

ನೋಡು, ಪದುಮಳ ಮುದ್ದು ಮುಖದೊಳಗೆಂಥ / ಪುಟ ೦೧

ಬಿಳಿ ದುಕೂಲವನುಟ್ಟು, ತಿಲಕ ಕತ್ತುರಿಯಿಟ್ಟು / ಪುಟ ೨೬

ಮನದ ತಾರ್ಕಣೆ ಸೃಜಸೆ ಬೇಸರದ ಕಾರಿರುಳ / ಪುಟ ೩೪

ಮುಂಬದಿಯ ಸಾಲಿನಲಿ ಕುಳಿತೋರ್ವ / ಪುಟ ೨೦

ಶಿಸ್ತು ಸಂಯಮ ಚಿತ್ತ ನಿಗ್ರಹ ಬೇಕು / ಪುಟ ೧೧

ಶ್ಯಾಮಸುಂದರ ನೀನೇ ಚಕ್ರಿ ಹರಿಯೇ / ಪುಟ ೧೫

ಸರಸವಿದೇನೇ ಶಾರದೆಯೇ / ಪುಟ ೧೦

ಸಾಲುಗಂಬದ ಮೇಲೆ ಸಾಗುತಿಹೆ / ಪುಟ ೧೮

ಹೃದಯ ತಂತಿ ಮಿಡಿಯಿತು / ಪುಟ ೦೮

೪ :: ಇದೇ ಲೇಖಕರ ಇತರೆ ಕೃತಿಗಳು

ಪ್ರಕಟಿತ ಕೃತಿಗಳು:

'ವಿದೇಶಕ್ಕೆ ಬಂದವರು' (ಕವನ ಸಂಕಲನ) (೧೯೯೮, ೨೦೦೫)

'ಮಾತಿನ ಮಂಟಪ' (ಪ್ರಬಂಧ ಸಂಕಲನ) (೨೦೦೨)

'ರಾಂಟ್‍ಜನ್' (ವೈಜ್ಞಾನಿಕ ಸಾಹಿತ್ಯ) (೧೯೭೫)

'ಮಾತಿನ ಚಪ್ಪರ' (ಪ್ರಬಂಧ ಸಂಕಲನ) (೨೦೦೫)

'ಕನ್ನಡ ಉಳಿಸಿ ಬೆಳಸುವ ಬಗೆ' (ಅಮೆರಿಕನ್ನಡ ಚಿಂತನೆಗಳು) (೨೦೦೫)

'ಕನ್ನಡದಲ್ಲಿ ಶ್ರೀ ಸತ್ಯನಾರಾಯಣ ಪೂಜೆ' (ಮಂತ್ರ ಮಂಜರಿ) (೨೦೦೬)

'ಮಾತಿನ ಮಲ್ಲಿಗೆ' (ಲಲಿತ ಪ್ರಬಂಧ ಸಂಕಲನ) (೨೦೦೯)

'ಮಾತೇ ಮುತ್ತು' (ವೈಚಾರಿಕ ಲೇಖನಗಳು) (೨೦೦೯)

'ಮುಕ್ತಕ ಕನ್ನಡ' (ಸಂಸ್ಕೃತ ಸುಭಾಷಿತ ಪದ್ಯಗಳ ಕನ್ನಡ ಭಾವಾನುವಾದ) (೨೦೦೯)

'ಗೀತಗುಚ್ಛ' (ಸಂಸ್ಕೃತ ಪದ್ಯಗಳ ಕನ್ನಡ ಭಾವಾನುವಾದ) (೨೦೦೯)

'ಗುಬ್ಬಿ ಗುಬ್ಬಚ್ಚಿ'- ಒಂದು ವಿವೇಚನೆ (೨೦೦೯)

ಪ್ರಧಾನ ಸಂಪಾದಕ:

'ಅಮೆರಿಕನ್ನಡ' (ಉತ್ತರ ಅಮೆರಿಕಾದ ಕನ್ನಡ ಸಾಹಿತ್ಯ ದ್ವೈಮಾಸಿಕ)

'ಸ್ಮರಣ ಸಂಚಿಕೆ' (ವಿಶ್ವಕನ್ನಡ ಸಮ್ಮೇಳನ, ಟೆಕ್ಸಸ್, ಯುಎಸ್‍ಎ, ೨೦೦೦)

'ದರ್ಶನ' (ವಿಶ್ವ ಕನ್ನಡ ಸಮ್ಮೇಳನ, ಟೆಕ್ಸಸ್, ಯುಎಸ್‍ಎ, ೨೦೦೦)

'ಚಿತ್ರಭಾನು' (ಉತ್ತರ ಕ್ಯಾಲಿಫೋರ್ನಿಯಾ, ಸಾಹಿತ್ಯ ವಾರ್ಷಿಕ, ೨೦೦೨)

'ತುರಾಯಿ' (ಮಧ್ಯವಲಯ ಕನ್ನಡ ಸಮ್ಮೇಳನ, ಮಿಸ್ಸೌರಿ, ೧೯೯೧)

'ಅಕ್ಷಯ' (ಪ್ರೊ| ಕೆ.ಭೈರವಮೂರ್ತಿರವರ ಅಭಿನಂದನಾ ಗ್ರಂಥ ೨೦೦೮)

'ಸಂವಹನ ಕಾವ್ಯ-೨೦೦೮' (ಕವನ ಸಂಕಲನ, ೨೦೦೮)

ಸಹ-ಸಂಪಾದಕ:

'ಕನ್ನಡ ಭಾಗವತ ಸಮಗ್ರ ಸಂಪುಟ' (ಪರಿಶೋಧಕ ಸಮಿತಿ ಸದಸ್ಯ, ೨೦೦೮)

'ಸ್ಪಂದನ ಸ್ಮರಣ ಸಂಚಿಕೆ' (ಅಕ್ಕ ವಿಶ್ವಕನ್ನಡ ಸಮ್ಮೇಳನ, ಡೆಟ್ರಾಯಿಟ್ ಯುಎಸ್‍ಎ, ೨೦೦೨)

'ವಿಕಾಸ ಸ್ಮರಣ ಸಂಚಿಕೆ' (ಅಕ್ಕ ವಿಶ್ವಕನ್ನಡ ಸಮ್ಮೇಳನ, ಫ್ಲಾರಿಡಾ ಯುಎಸ್‍ಎ, ೨೦೦೪)

'ಸಮ್ಮೇಳನ ಸ್ಮರಣ ಸಂಚಿಕೆ' (ಅಕ್ಕ ವಿಶ್ವಕನ್ನಡ ಸಮ್ಮೇಳನ ವಾಷಿಂಗ್‍ಟನ್ ಡಿಸಿ, ಯುಎಸ್‍ಎ, ೨೦೦೬)

'ಪ್ರಬಂಧ ವಾಹಿನಿ' (ಅಕ್ಕ ವಿಶ್ವಕನ್ನಡ ಸಮ್ಮೇಳನ ವಾಷಿಂಗ್‍ಟನ್ ಡಿಸಿ, ಯುಎಸ್‍ಎ, ೨೦೦೬)

ಪ್ರಕಟಣೆಯ ವಿವಿಧ ಹಂತಗಳಲ್ಲಿ:

'ಲಲ್ಲೇಶ್ವರಿ' (ಕಾಶ್ಮೀರ ಶೈವ ಅನುಭಾವಿಯ ವಚನಗಳ ವಿಶ್ಲೇಷಣೆ)

'ಬೈಬಲ್‍ನಲ್ಲಿ ಜಾನ್‍ನ ಸುವಾರ್ತೆ: ಒಂದು ಅಧ್ಯಯನ'

'ಫಾರ್ಸಿ ಕವನಗಳು' (ಆಯ್ದ ಕವನಗಳ ಕನ್ನಡ ಭಾವಾನುವಾದ)

'ತಿರುಗಾಟ' (ಕಾದಂಬರಿ)

'ಪೂರ್ವಸೂರಿಗಳು ಕಂಡ ಕಾಶ್ಮೀರ'(ಪ್ರಬಂಧ ಸಂಕಲನ)

'ಮನೆಗೆ ಮರಳಿದ ಮೇಲೆ' (ಕವನ ಸಂಕಲನ)

'ಮಾತಿನ ಚಾವಡಿ' (ಸಾಹಿತ್ಯ ವಿಮರ್ಶೆಯ ಲೇಖನಗಳು)

'ಮಾತಿನ ಮನೆ' (ಆತ್ಮ ಕಥನ)

'ಕನ್ನಡದಲ್ಲಿ ಮದುವೆ' (ಸಂಸ್ಕೃತ ಮಂತ್ರಗಳ ಕನ್ನಡ ಪದ್ಯಾನುವಾದಗಳು)

'ಪುರಂದರದಾಸರು' (ಗೀತ ನಾಟಕ)

'ಕನ್ನಡ ಮಂತ್ರ ಮಂಜರಿ' (ಕನ್ನಡದಲ್ಲಿ ದೇವರ ಪೂಜೆ)

'ಭಾಸ್ಕರನ ಲೀಲಾವತಿ' (ಸಂಸ್ಕೃತ ಗಣಿತ ಗ್ರಂಥದ ಒಳನೋಟ)

'ಯೇಸುಕ್ರಿಸ್ತನ ಕೊನೆಯ ದಿನಗಳು' (ಬೈಬಲ್‍ನ ಒಂದು ಅಧ್ಯಯನ)
